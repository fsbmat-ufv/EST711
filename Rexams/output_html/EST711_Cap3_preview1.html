Exam 1

  1. Question

    Seja \(X_i \overset{\text{iid}}{\sim} N(\mu, \sigma_0^2)\) com \(\sigma_0^2 = 9\) conhecido. Com base em uma amostra de tamanho \(n = 64\), o estimador de \(\mu\) pelo método dos momentos é:


    1. o momento de segunda ordem \(m_2 = \frac{1}{n}\sum X_i^2\).
    2. o primeiro momento amostral \(m_1 = \bar{X}_n\), que iguala o momento teórico \(\mu_1 = \mu\).
    3. o momento amostral de maior ordem disponível.
    4. o estimador que minimiza \(\sum (X_i - \theta)^2\).
    5. a mediana amostral, pois é mais robusta que a média.

    Solution

    O método dos momentos consiste em igualar os momentos teóricos da distribuição aos momentos amostrais. Para \(X \sim N(\mu, \sigma_0^2)\), o primeiro momento teórico é \(\mu_1 = E(X) = \mu\). Igualando ao primeiro momento amostral \(m_1 = \bar{X}_n\), obtemos \(\hat{\mu}_{MM} = \bar{X}_n\). Não é necessário usar momentos de ordem superior nem \(\sigma_0^2\), pois o parâmetro de interesse \(\mu\) já aparece no primeiro momento.


  2. Question

    Seja \(X_i \overset{\text{iid}}{\sim} N(\mu, 9)\) com \(\sigma_0^2 = 9\) conhecido. A log-verossimilhança é \(\ell(\mu) = -\frac{1}{2\sigma_0^2}\sum_{i=1}^n (X_i - \mu)^2 + \text{const}\). Com \(n = 36\) e \(\sum_{i=1}^n X_i = 107.7\), calcule o EMV \(\hat{\mu}_{MV} = \bar{X}\).


    1. 3.2917
    2. 2.6917
    3. 2.9917
    4. 3.2417
    5. 2.8342

    Solution

    Derivando \(\ell(\mu)\) em relação a \(\mu\) e igualando a zero: \[\frac{d\ell}{d\mu} = \frac{1}{\sigma_0^2}\sum_{i=1}^n (X_i - \mu) = 0 \implies \hat{\mu}_{MV} = \bar{X} = \frac{\sum X_i}{n} = \frac{107.7}{36} = 2.9917.\] O valor de \(\sigma_0^2 = 9\) não influencia o EMV de \(\mu\) quando \(\sigma^2\) é conhecido.
