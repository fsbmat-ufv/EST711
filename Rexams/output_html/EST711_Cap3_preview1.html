Exam 1

  1. Question

    Seja \(X_i \overset{\text{iid}}{\sim} N(\mu, \sigma_0^2)\) com \(\sigma_0^2 = 1\) conhecido. Com base em uma amostra de tamanho \(n = 25\), o estimador de \(\mu\) pelo método dos momentos é:


    1. o primeiro momento amostral \(m_1 = \bar{X}_n\), que iguala o momento teórico \(\mu_1 = \mu\).
    2. o momento de segunda ordem \(m_2 = \frac{1}{n}\sum X_i^2\).
    3. a mediana amostral, pois é mais robusta que a média.
    4. o estimador que minimiza \(\sum (X_i - \theta)^2\).
    5. o momento amostral de maior ordem disponível.

    Solution

    O método dos momentos consiste em igualar os momentos teóricos da distribuição aos momentos amostrais. Para \(X \sim N(\mu, \sigma_0^2)\), o primeiro momento teórico é \(\mu_1 = E(X) = \mu\). Igualando ao primeiro momento amostral \(m_1 = \bar{X}_n\), obtemos \(\hat{\mu}_{MM} = \bar{X}_n\). Não é necessário usar momentos de ordem superior nem \(\sigma_0^2\), pois o parâmetro de interesse \(\mu\) já aparece no primeiro momento.


  2. Question

    Seja \(X_i \overset{\text{iid}}{\sim} N(\mu, 1)\) com \(\sigma_0^2 = 1\) conhecido. A log-verossimilhança é \(\ell(\mu) = -\frac{1}{2\sigma_0^2}\sum_{i=1}^n (X_i - \mu)^2 + \text{const}\). Com \(n = 100\) e \(\sum_{i=1}^n X_i = 300.6\), calcule o EMV \(\hat{\mu}_{MV} = \bar{X}\).


    1. 3.016
    2. 3.006
    3. 3.306
    4. 2.706
    5. 2.9471

    Solution

    Derivando \(\ell(\mu)\) em relação a \(\mu\) e igualando a zero: \[\frac{d\ell}{d\mu} = \frac{1}{\sigma_0^2}\sum_{i=1}^n (X_i - \mu) = 0 \implies \hat{\mu}_{MV} = \bar{X} = \frac{\sum X_i}{n} = \frac{300.6}{100} = 3.006.\] O valor de \(\sigma_0^2 = 1\) não influencia o EMV de \(\mu\) quando \(\sigma^2\) é conhecido.
