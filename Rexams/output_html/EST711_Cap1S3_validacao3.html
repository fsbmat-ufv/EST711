Exam 3

  1. Question

    Ao modelar um experimento físico, uma função \(P\) é definida sobre os eventos de um espaço amostral \(\Omega\). Qual condição faz parte dos axiomas de probabilidade?


    1. Para eventos dois a dois disjuntos \(A_1,A_2,\ldots\), vale \(P(\bigcup_i A_i)=\sum_iP(A_i)\).
    2. Se \(A\subset B\), então \(P(A)>P(B)\).
    3. A probabilidade do espaço amostral pode assumir qualquer constante positiva.
    4. Para quaisquer eventos \(A\) e \(B\), vale sempre \(P(A\cup B)=P(A)+P(B)\).
    5. Todo evento não vazio deve ter probabilidade estritamente positiva.

    Solution

    Os axiomas exigem não negatividade, \(P(\Omega)=1\) e aditividade enumerável para eventos dois a dois disjuntos. Para eventos que se sobrepõem, a interseção precisa ser descontada. Eventos não vazios podem ter probabilidade zero.


  2. Question

    Em certo experimento, \(P(A)=0.20\). Qual é \(P(A^c)\)?


    1. \(0.0400\)
    2. \(0.8000\)
    3. \(0.8333\)
    4. \(0.6400\)
    5. \(0.2000\)

    Solution

    Como \(A\) e \(A^c\) são disjuntos e sua união é \(\Omega\), \[P(A^c)=1-P(A)=1-0.20=0.8000.\]


  3. Question

    Em uma população de 120 unidades, 31 satisfazem \(A\), 44 satisfazem \(B\) e 21 satisfazem ambos. Selecionando uma unidade ao acaso, qual é \(P(A\cup B)\)?


    1. \(0.450\)
    2. \(0.550\)
    3. \(0.800\)
    4. \(0.625\)
    5. \(0.175\)

    Solution

    Pela fórmula da adição, \[P(A\cup B)=P(A)+P(B)-P(A\cap B) =\frac{31+44-21}{120} =0.450.\]


  4. Question

    Para três eventos, são conhecidas as probabilidades \[(P(A),P(B),P(C))=(0.45,0.3,0.5),\] \[(P(A\cap B),P(A\cap C),P(B\cap C))=(0.1,0.14,0.13)\] e \(P(A\cap B\cap C)=0.05\). Calcule \(P(A\cup B\cup C)\).


    1. \(1.300\)
    2. \(0.930\)
    3. \(0.070\)
    4. \(0.880\)
    5. \(1.250\)

    Solution

    A inclusão-exclusão fornece \[P(A\cup B\cup C)=\sum P(A_i)-\sum P(A_i\cap A_j)+P(A\cap B\cap C) =0.930.\]


  5. Question

    Uma comissão de 4 pessoas é escolhida uniformemente, sem reposição, entre 6 mulheres e 9 homens. Qual é a probabilidade de a comissão conter exatamente 1 mulheres?


    1. \(0.2500\)
    2. \(0.6308\)
    3. \(0.0044\)
    4. \(0.2667\)
    5. \(0.3692\)

    Solution

    Há \(\binom{15}{4}\) comissões possíveis. As favoráveis escolhem 1 mulheres e 3 homens: \[P=\frac{\binom{6}{1}\binom{9}{3}} {\binom{15}{4}} =0.3692.\]


  6. Question

    Supondo 365 dias equiprováveis e ignorando anos bissextos, qual é a probabilidade de haver ao menos duas pessoas com o mesmo aniversário em um grupo de 35 pessoas?


    1. \(1.6301\)
    2. \(0.8144\)
    3. \(0.0916\)
    4. \(0.1856\)
    5. \(0.0959\)

    Solution

    Pelo complemento, \[P(\text{coincidência})=1-\prod_{j=0}^{34}\frac{365-j}{365} =0.8144.\]


  7. Question

    Para 3 eventos, as probabilidades individuais são 0.17, 0.16, 0.09. Sem informação sobre interseções, qual limite superior é garantido por Boole para \(P(\bigcup_iA_i)\)?


    1. \(0.0024\)
    2. \(0.1700\)
    3. \(0.4200\)
    4. \(0.5800\)
    5. \(0.1400\)

    Solution

    A desigualdade de Boole fornece \[P\left(\bigcup_iA_i\right)\leq\sum_iP(A_i).\] Como uma probabilidade não excede 1, o limite é \(\min\{1,\sum_iP(A_i)\}=0.4200\).


  8. Question

    Considere eventos crescentes \(A_1\subseteq A_2\subseteq\cdots\), por exemplo A_n = {X <= n}. Qual propriedade é correta?


    1. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}P(A_n)\).
    2. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}[1-P(A_n)]\).
    3. \(P(\bigcap_{n=1}^{\infty}A_n)=\sum_{n=1}^{\infty}P(A_n)\).
    4. \(P(\bigcup_{n=1}^{\infty}A_n)=\prod_{n=1}^{\infty}P(A_n)\).
    5. A igualdade com o limite exige que os \(A_n\) sejam disjuntos.

    Solution

    A probabilidade é contínua por baixo: \[P\left(\bigcup_{n=1}^{\infty}A_n\right)=\lim_{n\to\infty}P(A_n).\] A condição necessária aqui é a monotonicidade crescente, não a disjunção.


  9. Question

    Um dado honesto de 10 faces, numeradas de 1 a 10, é lançado. Qual é a probabilidade de o resultado ser pelo menos 3?


    1. \(0.2000\)
    2. \(0.3000\)
    3. \(0.8000\)
    4. \(0.3333\)
    5. \(0.7000\)

    Solution

    Há 8 resultados favoráveis entre 10 equiprováveis. Logo, \[P(X\geq 3)=\frac{8}{10} =0.8000.\]


  10. Question

    Sabe-se apenas que \(P(A)=0.85\) e \(P(B)=0.55\). Qual é o limite inferior de Bonferroni para \(P(A\cap B)\)?


    1. \(0.550\)
    2. \(0.600\)
    3. \(0.468\)
    4. \(0.400\)
    5. \(-0.400\)

    Solution

    Como \(P(A\cup B)\leq1\), \[P(A\cap B)\geq P(A)+P(B)-1 =0.400.\]
