Exam 1

  1. Question

    Seja \(T_n\) um estimador de \(\theta\) baseado em uma amostra de tamanho \(n\). Qual das afirmações abaixo define corretamente a convergência em probabilidade \(T_n \xrightarrow{P} \theta\)?


    1. \(P(|T_n - \theta| < \varepsilon) \to 0\) quando \(n \to \infty\), para todo \(\varepsilon > 0\).
    2. \(P(|\theta - T_n| \leq \varepsilon) \to 0\) quando \(n \to \infty\), para todo \(\varepsilon > 0\).
    3. \(P(|T_n - \theta| \geq \varepsilon) \to 0\) quando \(n \to \infty\), para todo \(\varepsilon > 0\).
    4. \(P(|T_n - \theta| = 0) \to 1\) quando \(n \to \infty\).
    5. \(T_n \to \theta\) quase certamente (com probabilidade 1).

    Solution

    A definição correta de convergência em probabilidade é: para todo \(\varepsilon > 0\), \(P(|T_n - \theta| \geq \varepsilon) \to 0\) quando \(n \to \infty\).

    Equivalentemente, \(P(|T_n - \theta| < \varepsilon) \to 1\), mas a definição padrão usa a probabilidade do evento \(\{|T_n - \theta| \geq \varepsilon\}\) ir a zero. As demais alternativas confundem o sentido da desigualdade, trocam por convergência quase certa (mais forte), ou usam \(P(|T_n - \theta| = 0)\), que não é a definição correta.


  2. Question

    Seja \((X_n)_{n \geq 1}\) uma sequência de variáveis aleatórias e \(X\) uma variável aleatória com função de distribuição acumulada \(F_X\). Qual afirmação define corretamente a convergência em distribuição \(X_n \xrightarrow{D} X\)?


    1. \(E(X_n) \to E(X)\) e \(\text{Var}(X_n) \to \text{Var}(X)\).
    2. \(F_{X_n}(x) \to F_X(x)\) para todo \(x\) que seja ponto de continuidade de \(F_X\) (conjunto \(C(F_X)\)).
    3. \(F_{X_n}(x) \to F_X(x)\) para todo \(x \in \mathbb{R}\), incluindo pontos de descontinuidade de \(F_X\).
    4. \(X_n - X \xrightarrow{P} 0\), ou seja, a diferença converge em probabilidade a zero.
    5. \(P(X_n = x) \to P(X = x)\) para todo \(x \in \mathbb{R}\).

    Solution

    A definição correta de convergência em distribuição (também chamada de convergência fraca) é: \(X_n \xrightarrow{D} X\) se e somente se \(F_{X_n}(x) \to F_X(x)\) para todo \(x \in C(F_X)\), onde \(C(F_X)\) é o conjunto dos pontos de continuidade de \(F_X\).

    A restrição aos pontos de continuidade é fundamental: se \(F_X\) for descontínua em \(x_0\) (como ocorre em distribuições discretas), não se exige convergência pontual em \(x_0\). As demais alternativas exigem convergência em todos os pontos (mais forte do que o necessário), confundem com convergência de momentos ou com convergência em probabilidade.


  3. Question

    Ao modelar um experimento físico, uma função \(P\) é definida sobre os eventos de um espaço amostral \(\Omega\). Qual condição faz parte dos axiomas de probabilidade?


    1. Para quaisquer eventos \(A\) e \(B\), vale sempre \(P(A\cup B)=P(A)+P(B)\).
    2. Se \(A\subset B\), então \(P(A)>P(B)\).
    3. Para eventos dois a dois disjuntos \(A_1,A_2,\ldots\), vale \(P(\bigcup_i A_i)=\sum_iP(A_i)\).
    4. A probabilidade do espaço amostral pode assumir qualquer constante positiva.
    5. Todo evento não vazio deve ter probabilidade estritamente positiva.

    Solution

    Os axiomas exigem não negatividade, \(P(\Omega)=1\) e aditividade enumerável para eventos dois a dois disjuntos. Para eventos que se sobrepõem, a interseção precisa ser descontada. Eventos não vazios podem ter probabilidade zero.


  4. Question

    Em 110 registros, 39 apresentam \(B\) e 9 apresentam simultaneamente \(A\) e \(B\). Qual é \(P(A\mid B)\)?


    1. \(0.2308\)
    2. \(0.6455\)
    3. \(0.3545\)
    4. \(0.0818\)
    5. \(0.7692\)

    Solution

    \[P(A\mid B)=\frac{P(A\cap B)}{P(B)} =\frac{9}{39}=0.2308.\]


  5. Question

    Qual conjunto de propriedades é necessário para que \(H\) seja uma função de distribuição acumulada em \(\mathbb{R}\)?


    1. Ser estritamente crescente, contínua e possuir derivada positiva em todos os pontos.
    2. Ser não decrescente, contínua à direita, com limites 0 em \(-\infty\) e 1 em \(+\infty\).
    3. Ser não crescente, contínua à esquerda e integrar 1.
    4. Assumir apenas os valores 0 e 1.
    5. Ser simétrica em torno de zero e limitada por uma densidade.

    Solution

    Uma FDA é não decrescente e contínua à direita, com \(\lim_{x\to-\infty}F(x)=0\) e \(\lim_{x\to\infty}F(x)=1\). Ela pode ter saltos, trechos constantes e não precisa possuir densidade.


  6. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,9\). Qual valor normaliza a pmf?


    1. \(0.11111\)
    2. \(0.01111\)
    3. \(0.90000\)
    4. \(0.02222\)
    5. \(0.10000\)

    Solution

    Deve valer \[1=\sum_{x=1}^{9}cx=c\frac{9(10)}2,\] portanto \(c=2/[9(10)]=0.02222\).


  7. Question

    Uma função é dada por \(f(x)=c x^{3}\) para \(0<x<5\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.00160\)
    2. \(0.20000\)
    3. \(0.03200\)
    4. \(0.00640\)
    5. \(1.25000\)

    Solution

    A normalização exige \[1=c\int_0^{5}x^{3}dx =c\frac{5^{4}}{4}.\] Logo, \(c=(4)/5^{4}=0.00640\).


  8. Question

    Uma variável discreta assume valores \((-2, 1, 3, 5)\) com probabilidades \((0.125, 0.188, 0.188, 0.500)\). Qual é \(E(X)\)?


    1. \(3.5000\)
    2. \(7.0000\)
    3. \(3.0000\)
    4. \(1.0000\)
    5. \(1.7500\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=3.0000.\]


  9. Question

    Se \(E(X)=-2\) e \(E(X^2)=12\), qual é \(\operatorname{Var}(X)\)?


    1. \(3.464\)
    2. \(14.000\)
    3. \(4.000\)
    4. \(16.000\)
    5. \(8.000\)

    Solution

    \[\operatorname{Var}(X)=E(X^2)-[E(X)]^2 =12-(-2)^2=8.000.\]


  10. Question

    Sabe-se que \(E(|X|^{5})<\infty\). Qual conclusão é garantida?


    1. \(X\) possui suporte limitado.
    2. \(E(|X|^{4})<\infty\).
    3. \(E(|X|^{6})<\infty\).
    4. \(X\) tem distribuição simétrica.
    5. A fgm de \(X\) existe para todo \(t\).

    Solution

    A existência de um momento absoluto de ordem \(m\) implica a existência de todos os momentos absolutos de ordens menores. Isso decorre de \(|x|^k\leq1+|x|^m\) para \(k\leq m\).
