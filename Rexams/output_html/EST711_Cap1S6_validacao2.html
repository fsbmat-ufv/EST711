Exam 2

  1. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,7\). Qual valor normaliza a pmf?


    1. \(0.87500\)
    2. \(0.14286\)
    3. \(0.12500\)
    4. \(0.03571\)
    5. \(0.01786\)

    Solution

    Deve valer \[1=\sum_{x=1}^{7}cx=c\frac{7(8)}2,\] portanto \(c=2/[7(8)]=0.03571\).


  2. Question

    Em 8 ensaios de Bernoulli independentes, com probabilidade de sucesso 0.6, qual é a probabilidade de exatamente 1 sucessos?


    1. \(0.00098\)
    2. \(4.80000\)
    3. \(0.08958\)
    4. \(0.99214\)
    5. \(0.00786\)

    Solution

    Para \(X\sim\operatorname{Binomial}(8,0.6)\), \[P(X=1)=\binom{8}{1}p^{1}(1-p)^{7} =0.00786.\]


  3. Question

    Se \(X\) é o número de tentativas independentes até o primeiro sucesso, com probabilidade de sucesso \(p=0.2\), calcule \(P(X=7)\).


    1. \(0.79028\)
    2. \(0.05243\)
    3. \(0.00005\)
    4. \(0.20972\)
    5. \(0.00001\)

    Solution

    Para o primeiro sucesso ocorrer na tentativa 7, devem ocorrer 6 fracassos e então um sucesso: \[P(X=7)=(1-p)^{6}p =0.05243.\]


  4. Question

    Uma variável tem suporte \((0,1,2,3,4)\) e pmf correspondente \((0.077, 0.269, 0.231, 0.192, 0.231)\). Qual é \(F_X(2)\)?


    1. \(0.5769\)
    2. \(0.3462\)
    3. \(0.2308\)
    4. \(0.4231\)
    5. \(0.6538\)

    Solution

    A FDA acumula todas as massas até 2: \[F_X(2)=\sum_{x\leq2}p_X(x) =0.5769.\]


  5. Question

    Uma variável \(X\) assume $-5, \(0\) e $5 com probabilidades 0.15, 0.2 e 0.65. Para \(Y=X^2\), qual é \(P(Y=25)\)?


    1. \(0.150\)
    2. \(0.650\)
    3. \(0.800\)
    4. \(0.500\)
    5. \(0.200\)

    Solution

    Os valores \(X=-5\) e \(X=5\) produzem o mesmo valor \(Y=25\). Portanto, somam-se as massas: \[P(Y=25)=0.15+0.65 =0.800.\]


  6. Question

    Uma variável tem suporte \((-2,-1,0,1,2)\) e probabilidades \((0.296, 0.148, 0.259, 0.148, 0.148)\). Se \(Y=|X|\), qual é \(P(Y=2)\)?


    1. \(0.4444\)
    2. \(0.5556\)
    3. \(0.2963\)
    4. \(0.1481\)
    5. \(1.0000\)

    Solution

    As pré-imagens de \(Y=2\) são \(X=-2\) e \(X=2\). Logo, \[P(Y=2)=P(X=-2)+P(X=2) =0.4444.\]


  7. Question

    Para a transformação injetiva \(Y=3X+1\), sabe-se que \(P(X=-2)=0.6\). Qual é \(P(Y=-5)\)?


    1. \(0.200\)
    2. \(0.150\)
    3. \(1.800\)
    4. \(0.400\)
    5. \(0.600\)

    Solution

    Como a transformação é injetiva, \(Y=-5\) ocorre exatamente quando \(X=-2\). Em transformações discretas não aparece fator jacobiano: \[P(Y=-5)=0.6.\]


  8. Question

    De 25 itens, 5 são especiais. Uma amostra de 4 itens é retirada sem reposição. Qual é a probabilidade de obter exatamente 1 itens especiais?


    1. \(5700.00000\)
    2. \(0.00040\)
    3. \(0.40960\)
    4. \(0.45059\)
    5. \(0.54941\)

    Solution

    A contagem é hipergeométrica: \[P(X=1)= \frac{\binom{5}{1}\binom{20}{3}} {\binom{25}{4}} =0.45059.\]


  9. Question

    Se \(X\sim\operatorname{Poisson}(\lambda)\), com \(\lambda=4.5\), qual é a razão \(p_X(8)/p_X(7)\)?


    1. \(1.7778\)
    2. \(8.0000\)
    3. \(0.5000\)
    4. \(0.5625\)
    5. \(4.5000\)

    Solution

    Pela pmf de Poisson, \[\frac{p(x+1)}{p(x)} =\frac{\lambda^{x+1}/(x+1)!}{\lambda^x/x!} =\frac{\lambda}{x+1} =0.5625.\]


  10. Question

    Para \(X=\mathbf{1}_A\) com \(P(A)=0.9\), qual é \(p_X(0)\)?


    1. \(0.100\)
    2. \(0.010\)
    3. \(0.900\)
    4. \(0.810\)
    5. \(0.090\)

    Solution

    A indicadora tem distribuição Bernoulli: \(P(X=1)=p\) e \(P(X=0)=1-p\). Assim, \(p_X(0)=0.100\).
