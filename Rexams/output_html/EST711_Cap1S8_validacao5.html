Exam 5

  1. Question

    Uma variável discreta assume valores \((-1, 0, 2, 5)\) com probabilidades \((0.111, 0.444, 0.111, 0.333)\). Qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(1.5000\)
    3. \(2.0000\)
    4. \(6.0000\)
    5. \(1.7778\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=1.7778.\]


  2. Question

    Se $f(x)=(3)x^{2}/343 para \(0<x<7\), qual é \(E(X)\)?


    1. \(5.2500\)
    2. \(9.3333\)
    3. \(0.7500\)
    4. \(3.5000\)
    5. \(1.7500\)

    Solution

    \[E(X)=\int_0^{7}x\frac{(3)x^{2}}{343}dx =7\frac{3}{4} =5.2500.\]


  3. Question

    Para valores \((-2, -1, 0, 5)\) com probabilidades \((0.071, 0.357, 0.357, 0.214)\), determine \(E(X^2)\) sem obter a distribuição de \(Y=X^2\).


    1. \(9.6667\)
    2. \(1.5714\)
    3. \(6.0000\)
    4. \(0.3265\)
    5. \(0.5714\)

    Solution

    Pela regra da esperança de uma função, \[E(X^2)=\sum_xx^2p_X(x)=6.0000.\]


  4. Question

    Se \(E(X)=2\), qual é \(E(-2X+5)\)?


    1. \(1.000\)
    2. \(-4.000\)
    3. \(7.000\)
    4. \(13.000\)
    5. \(-14.000\)

    Solution

    A linearidade fornece \[E(aX+b)=aE(X)+b=(-2)(2)+5 =1.000.\]


  5. Question

    Um jogo paga \((5, 10, 0)\) unidades monetárias com probabilidades \((0.400, 0.200, 0.400)\). Qual preço torna o ganho líquido esperado do jogador igual a zero?


    1. \(15.00\)
    2. \(10.00\)
    3. \(2.00\)
    4. \(5.00\)
    5. \(4.00\)

    Solution

    Se \(R\) é o prêmio e \(c\) o preço, o ganho é \(G=R-c\). Um jogo justo satisfaz \(E(G)=E(R)-c=0\), logo \(c=E(R)=4.00\).


  6. Question

    Se \(I_A=\mathbf{1}_A\) e \(P(A)=0.4\), qual é \(E(I_A)\)?


    1. \(0.400\)
    2. \(0.160\)
    3. \(0.600\)
    4. \(0.240\)
    5. \(1.000\)

    Solution

    A indicadora vale 1 em \(A\) e 0 fora de \(A\): \[E(I_A)=1P(A)+0P(A^c)=P(A)=0.4.\]


  7. Question

    Se \(X\) tem densidade exponencial \(f(x)=\lambda e^{-\lambda x}\), \(x>0\), com \(\lambda=3\), qual é \(E(e^{-0.5X})\)?


    1. \(0.2231\)
    2. \(6.0000\)
    3. \(0.1429\)
    4. \(0.8571\)
    5. \(0.2857\)

    Solution

    Pela regra de transformação dentro da esperança, \[E(e^{-tX})=\int_0^\infty e^{-tx}\lambda e^{-\lambda x}dx =\frac{\lambda}{\lambda+t} =0.8571.\]


  8. Question

    Considere \(f(x)=1/x^2\) para \(x>1\) e zero caso contrário. Qual afirmação sobre \(E(X)\) é correta?


    1. A esperança vale 1, pois a densidade integra 1.
    2. A esperança não existe como número finito, pois \(\int_1^\infty x/x^2\,dx=\infty\).
    3. A esperança existe somente após restringir o cálculo a \(X>4\).
    4. A esperança vale 2, pela simetria da densidade.
    5. A esperança é zero, pois a cauda tende a zero.

    Solution

    Embora \(f\) seja uma densidade, o primeiro momento absoluto diverge: \[E|X|=\int_1^\infty\frac1x\,dx=\infty.\] Normalização da densidade não garante existência da média.


  9. Question

    Uma variável geométrica \(X\) conta tentativas até o primeiro sucesso, com probabilidade \(p=0.65\). Usando \(E(X)=\sum_{k\geq0}P(X>k)\), qual é \(E(X)\)?


    1. \(0.6500\)
    2. \(1.5385\)
    3. \(0.5385\)
    4. \(1.8571\)
    5. \(0.3500\)

    Solution

    Como \(P(X>k)=(1-p)^k\), \[E(X)=\sum_{k=0}^{\infty}(1-p)^k=\frac1p =1.5385.\]


  10. Question

    Uma mistura escolhe a população 1 com probabilidade 0.2 e a população 2 com probabilidade 0.8. As médias são \(\mu_1=3\) e \(\mu_2=9\). Qual é a média da mistura?


    1. \(6.000\)
    2. \(7.800\)
    3. \(-6.800\)
    4. \(0.600\)
    5. \(12.000\)

    Solution

    Pela esperança total, \[E(X)=0.2\mu_1+0.8\mu_2 =7.800.\]
