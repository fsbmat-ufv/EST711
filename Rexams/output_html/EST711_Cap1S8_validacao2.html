  1. Question

    Uma variável discreta assume valores \((-2, -1, 0, 5)\) com probabilidades \((0.125, 0.188, 0.250, 0.438)\). Qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(0.5000\)
    3. \(2.6250\)
    4. \(2.0000\)
    5. \(1.7500\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=1.7500.\]


  2. Question

    Se $f(x)=(6)x^{5}/1.5625 × 104 para \(0<x<5\), qual é \(E(X)\)?


    1. \(0.7143\)
    2. \(2.5000\)
    3. \(5.8333\)
    4. \(4.2857\)
    5. \(0.8571\)

    Solution

    \[E(X)=\int_0^{5}x\frac{(6)x^{5}}{1.5625 &times; 10<sup>4</sup>}dx =5\frac{6}{7} =4.2857.\]


  3. Question

    Para valores \((-4, -1, 0, 2)\) com probabilidades \((0.211, 0.211, 0.263, 0.316)\), determine \(E(X^2)\) sem obter a distribuição de \(Y=X^2\).


    1. \(6.2500\)
    2. \(-0.4211\)
    3. \(0.1773\)
    4. \(4.8421\)
    5. \(1.6842\)

    Solution

    Pela regra da esperança de uma função, \[E(X^2)=\sum_xx^2p_X(x)=4.8421.\]


  4. Question

    Se \(E(X)=1.5\), qual é \(E(2X+2)\)?


    1. \(3.000\)
    2. \(3.500\)
    3. \(7.000\)
    4. \(8.000\)
    5. \(5.000\)

    Solution

    A linearidade fornece \[E(aX+b)=aE(X)+b=(2)(1.5)+2 =5.000.\]


  5. Question

    Um jogo paga \((20, 10, 50)\) unidades monetárias com probabilidades \((0.125, 0.438, 0.438)\). Qual preço torna o ganho líquido esperado do jogador igual a zero?


    1. \(26.67\)
    2. \(50.00\)
    3. \(14.38\)
    4. \(28.75\)
    5. \(80.00\)

    Solution

    Se \(R\) é o prêmio e \(c\) o preço, o ganho é \(G=R-c\). Um jogo justo satisfaz \(E(G)=E(R)-c=0\), logo \(c=E(R)=28.75\).


  6. Question

    Se \(I_A=\mathbf{1}_A\) e \(P(A)=0.3\), qual é \(E(I_A)\)?


    1. \(0.210\)
    2. \(1.000\)
    3. \(0.700\)
    4. \(0.300\)
    5. \(0.090\)

    Solution

    A indicadora vale 1 em \(A\) e 0 fora de \(A\): \[E(I_A)=1P(A)+0P(A^c)=P(A)=0.3.\]


  7. Question

    Se \(X\) tem densidade exponencial \(f(x)=\lambda e^{-\lambda x}\), \(x>0\), com \(\lambda=3.5\), qual é \(E(e^{-0.5X})\)?


    1. \(7.0000\)
    2. \(0.1738\)
    3. \(0.1250\)
    4. \(0.8750\)
    5. \(0.2500\)

    Solution

    Pela regra de transformação dentro da esperança, \[E(e^{-tX})=\int_0^\infty e^{-tx}\lambda e^{-\lambda x}dx =\frac{\lambda}{\lambda+t} =0.8750.\]


  8. Question

    Considere \(f(x)=1/x^2\) para \(x>1\) e zero caso contrário. Qual afirmação sobre \(E(X)\) é correta?


    1. A esperança vale 2, pela simetria da densidade.
    2. A esperança não existe como número finito, pois \(\int_1^\infty x/x^2\,dx=\infty\).
    3. A esperança é zero, pois a cauda tende a zero.
    4. A esperança existe somente após restringir o cálculo a \(X>3\).
    5. A esperança vale 1, pois a densidade integra 1.

    Solution

    Embora \(f\) seja uma densidade, o primeiro momento absoluto diverge: \[E|X|=\int_1^\infty\frac1x\,dx=\infty.\] Normalização da densidade não garante existência da média.


  9. Question

    Uma variável geométrica \(X\) conta tentativas até o primeiro sucesso, com probabilidade \(p=0.45\). Usando \(E(X)=\sum_{k\geq0}P(X>k)\), qual é \(E(X)\)?


    1. \(1.2222\)
    2. \(0.4500\)
    3. \(0.8182\)
    4. \(0.5500\)
    5. \(2.2222\)

    Solution

    Como \(P(X>k)=(1-p)^k\), \[E(X)=\sum_{k=0}^{\infty}(1-p)^k=\frac1p =2.2222.\]


  10. Question

    Uma mistura escolhe a população 1 com probabilidade 0.3 e a população 2 com probabilidade 0.7. As médias são \(\mu_1=2.5\) e \(\mu_2=7\). Qual é a média da mistura?


    1. \(4.750\)
    2. \(-4.650\)
    3. \(0.750\)
    4. \(9.500\)
    5. \(5.650\)

    Solution

    Pela esperança total, \[E(X)=0.3\mu_1+0.7\mu_2 =5.650.\]
