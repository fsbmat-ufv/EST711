  1. Question

    Seja X1,X2,…X_1, X_2, \ldots uma sequência de variáveis aleatórias iid com E(Xi)=μ=2E(X_i) = \mu = 2 e Var(Xi)=σ2=9\text{Var}(X_i) = \sigma^2 = 9 finita. Qual das afirmações abaixo enuncia corretamente o Teorema Central do Limite (TCL)?


    1. n(X‾n−μ)→DN(0,1)\sqrt{n}(\bar{X}_n - \mu) \xrightarrow{D} N(0,1) (sem dividir por σ\sigma).
    2. X‾n−μσ→DN(0,1)\dfrac{\bar{X}_n - \mu}{\sigma} \xrightarrow{D} N(0,1) (sem o fator n\sqrt{n}).
    3. n(X‾n−μ)σ2→DN(0,1)\dfrac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma^2} \xrightarrow{D} N(0,1) (usando σ2\sigma^2 em vez de σ\sigma).
    4. X‾n→DN(μ,σ2)\bar{X}_n \xrightarrow{D} N(\mu, \sigma^2) para todo nn suficientemente grande.
    5. n(X‾n−μ)σ→DN(0,1)\dfrac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{D} N(0,1).

    Solution

    O TCL afirma que n(X‾n−μ)σ→DN(0,1)\dfrac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{D} N(0,1).

    Os erros comuns são: (i) esquecer o fator n\sqrt{n} (sem ele, (X‾n−μ)/σ(\bar{X}_n - \mu)/\sigma converge a 0 em probabilidade); (ii) usar σ2\sigma^2 em vez de σ\sigma no denominador; (iii) dizer que X‾n\bar{X}_n converge em distribuição para N(μ,σ2)N(\mu,\sigma^2) — na verdade, é a estatística padronizada que converge para N(0,1)N(0,1).

    3. χ12\chi^2_1 (qui-quadrado com 1 grau de liberdade).

  8. Question

    Pelo TCL, para nn grande, X‾n\bar{X}_n tem distribuição aproximadamente N(μ,σ2/n)=N(3,0.4444)N\!\left(\mu, \sigma^2/n\right) = N\!\left(3, 0.4444\right) quando μ=3\mu = 3, σ=4\sigma = 4 e n=36n = 36. Qual afirmação sobre essa aproximação é correta?


    1. A aproximação melhora à medida que nn aumenta, e a taxa de convergência depende do grau de não-normalidade da distribuição original.
    2. A aproximação é exata quando a distribuição original é exponencial.
    3. A aproximação normal só é válida para distribuições simétricas.
    4. A aproximação é exata para qualquer n≥30n \geq 30, independentemente da distribuição original.
    5. O TCL garante exatidão da aproximação normal para todo nn, independentemente da distribuição.

    Solution

    A afirmação correta é que a qualidade da aproximação normal melhora com nn, e a taxa de convergência depende da assimetria e curtose da distribuição original (Teorema de Berry-Esseen formaliza isso).

    O critério empírico “n≥30n \geq 30” não tem respaldo teórico universal: para distribuições muito assimétricas ou com caudas pesadas, são necessários nn muito maiores. A aproximação não é exata para nenhuma distribuição não-normal (apenas para normais, onde X‾n\bar{X}_n é exatamente normal para qualquer nn).

  11. Question

    Suponha que n(X‾n−μ)→DN(0,σ2)\sqrt{n}(\bar{X}_n - \mu) \xrightarrow{D} N(0, \sigma^2) e que g:ℝ→ℝg : \mathbb{R} \to \mathbb{R} é diferenciável com g′(μ)≠0g'(\mu) \neq 0. O Método Delta (Delta Method) afirma que:


    1. n(g(X‾n)−g(μ))→DN(0,[g′(μ)]2σ2)\sqrt{n}(g(\bar{X}_n) - g(\mu)) \xrightarrow{D} N(0,\, [g'(\mu)]^2 \sigma^2).

    3. n(g(X‾n)−g(μ))→DN(0,[g(μ)]2σ2)\sqrt{n}(g(\bar{X}_n) - g(\mu)) \xrightarrow{D} N(0,\, [g(\mu)]^2 \sigma^2) (usando g(μ)g(\mu) em vez de g′(μ)g'(\mu)).
    4. n(g(X‾n)−g(μ))→DN(0,g′(μ)σ2)\sqrt{n}(g(\bar{X}_n) - g(\mu)) \xrightarrow{D} N(0,\, g'(\mu)\, \sigma^2) (sem elevar ao quadrado).
    5. n(g(X‾n)−g(μ))→DN(0,[g′(μ)σ]2/n)\sqrt{n}(g(\bar{X}_n) - g(\mu)) \xrightarrow{D} N(0,\, [g'(\mu)\,\sigma]^2/n) (variância dividida por nn erroneamente).

  19. Question

    O Método Delta de primeira ordem requer que g′(μ)≠0g'(\mu) \neq 0 no ponto de expansão μ\mu. Por que essa condição é necessária?


    1. Se g′(μ)=0g'(\mu) = 0, a transformação gg não é invertível em μ\mu e portanto não faz sentido aplicar o método.
    2. Se g′(μ)=0g'(\mu) = 0, o método delta não pode ser aplicado pois gg não é diferenciável nesse ponto.
    3. Se g′(μ)=0g'(\mu) = 0, a variância assintótica de g(X‾n)g(\bar{X}_n) torna-se infinita.
    4. Se g′(μ)=0g'(\mu) = 0, o método delta de primeira ordem dá variância assintótica zero, sendo necessário usar o método delta de segunda ordem.
    5. Se g′(μ)=0g'(\mu) = 0, g(X‾n)g(\bar{X}_n) não converge em nenhum sentido estocástico.

    Solution

    Quando g′(μ)=0g'(\mu) = 0, o método delta de primeira ordem produz variância assintótica [g′(μ)]2σ2=0[g'(\mu)]^2\sigma^2 = 0, o que é degenerado e não informativo. Nesse caso, a taxa de convergência é diferente da usual n\sqrt{n} e é necessário o método delta de segunda ordem:

    n(g(X‾n)−g(μ))→Dg″(μ)2σ2χ12n(g(\bar{X}_n) - g(\mu)) \xrightarrow{D} \frac{g''(\mu)}{2}\sigma^2\chi^2_1

    (quando g′(μ)=0g'(\mu) = 0 e g″(μ)≠0g''(\mu) \neq 0).

    Note que g′(μ)=0g'(\mu) = 0 não implica que gg não seja diferenciável — pelo contrário, gg é diferenciável, mas a derivada se anula. Também não implica que gg seja não-invertível ou que g(X‾n)g(\bar{X}_n) divirja.
