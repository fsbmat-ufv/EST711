  1. Question

    Seja Xi∼iidN(μ,σ02)X_i \overset{\text{iid}}{\sim} N(\mu, \sigma_0^2) com σ02=1\sigma_0^2 = 1 conhecido. Com base em uma amostra de tamanho n=100n = 100, o estimador de μ\mu pelo método dos momentos é:


    1. o momento amostral de maior ordem disponível.
    2. o momento de segunda ordem m2=1n∑Xi2m_2 = \frac{1}{n}\sum X_i^2.
    3. o estimador que minimiza ∑(Xi−θ)2\sum (X_i - \theta)^2.
    4. o primeiro momento amostral m1=X‾nm_1 = \bar{X}_n, que iguala o momento teórico μ1=μ\mu_1 = \mu.
    5. a mediana amostral, pois é mais robusta que a média.

    Solution

    O método dos momentos consiste em igualar os momentos teóricos da distribuição aos momentos amostrais. Para X∼N(μ,σ02)X \sim N(\mu, \sigma_0^2), o primeiro momento teórico é μ1=E(X)=μ\mu_1 = E(X) = \mu. Igualando ao primeiro momento amostral m1=X‾nm_1 = \bar{X}_n, obtemos μ̂MM=X‾n\hat{\mu}_{MM} = \bar{X}_n. Não é necessário usar momentos de ordem superior nem σ02\sigma_0^2, pois o parâmetro de interesse μ\mu já aparece no primeiro momento.

  7. Question

    Qual propriedade geralmente garante que estimadores pelo método dos momentos (EMM) são consistentes?


    1. Os EMM são sempre não-viesados, o que implica consistência.
    2. Os EMM são consistentes apenas se a distribuição for simétrica.
    3. Pela Lei dos Grandes Números, os momentos amostrais mk=1n∑Xikm_k = \frac{1}{n}\sum X_i^k convergem em probabilidade para os momentos teóricos μk\mu_k. Se os parâmetros são funções contínuas dos momentos, os EMM são consistentes.
    4. Os EMM minimizam o erro quadrático médio, garantindo consistência.
    5. A consistência dos EMM segue apenas para distribuições da família exponencial.

    Solution

    A consistência dos EMM é garantida pela Lei dos Grandes Números (LGN): para XiX_i i.i.d. com E|Xk|<∞E|X^k| < \infty, temos mk=1n∑Xik→Pμk=E(Xk)m_k = \frac{1}{n}\sum X_i^k \xrightarrow{P} \mu_k = E(X^k). Se θ̂=g(m1,…,mk)\hat{\theta} = g(m_1, \ldots, m_k) e gg é uma função contínua, então pelo teorema de mapeamento contínuo θ̂→Pg(μ1,…,μk)=θ\hat{\theta} \xrightarrow{P} g(\mu_1, \ldots, \mu_k) = \theta. Não-viesamento não implica consistência em geral; e a LGN não exige simetria nem família exponencial.


  8. Question

    Qual afirmação descreve corretamente a diferença entre o método dos momentos (MM) e o método da máxima verossimilhança (MV)?


    1. O MM maximiza a log-verossimilhança, enquanto o MV minimiza a soma de quadrados.
    2. O MM usa todos os dados igualmente, enquanto o MV pondera cada observação pela sua verossimilhança.
    3. O MM iguala momentos amostrais a momentos teóricos; o MV maximiza a função de verossimilhança. Para muitas distribuições, os dois coincdem, mas em geral o EMV tem melhores propriedades assintóticas.
    4. O MM é sempre mais eficiente que o MV para amostras pequenas.
    5. Ambos os métodos sempre produzem os mesmos estimadores para qualquer distribuição.

    Solution

    O método dos momentos (MM) é baseado em igualar momentos populacionais a momentos amostrais. O método da máxima verossimilhança (MV) maximiza L(θ;𝐱)=∏f(xi;θ)L(\theta; \mathbf{x}) = \prod f(x_i; \theta). Embora para algumas distribuições (como Poisson e Normal) os dois métodos produzam estimadores idênticos, em geral divergem. O EMV possui melhores propriedades assintóticas: é consistente, assintoticamente normal e assintoticamente eficiente (atinge o limite de Cramér-Rao). O MM é computacionalmente mais simples, mas pode ser menos eficiente.

  10. Question

    Para Xi∼iidPoisson(λ)X_i \overset{\text{iid}}{\sim} \text{Poisson}(\lambda), por que o estimador do método dos momentos (EMM) e o estimador de máxima verossimilhança (EMV) são iguais?


    1. Porque o EMV e o EMM são sempre idênticos para distribuições com suporte em {0,1,2,…}\{0, 1, 2, \ldots\}.
    2. Porque E(X)=λE(X) = \lambda, logo o EMM é λ̂MM=X‾\hat{\lambda}_{MM} = \bar{X}. A log-verossimilhança ℓ(λ)=∑Xilog⁡λ−nλ+const\ell(\lambda) = \sum X_i \log\lambda - n\lambda + \text{const} também tem máximo em λ̂MV=X‾\hat{\lambda}_{MV} = \bar{X}. Ambos coincidem.
    3. Porque a Poisson é uma distribuição simétrica, o que garante a igualdade entre MM e MV.
    4. Porque a distribuição de Poisson pertence à família exponencial, e nessa família os dois métodos sempre coincidem.
    5. Por coincidência particular; em geral, o MM e o MV são sempre diferentes para qualquer distribuição.

    Solution

    Para Xi∼Poisson(λ)X_i \sim \text{Poisson}(\lambda): (i) o EMM iguala μ1=E(X)=λ\mu_1 = E(X) = \lambda a m1=X‾m_1 = \bar{X}, obtendo λ̂MM=X‾\hat{\lambda}_{MM} = \bar{X}; (ii) a log-verossimilhança é ℓ(λ)=(∑Xi)log⁡λ−nλ+const\ell(\lambda) = (\sum X_i)\log\lambda - n\lambda + \text{const}, e dℓ/dλ=∑Xi/λ−n=0d\ell/d\lambda = \sum X_i/\lambda - n = 0 implica λ̂MV=X‾\hat{\lambda}_{MV} = \bar{X}. A coincidência não ocorre para todos os membros da família exponencial (por exemplo, para a Gama com dois parâmetros os dois métodos também coincidem, mas isso precisa de verificação caso a caso). O argumento correto é a derivação direta de ambos os estimadores.

  17. Question

    Qual propriedade do EMV é enunciada por: “Se θ̂\hat{\theta} é o EMV de θ\theta, então g(θ̂)g(\hat{\theta}) é o EMV de g(θ)g(\theta)”?


    1. Propriedade de invariância (ou equivariância) do EMV: se θ̂\hat{\theta} é o EMV de θ\theta, então g(θ̂)g(\hat{\theta}) é o EMV de g(θ)g(\theta), para qualquer função gg.
    2. Suficiência: o EMV é sempre função de estatísticas suficientes para o parâmetro.
    3. Não-viesamento: o EMV é não-viesado para qualquer transformação g(θ)g(\theta).
    4. Consistência: o EMV converge em probabilidade para o verdadeiro parâmetro à medida que n→∞n \to \infty.
    5. Eficiência assintótica: o EMV atinge o limite de Cramér-Rao assintoticamente.

    Solution

    A propriedade descrita é a invariância (ou equivariância) do EMV. Formalmente: se θ̂\hat{\theta} maximiza L(θ;𝐱)L(\theta; \mathbf{x}) e gg é uma função (não necessariamente bijetora), então o EMV de g(θ)g(\theta) é g(θ̂)g(\hat{\theta}). Por exemplo, se σ̂2\hat{\sigma}^2 é o EMV de σ2\sigma^2, então σ̂=σ̂2\hat{\sigma} = \sqrt{\hat{\sigma}^2} é o EMV de σ\sigma. As demais propriedades listadas (consistência, eficiência assintótica, suficiência) são propriedades reais do EMV sob condições de regularidade, mas não descrevem a invariância.

  19. Question

    O EMV de σ2\sigma^2 na distribuição Normal é σ̂MV2=1n∑i=1n(Xi−X‾)2\hat{\sigma}^2_{MV} = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^2. Para uma amostra de tamanho n=50n = 50, qual é o vício deste estimador?


    1. O EMV de σ2\sigma^2 é viesado apenas para amostras pequenas, sendo não-viesado para n≥30n \geq 30.
    2. O EMV de σ2\sigma^2 é não-viesado, pois o método da máxima verossimilhança sempre produz estimadores não-viesados.
    3. O vício é −σ2/(n−1)-\sigma^2/(n-1), pois o denominador correto seria n−1n-1 para n=50n = 50.
    4. O vício é +σ2/n+\sigma^2/n, pois o EMV tende a superestimar a variância populacional.
    5. E(σ̂MV2)=n−1nσ2E(\hat{\sigma}^2_{MV}) = \frac{n-1}{n}\sigma^2, portanto o vício é E(σ̂MV2)−σ2=−σ2nE(\hat{\sigma}^2_{MV}) - \sigma^2 = -\frac{\sigma^2}{n}. Para n=50n = 50, o vício é −σ2/50-\sigma^2/50.

    Solution

    Sabe-se que (n−1)S2σ2∼χn−12\frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1}, onde S2=1n−1∑(Xi−X‾)2S^2 = \frac{1}{n-1}\sum(X_i-\bar{X})^2. Portanto E(S2)=σ2E(S^2) = \sigma^2. Como σ̂MV2=n−1nS2\hat{\sigma}^2_{MV} = \frac{n-1}{n}S^2, temos: E(σ̂MV2)=n−1nσ2.E(\hat{\sigma}^2_{MV}) = \frac{n-1}{n}\sigma^2. O vício é E(σ̂MV2)−σ2=−σ2nE(\hat{\sigma}^2_{MV}) - \sigma^2 = -\frac{\sigma^2}{n}, que é negativo (subestimação). Para n=50n = 50, o vício é −σ2/50-\sigma^2/50. O EMV é assintoticamente não-viesado (o vício →0\to 0 quando n→∞n \to \infty), mas é viesado para todo nn finito.


  20. Question

    Para Xi∼iidPoisson(λ)X_i \overset{\text{iid}}{\sim} \text{Poisson}(\lambda), o score (derivada da log-verossimilhança) é dℓdλ=∑Xiλ−n\frac{d\ell}{d\lambda} = \frac{\sum X_i}{\lambda} - n. Com n=25n = 25 e ∑i=1nXi=99\sum_{i=1}^n X_i = 99, qual das alternativas descreve corretamente a equação de verossimilhança e sua solução?


    1. A log-verossimilhança da Poisson é maximizada em λ̂MV=n/∑Xi=0.2525\hat{\lambda}_{MV} = n/\sum X_i = 0.2525.
    2. A equação de verossimilhança resulta em λ̂MV=∑Xi=99\hat{\lambda}_{MV} = \sum X_i = 99.
    3. A equação de verossimilhança dℓdλ=0\frac{d\ell}{d\lambda} = 0 resulta em λ̂MV=X‾=3.96\hat{\lambda}_{MV} = \bar{X} = 3.96.
    4. O score para Poisson é ℓ′(λ)=∑Xi⋅log⁡λ−n\ell'(\lambda) = \sum X_i \cdot \log\lambda - n, e a equação de verossimilhança não possui solução fechada.
    5. A equação de verossimilhança é resolvida por λ̂MV=1/X‾=0.2525\hat{\lambda}_{MV} = 1/\bar{X} = 0.2525.

    Solution

    A log-verossimilhança para Xi∼Poisson(λ)X_i \sim \text{Poisson}(\lambda) é: ℓ(λ)=(∑i=1nXi)log⁡λ−nλ+const.\ell(\lambda) = \left(\sum_{i=1}^n X_i\right)\log\lambda - n\lambda + \text{const}. O score é ℓ′(λ)=∑Xi/λ−n\ell'(\lambda) = \sum X_i/\lambda - n. Igualando a zero: ∑Xiλ−n=0⟹λ̂MV=∑Xin=9925=3.96.\frac{\sum X_i}{\lambda} - n = 0 \implies \hat{\lambda}_{MV} = \frac{\sum X_i}{n} = \frac{99}{25} = 3.96. A segunda derivada ℓ″(λ)=−∑Xi/λ2<0\ell''(\lambda) = -\sum X_i/\lambda^2 < 0 confirma que é máximo. O score correto é ∑Xi/λ−n\sum X_i/\lambda - n, não ∑Xi⋅log⁡λ−n\sum X_i \cdot \log\lambda - n.
