  1. Question

    Uma variável discreta assume valores \((-3, -2, 1, 2)\) com probabilidades \((0.056, 0.444, 0.389, 0.111)\). Qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(1.6667\)
    3. \(-0.4444\)
    4. \(-2.0000\)
    5. \(-0.5000\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=-0.4444.\]


  2. Question

    Se $f(x)=(3)x^{2}/64 para \(0<x<4\), qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(2.0000\)
    3. \(3.0000\)
    4. \(0.7500\)
    5. \(5.3333\)

    Solution

    \[E(X)=\int_0^{4}x\frac{(3)x^{2}}{64}dx =4\frac{3}{4} =3.0000.\]


  3. Question

    Para valores \((-3, -2, -1, 4)\) com probabilidades \((0.059, 0.412, 0.059, 0.471)\), determine \(E(X^2)\) sem obter a distribuição de \(Y=X^2\).


    1. \(0.6782\)
    2. \(9.7647\)
    3. \(2.9412\)
    4. \(0.8235\)
    5. \(9.6667\)

    Solution

    Pela regra da esperança de uma função, \[E(X^2)=\sum_xx^2p_X(x)=9.7647.\]


  4. Question

    Se \(E(X)=0.5\), qual é \(E(2X+-3)\)?


    1. \(-5.000\)
    2. \(-2.000\)
    3. \(-1.000\)
    4. \(1.000\)
    5. \(-2.500\)

    Solution

    A linearidade fornece \[E(aX+b)=aE(X)+b=(2)(0.5)+-3 =-2.000.\]


  5. Question

    Um jogo paga \((0, 20, 50)\) unidades monetárias com probabilidades \((0.091, 0.455, 0.455)\). Qual preço torna o ganho líquido esperado do jogador igual a zero?


    1. \(31.82\)
    2. \(70.00\)
    3. \(23.33\)
    4. \(15.91\)
    5. \(50.00\)

    Solution

    Se \(R\) é o prêmio e \(c\) o preço, o ganho é \(G=R-c\). Um jogo justo satisfaz \(E(G)=E(R)-c=0\), logo \(c=E(R)=31.82\).


  6. Question

    Se \(I_A=\mathbf{1}_A\) e \(P(A)=0.9\), qual é \(E(I_A)\)?


    1. \(0.810\)
    2. \(0.900\)
    3. \(0.100\)
    4. \(0.090\)
    5. \(1.000\)

    Solution

    A indicadora vale 1 em \(A\) e 0 fora de \(A\): \[E(I_A)=1P(A)+0P(A^c)=P(A)=0.9.\]


  7. Question

    Se \(X\) tem densidade exponencial \(f(x)=\lambda e^{-\lambda x}\), \(x>0\), com \(\lambda=2\), qual é \(E(e^{-3X})\)?


    1. \(0.6000\)
    2. \(0.0025\)
    3. \(0.4000\)
    4. \(0.6667\)
    5. \(0.2000\)

    Solution

    Pela regra de transformação dentro da esperança, \[E(e^{-tX})=\int_0^\infty e^{-tx}\lambda e^{-\lambda x}dx =\frac{\lambda}{\lambda+t} =0.4000.\]


  8. Question

    Considere \(f(x)=1/x^2\) para \(x>1\) e zero caso contrário. Qual afirmação sobre \(E(X)\) é correta?


    1. A esperança vale 2, pela simetria da densidade.
    2. A esperança existe somente após restringir o cálculo a \(X>3\).
    3. A esperança não existe como número finito, pois \(\int_1^\infty x/x^2\,dx=\infty\).
    4. A esperança vale 1, pois a densidade integra 1.
    5. A esperança é zero, pois a cauda tende a zero.

    Solution

    Embora \(f\) seja uma densidade, o primeiro momento absoluto diverge: \[E|X|=\int_1^\infty\frac1x\,dx=\infty.\] Normalização da densidade não garante existência da média.


  9. Question

    Uma variável geométrica \(X\) conta tentativas até o primeiro sucesso, com probabilidade \(p=0.4\). Usando \(E(X)=\sum_{k\geq0}P(X>k)\), qual é \(E(X)\)?


    1. \(0.6667\)
    2. \(0.4000\)
    3. \(0.6000\)
    4. \(1.5000\)
    5. \(2.5000\)

    Solution

    Como \(P(X>k)=(1-p)^k\), \[E(X)=\sum_{k=0}^{\infty}(1-p)^k=\frac1p =2.5000.\]


  10. Question

    Uma mistura escolhe a população 1 com probabilidade 0.35 e a população 2 com probabilidade 0.65. As médias são \(\mu_1=-2\) e \(\mu_2=8.5\). Qual é a média da mistura?


    1. \(4.825\)
    2. \(6.500\)
    3. \(-3.825\)
    4. \(-0.700\)
    5. \(3.250\)

    Solution

    Pela esperança total, \[E(X)=0.35\mu_1+0.65\mu_2 =4.825.\]
