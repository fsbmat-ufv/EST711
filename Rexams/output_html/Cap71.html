  1. Question

    Sob condições de regularidade (Cramér), o Estimador de Máxima Verossimilhança 𝛉̂n\hat{\boldsymbol{\theta}}_n para 𝛉∈ℝk\boldsymbol{\theta} \in \mathbb{R}^k tem distribuição assintótica. Qual das alternativas abaixo descreve corretamente essa distribuição?


    1. O EMV multivariado não tem distribuição assintótica conhecida no caso geral.
    2. n(𝛉̂n−𝛉)→DNk(𝟎,I(𝛉)−1)n(\hat{\boldsymbol{\theta}}_n - \boldsymbol{\theta}) \xrightarrow{D} N_k(\mathbf{0}, I(\boldsymbol{\theta})^{-1}), com fator nn em vez de n\sqrt{n}.
    3. n(𝛉̂n−𝛉)→DNk(𝟎,I(𝛉))\sqrt{n}(\hat{\boldsymbol{\theta}}_n - \boldsymbol{\theta}) \xrightarrow{D} N_k(\mathbf{0}, I(\boldsymbol{\theta})), sem inversão da matriz de informação.
    4. n𝛉̂n→DNk(𝛉,I(𝛉)−1)\sqrt{n}\hat{\boldsymbol{\theta}}_n \xrightarrow{D} N_k(\boldsymbol{\theta}, I(\boldsymbol{\theta})^{-1}), centrando em 𝛉\boldsymbol{\theta} e não em zero.
    5. n(𝛉̂n−𝛉)→DNk(𝟎,I(𝛉)−1)\sqrt{n}(\hat{\boldsymbol{\theta}}_n - \boldsymbol{\theta}) \xrightarrow{D} N_k(\mathbf{0}, I(\boldsymbol{\theta})^{-1}), onde I(𝛉)I(\boldsymbol{\theta}) é a matriz de informação de Fisher.

    Solution

    A distribuição assintótica correta do EMV multivariado é: n(𝛉̂n−𝛉)→DNk(𝟎,I(𝛉)−1),\sqrt{n}(\hat{\boldsymbol{\theta}}_n - \boldsymbol{\theta}) \xrightarrow{D} N_k(\mathbf{0}, I(\boldsymbol{\theta})^{-1}), onde I(𝛉)I(\boldsymbol{\theta}) é a matriz de informação de Fisher k×kk \times k avaliada no verdadeiro valor do parâmetro. A fatoração é n\sqrt{n} (não nn), o desvio é centrado em zero, e a matriz de covariâncias assintótica é a inversa de I(𝛉)I(\boldsymbol{\theta}), não a própria I(𝛉)I(\boldsymbol{\theta}).


  2. Question

    Para X∼N(μ,σ2)X \sim N(\mu, \sigma^2) com vetor de parâmetros 𝛉=(μ,σ2)⊤\boldsymbol{\theta} = (\mu, \sigma^2)^\top, a matriz de informação de Fisher é diagonal, com entradas: I11(𝛉)=1σ2,I22(𝛉)=12σ4.I_{11}(\boldsymbol{\theta}) = \frac{1}{\sigma^2}, \quad I_{22}(\boldsymbol{\theta}) = \frac{1}{2\sigma^4}.

    Com σ2=4\sigma^2 = 4, qual é o valor de I11(𝛉)I_{11}(\boldsymbol{\theta})?


    1. 4
    2. 0.0312
    3. 0.0625
    4. 0.5
    5. 0.25

    Solution

    A entrada (1,1)(1,1) da matriz de informação de Fisher para a distribuição Normal é: I11(𝛉)=1σ2.I_{11}(\boldsymbol{\theta}) = \frac{1}{\sigma^2}.

    Com σ2=4\sigma^2 = 4: I11=14=0.25.I_{11} = \frac{1}{4} = 0.25.

    A entrada (2,2)(2,2) seria I22=1/(2σ4)=0.0312I_{22} = 1/(2\sigma^4) = 0.0312, que é a informação sobre σ2\sigma^2, não sobre μ\mu.


  3. Question

    Seja n(𝛉̂−𝛉)→DNk(𝟎,Σ)\sqrt{n}(\hat{\boldsymbol{\theta}} - \boldsymbol{\theta}) \xrightarrow{D} N_k(\mathbf{0}, \Sigma) e g:ℝk→ℝg:\mathbb{R}^k \to \mathbb{R} uma função diferenciável com gradiente ∇g(𝛉)\nabla g(\boldsymbol{\theta}) não-nulo. O Método Delta Multivariado afirma que:


    1. n(g(𝛉̂)−g(𝛉))→DNk(0,g′(𝛉)Σg′(𝛉))\sqrt{n}(g(\hat{\boldsymbol{\theta}}) - g(\boldsymbol{\theta})) \xrightarrow{D} N_k(0, g'(\boldsymbol{\theta})\Sigma g'(\boldsymbol{\theta})) como distribuição kk-dimensional.
    2. n(g(𝛉̂)−g(𝛉))→DN(0,∥∇g(𝛉)∥2∥Σ∥)\sqrt{n}(g(\hat{\boldsymbol{\theta}}) - g(\boldsymbol{\theta})) \xrightarrow{D} N(0, \|\nabla g(\boldsymbol{\theta})\|^2 \|\Sigma\|), usando normas matriciais.
    3. n(g(𝛉̂)−g(𝛉))→DN(0,tr(Σ)∥g′(𝛉)∥2)\sqrt{n}(g(\hat{\boldsymbol{\theta}}) - g(\boldsymbol{\theta})) \xrightarrow{D} N(0, \text{tr}(\Sigma)\|g'(\boldsymbol{\theta})\|^2).
    4. g(𝛉̂)→Pg(𝛉)g(\hat{\boldsymbol{\theta}}) \xrightarrow{P} g(\boldsymbol{\theta}) mas sem distribuição assintótica normal conhecida para gg não-linear.
    5. n(g(𝛉̂)−g(𝛉))→DN(0,∇g(𝛉)⊤Σ∇g(𝛉))\sqrt{n}(g(\hat{\boldsymbol{\theta}}) - g(\boldsymbol{\theta})) \xrightarrow{D} N(0, \nabla g(\boldsymbol{\theta})^\top \Sigma \nabla g(\boldsymbol{\theta})).

    A variância assintótica é a forma quadrática ∇g⊤Σ∇g\nabla g^\top \Sigma \nabla g, onde ∇g\nabla g é o vetor gradiente k×1k \times 1 e Σ\Sigma é a matriz de covariâncias assintótica. O resultado transforma uma distribuição kk-dimensional em uma distribuição escalar normal, com variância que captura a projeção da incerteza de 𝛉̂\hat{\boldsymbol{\theta}} ao longo da direção do gradiente de gg.

    Para Xi∼iidN(μ,σ2)X_i \overset{\text{iid}}{\sim} N(\mu, \sigma^2) com μ\mu e σ2\sigma^2 ambos desconhecidos, os EMVs são μ̂MV=X‾\hat{\mu}_{MV} = \bar{X} e σ̂MV2=1n∑(Xi−X‾)2\hat{\sigma}^2_{MV} = \frac{1}{n}\sum(X_i - \bar{X})^2.

    A variância assintótica de μ̂MV\hat{\mu}_{MV} é dada pela entrada (1,1)(1,1) de I(𝛉)−1/nI(\boldsymbol{\theta})^{-1}/n, que é σ2/n\sigma^2/n.

    Com σ2=9\sigma^2 = 9 e n=36n = 36, qual é a variância assintótica de μ̂MV\hat{\mu}_{MV}?

    2. 0.125

    4. 0.25

  6. Question

    Seja 𝛉=(θ1,𝛌)\boldsymbol{\theta} = (\theta_1, \boldsymbol{\lambda}) onde θ1\theta_1 é o parâmetro de interesse e 𝛌\boldsymbol{\lambda} são os parâmetros de perturbação (nuisance parameters). A verossimilhança de perfil (profile likelihood) para θ1\theta_1 é definida como:


    1. LP(θ1)=∫L(θ1,𝛌)π(𝛌)d𝛌L_P(\theta_1) = \int L(\theta_1, \boldsymbol{\lambda})\, \pi(\boldsymbol{\lambda})\, d\boldsymbol{\lambda} — marginalização bayesiana com prior π(𝛌)\pi(\boldsymbol{\lambda}).
    2. LP(θ1)=sup⁡𝛌L(θ1,𝛌)L_P(\theta_1) = \sup_{\boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda}) — a verossimilhança maximizada sobre os parâmetros de perturbação.
    3. LP(θ1)=L(θ1,𝛌̂)L_P(\theta_1) = L(\theta_1, \hat{\boldsymbol{\lambda}}) onde 𝛌̂\hat{\boldsymbol{\lambda}} é fixado em seu EMV incondicional, ignorando a dependência em θ1\theta_1.
    4. LP(θ1)=L(θ1,𝛌0)L_P(\theta_1) = L(\theta_1, \boldsymbol{\lambda}_0) para algum 𝛌0\boldsymbol{\lambda}_0 fixado a priori e independente dos dados.
    5. LP(θ1)=min⁡𝛌L(θ1,𝛌)L_P(\theta_1) = \min_{\boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda}) — mínimo sobre os parâmetros de perturbação.

    Solution

    A verossimilhança de perfil é definida como: LP(θ1)=sup𝛌L(θ1,𝛌).L_P(\theta_1) = \sup_{\boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda}).

    Para cada valor fixado de θ1\theta_1, maximiza-se a verossimilhança sobre todos os valores possíveis de 𝛌\boldsymbol{\lambda}. Isso elimina os parâmetros de perturbação do problema. O estimador condicional 𝛌̂(θ1)=arg⁡max⁡𝛌L(θ1,𝛌)\hat{\boldsymbol{\lambda}}(\theta_1) = \arg\max_{\boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda}) depende de θ1\theta_1.

    Distinção importante: fixar 𝛌̂\hat{\boldsymbol{\lambda}} no EMV incondicional (opção 3) produz a verossimilhança concentrada ou estimativa plug-in, que difere da verossimilhança de perfil verdadeira por não capturar como 𝛌̂\hat{\boldsymbol{\lambda}} varia com θ1\theta_1.


  7. Question

    Para 𝛉∈ℝk\boldsymbol{\theta} \in \mathbb{R}^k, o Limite Inferior de Cramér-Rao (LICR) Multivariado estabelece um limite para a matriz de covariâncias de qualquer estimador não-viesado 𝛉̂\hat{\boldsymbol{\theta}}. Qual das alternativas abaixo descreve corretamente esse limite?


    1. A variância de cada componente satisfaz Var(θ̂j)≥1/[In(𝛉)]jj\text{Var}(\hat{\theta}_j) \geq 1/[I_n(\boldsymbol{\theta})]_{jj}, usando a entrada diagonal de InI_n (não sua inversa).
    2. Cov(𝛉̂)=In(𝛉)−1\text{Cov}(\hat{\boldsymbol{\theta}}) = I_n(\boldsymbol{\theta})^{-1} para o EMV, com igualdade exata para qualquer nn finito.
    3. Var(θ̂j)≥[In(𝛉)−1]jj\text{Var}(\hat{\theta}_j) \geq [I_n(\boldsymbol{\theta})^{-1}]_{jj} somente para os estimadores individuais, sem implicação sobre a matriz de covariância completa.
    4. O limite de Cramér-Rao não se generaliza para o caso multiparamétrico; vale apenas para k=1k = 1.
    5. Cov(𝛉̂)≥In(𝛉)−1\text{Cov}(\hat{\boldsymbol{\theta}}) \geq I_n(\boldsymbol{\theta})^{-1} no sentido de matrizes positivo-semidefinidas, onde In(𝛉)I_n(\boldsymbol{\theta}) é a matriz de informação de Fisher para nn observações.

    Solution

    O Limite de Cramér-Rao Multivariado afirma que, para qualquer estimador não-viesado 𝛉̂\hat{\boldsymbol{\theta}} de 𝛉∈ℝk\boldsymbol{\theta} \in \mathbb{R}^k: Cov(𝛉̂)−In(𝛉)−1≥0\text{Cov}(\hat{\boldsymbol{\theta}}) - I_n(\boldsymbol{\theta})^{-1} \geq 0 no sentido de matrizes positivo-semidefinidas (PSD), onde In(𝛉)=nI(𝛉)I_n(\boldsymbol{\theta}) = n\,I(\boldsymbol{\theta}) é a matriz de informação de Fisher para nn observações.

    Implicações: - A desigualdade matricial PSD implica que Var(θ̂j)≥[In(𝛉)−1]jj\text{Var}(\hat{\theta}_j) \geq [I_n(\boldsymbol{\theta})^{-1}]_{jj} para cada componente. - Note que [In(𝛉)−1]jj≥1/[In(𝛉)]jj[I_n(\boldsymbol{\theta})^{-1}]_{jj} \geq 1/[I_n(\boldsymbol{\theta})]_{jj} (pela desigualdade de Cauchy-Schwarz para matrizes), portanto a opção 4 é um limite mais fraco que o LICR correto. - A igualdade Cov=In−1\text{Cov} = I_n^{-1} vale apenas assintoticamente para o EMV, não para nn finito em geral.


  8. Question

    Para X∼Gama(α,β)X \sim \text{Gama}(\alpha, \beta) com função de densidade f(x)=βαΓ(α)xα−1e−βxf(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, os EMVs α̂MV\hat{\alpha}_{MV} e β̂MV\hat{\beta}_{MV} são obtidos resolvendo o sistema de equações de score conjuntas. No verdadeiro valor (α,β)=(3,1)(\alpha, \beta) = (3, 1), qual afirmação sobre a variância assintótica de β̂MV\hat{\beta}_{MV} é correta?


    1. No caso Gama, os EMVs de α\alpha e β\beta são assintoticamente independentes, pois a distribuição pertence à família exponencial.
    2. No caso multiparamétrico, a variância assintótica de β̂MV\hat{\beta}_{MV} leva em conta a informação cruzada entre α̂\hat{\alpha} e β̂\hat{\beta}, dada pela matriz de informação inversa I(α,β)−1I(\alpha, \beta)^{-1}.
    3. A variância assintótica de β̂MV\hat{\beta}_{MV} é simplesmente 1/I22(α,β)1/I_{22}(\alpha,\beta), ignorando os termos fora da diagonal da matriz de informação.
    4. A variância assintótica de β̂MV\hat{\beta}_{MV} é β2/(α⋅n)\beta^2/(\alpha \cdot n) para qualquer nn grande.
    5. A matriz de informação de Fisher da distribuição Gama é sempre diagonal, sem termos de covariância entre α\alpha e β\beta.

    Solution

    A matriz de informação de Fisher para a distribuição Gama(α,β)(\alpha, \beta) é: I(α,β)=(ψ′(α)−1/β−1/βα/β2),I(\alpha, \beta) = \begin{pmatrix} \psi'(\alpha) & -1/\beta \\ -1/\beta & \alpha/\beta^2 \end{pmatrix}, onde ψ′(α)=∑k=0∞1(α+k)2\psi'(\alpha) = \sum_{k=0}^\infty \frac{1}{(\alpha+k)^2} é a função trigamma.

    Fora da diagonal (covariância entre os scores de α\alpha e β\beta) é −1/β≠0-1/\beta \neq 0, portanto a matriz de informação não é diagonal e os EMVs não são assintoticamente independentes.

    A variância assintótica de β̂MV\hat{\beta}_{MV} é [I(α,β)−1]22[I(\alpha,\beta)^{-1}]_{22}, que depende de todos os elementos de II (inclusive os termos cruzados), via fórmula de inversão de matrizes 2×22\times 2: [I−1]22=I11det⁡(I)=ψ′(α)ψ′(α)α/β2−1/β2.[I^{-1}]_{22} = \frac{I_{11}}{\det(I)} = \frac{\psi'(\alpha)}{\psi'(\alpha)\,\alpha/\beta^2 - 1/\beta^2}.

    Ignorar os termos fora da diagonal superestima a variância assintótica.

    3. Wn=n(𝐀𝛉̂−𝐛)⊤[𝐀Î−1(𝛉̂)𝐀⊤]−1(𝐀𝛉̂−𝐛)→Dχr2W_n = n(\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b})^\top [\boldsymbol{A}\hat{I}^{-1}(\hat{\boldsymbol{\theta}})\boldsymbol{A}^\top]^{-1} (\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b}) \xrightarrow{D} \chi^2_r.
    4. Wn=n(𝛉̂−𝛉0)⊤Î(𝛉̂)n(𝛉̂−𝛉0)→Dχk2W_n = \sqrt{n}(\hat{\boldsymbol{\theta}} - \boldsymbol{\theta}_0)^\top \hat{I}(\hat{\boldsymbol{\theta}}) \sqrt{n}(\hat{\boldsymbol{\theta}} - \boldsymbol{\theta}_0) \xrightarrow{D} \chi^2_k, sem incorporar a restrição linear 𝐀\boldsymbol{A}.
    5. Wn=n(𝐀𝛉̂−𝐛)⊤(𝐀𝐀⊤)−1(𝐀𝛉̂−𝐛)→Dχr2W_n = n(\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b})^\top (\boldsymbol{A}\boldsymbol{A}^\top)^{-1} (\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b}) \xrightarrow{D} \chi^2_r, ignorando a informação de Fisher.

    Solution

    A estatística de Wald para H0:𝐀𝛉=𝐛H_0: \boldsymbol{A}\boldsymbol{\theta} = \mathbf{b} é: Wn=n(𝐀𝛉̂−𝐛)⊤[𝐀Î−1(𝛉̂)𝐀⊤]−1(𝐀𝛉̂−𝐛)→Dχr2.W_n = n(\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b})^\top \left[\boldsymbol{A}\,\hat{I}^{-1}(\hat{\boldsymbol{\theta}})\,\boldsymbol{A}^\top\right]^{-1} (\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b}) \xrightarrow{D} \chi^2_r.

    Intuição: Pelo Método Delta, n(𝐀𝛉̂−𝐛)→DNr(𝟎,𝐀I−1𝐀⊤)\sqrt{n}(\boldsymbol{A}\hat{\boldsymbol{\theta}} - \mathbf{b}) \xrightarrow{D} N_r(\mathbf{0}, \boldsymbol{A} I^{-1} \boldsymbol{A}^\top) sob H0H_0. A forma quadrática com a inversa dessa matriz de covariâncias produz uma χr2\chi^2_r.

    Erros comuns: - Usar Î\hat{I} sem inverter: a ponderação seria pela informação (não pela variância), produzindo resultado incorreto. - Ignorar 𝐀\boldsymbol{A} na normalização: a estatística não captura a estrutura da hipótese linear.

    O Teste de Score (também chamado de Teste de Rao ou Teste do Multiplicador de Lagrange) para H0:𝛉=𝛉0H_0: \boldsymbol{\theta} = \boldsymbol{\theta}_0 tem como vantagem não exigir a estimação do modelo completo. Qual das alternativas descreve corretamente sua estatística?

  14. Question

    Na teoria assintótica, os três grandes testes de hipóteses — Wald (W), Razão de Verossimilhanças (TRV/LRT) e Score/Rao (S) — são frequentemente utilizados para testar H0:𝛉∈Ω0H_0: \boldsymbol{\theta} \in \Omega_0. Como se relacionam esses três testes?


    1. O TRV é sempre mais poderoso que Wald e Score para amostras finitas, sendo a escolha ótima em qualquer situação.
    2. Sempre numericamente idênticos para qualquer tamanho amostral nn, produzindo exatamente os mesmos p-valores.
    3. O teste de Score é sempre mais conservador que o de Wald, independente do modelo ou tamanho amostral.
    4. Assintoticamente equivalentes sob H0H_0: todos têm distribuição χr2\chi^2_r e são assintoticamente iguais em primeira ordem. Podem diferir em amostras finitas.
    5. São equivalentes apenas quando a hipótese é simples (r=1r = 1); para r>1r > 1 têm distribuições asintóticas distintas.

    Solution

    Os três testes (Wald, TRV e Score) são assintoticamente equivalentes em primeira ordem: - Todos têm distribuição χr2\chi^2_r sob H0H_0 (pelo Teorema de Wilks e análogos). - Diferem por termos de ordem Op(n−1)O_p(n^{-1}): para amostras finitas, podem dar resultados ligeiramente diferentes. - Nenhum domina os outros em poder para amostras finitas em geral — a escolha depende da conveniência computacional.

    Comparação prática: - Wald: requer apenas o EMV irrestrito 𝛉̂\hat{\boldsymbol{\theta}}. - Score/Rao: requer apenas estimativas sob H0H_0 (útil quando o modelo restrito é fácil). - TRV: requer o EMV tanto restrito quanto irrestrito, mas é invariante a reparametrizações.

    A equivalência assintótica vale para qualquer r≥1r \geq 1 (não apenas r=1r=1).

    1. Tem distribuição χ1+dim⁡(𝛌)2\chi^2_{1+\dim(\boldsymbol{\lambda})} para acomodar a incerteza adicional em 𝛌\boldsymbol{\lambda}.
    2. A presença de 𝛌\boldsymbol{\lambda} invalida o Teorema de Wilks para o teste de θ1\theta_1, exigindo simulação ou bootstrap.
    3. Não pode ser aplicado na presença de parâmetros de perturbação sem hipóteses adicionais de ortogonalidade entre θ1\theta_1 e 𝛌\boldsymbol{\lambda}.
    4. Exige estimar 𝛌\boldsymbol{\lambda} separadamente por outro método (ex: momentos) antes de testar θ1\theta_1, introduzindo duas etapas distintas.
    5. Usa a verossimilhança perfilada LP(θ1)=sup⁡𝛌L(θ1,𝛌)L_P(\theta_1) = \sup_{\boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda}); a estatística −2log⁡ΛP-2\log\Lambda_P tem distribuição χ12\chi^2_1 assintoticamente, eliminando 𝛌\boldsymbol{\lambda} do teste.

    Solution

    O TRV de Perfil para H0:θ1=θ1,0H_0: \theta_1 = \theta_{1,0} é: ΛP=sup𝛌L(θ1,0,𝛌)supθ1,𝛌L(θ1,𝛌)=LP(θ1,0)L(θ̂1,𝛌̂).\Lambda_P = \frac{\sup_{\boldsymbol{\lambda}} L(\theta_{1,0}, \boldsymbol{\lambda})}{\sup_{\theta_1, \boldsymbol{\lambda}} L(\theta_1, \boldsymbol{\lambda})} = \frac{L_P(\theta_{1,0})}{L(\hat{\theta}_1, \hat{\boldsymbol{\lambda}})}.

  20. Question

    No contexto de testes múltiplos simultâneos H0jH_{0j}, j=1,…,mj = 1, \ldots, m, o problema do nível de significância inflado (ou inflação do erro tipo I) é controlado pela Correção de Bonferroni. Como funciona essa correção?


    1. Nível α*=α/m\alpha^* = \alpha/m para cada teste individual, garantindo que a probabilidade de pelo menos uma rejeição falsa (FWER) não exceda α\alpha.
    2. Nível α*=1−(1−α)m\alpha^* = 1-(1-\alpha)^m, que é a probabilidade de ao menos uma rejeição quando todos os H0jH_{0j} são verdadeiros.
    3. Nível α*=α/m\alpha^* = \alpha/\sqrt{m}, que é a correção de Šidák para testes correlacionados positivamente.
    4. Rejeitar H0jH_{0j} quando o p-valorj≤α_j \leq \alpha para todos os jj simultaneamente — exige consenso entre todos os testes.
    5. Nível α*=α⋅m\alpha^* = \alpha \cdot m para cada teste, compensando a redução de poder estatístico nos testes individuais.

    Solution

    A Correção de Bonferroni usa nível α*=α/m\alpha^* = \alpha/m em cada teste individual. Isso garante: FWER=P(ao menos uma rejeição falsa)≤∑j=1mP(rejeitar H0j|H0j verdadeira)=m⋅αm=α.\text{FWER} = P(\text{ao menos uma rejeição falsa}) \leq \sum_{j=1}^m P(\text{rejeitar } H_{0j} | H_{0j} \text{ verdadeira}) = m \cdot \frac{\alpha}{m} = \alpha.

    Comparação com as alternativas: - Opção 3 (1−(1−α)m1-(1-\alpha)^m): é o FWER sem correção, quando os mm testes são independentes — o nível real que queremos controlar. - Opção 4 (exigir consenso): equivale a testar a interseção ⋂jH0j\bigcap_j H_{0j}, um critério muito conservador. - Correção de Šidák: α*=1−(1−α)1/m\alpha^* = 1-(1-\alpha)^{1/m} (não α/m\alpha/\sqrt{m}) — válida para testes independentes e levemente menos conservadora que Bonferroni.

    Limitação: Bonferroni é conservadora (controla o FWER, mas pode ser excessivamente restritiva). Para controlar a FDR (false discovery rate), usa-se o procedimento de Benjamini-Hochberg.
