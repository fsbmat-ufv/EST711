Exam 2

  1. Question

    Sabe-se que \(E(|X|^{7})<\infty\). Qual conclusão é garantida?


    1. \(X\) possui suporte limitado.
    2. A fgm de \(X\) existe para todo \(t\).
    3. \(X\) tem distribuição simétrica.
    4. \(E(|X|^{2})<\infty\).
    5. \(E(|X|^{8})<\infty\).

    Solution

    A existência de um momento absoluto de ordem \(m\) implica a existência de todos os momentos absolutos de ordens menores. Isso decorre de \(|x|^k\leq1+|x|^m\) para \(k\leq m\).


  2. Question

    Uma variável não negativa tem \(E(X)=6\). Pelo limite de Markov, qual é o maior valor garantido para \(P(X\geq23)\)?


    1. \(6.0000\)
    2. \(3.8333\)
    3. \(0.0113\)
    4. \(0.2609\)
    5. \(0.7391\)

    Solution

    A desigualdade de Markov dá \[P(X\geq c)\leq\frac{E(X)}c =\frac{6}{23}=0.2609.\]


  3. Question

    Sabe-se que \(E(X^2)=6\). Aplicando Markov a \(X^2\), qual limite superior se obtém para \(P(|X|\geq12)\)?


    1. \(6.0000\)
    2. \(0.9583\)
    3. \(0.2041\)
    4. \(0.0417\)
    5. \(0.5000\)

    Solution

    O evento \(|X|\geq c\) equivale a \(X^2\geq c^2\). Como \(X^2\geq0\), \[P(|X|\geq c)\leq\frac{E(X^2)}{c^2} =0.0417.\]


  4. Question

    Uma variável tem média 3 e desvio-padrão 2. Pelo limite de Chebyshev, qual é o limite superior para \(P(|X-\mu|\geq3.5\sigma)\)?


    1. \(0.9184\)
    2. \(12.2500\)
    3. \(0.0816\)
    4. \(0.3265\)
    5. \(0.2857\)

    Solution

    Chebyshev afirma \[P(|X-\mu|\geq k\sigma)\leq\frac1{k^2} =0.0816.\] O limite independe dos valores específicos de \(\mu\) e \(\sigma\).


  5. Question

    Sem supor uma família de distribuições, qual limite inferior Chebyshev garante para \(P(|X-\mu|<3\sigma)\)?


    1. \(0.7500\)
    2. \(0.6667\)
    3. \(0.1111\)
    4. \(0.8889\)
    5. \(0.3333\)

    Solution

    Pelo complemento da forma usual de Chebyshev, \[P(|X-\mu|<k\sigma)\geq1-\frac1{k^2} =0.8889.\]


  6. Question

    Se \(X\sim U(-\sqrt3,\sqrt3)\), então \(E(X)=0\) e \(\operatorname{Var}(X)=1\). Para \(k=1.7\), qual é o valor exato de \(P(|X|\geq k)\)?


    1. \(0.6540\)
    2. \(0.3460\)
    3. \(0.0185\)
    4. \(0.5882\)
    5. \(0.9815\)

    Solution

    O comprimento total do suporte é \(2\sqrt3\), e o intervalo central \((-k,k)\) tem comprimento \(2k\). Logo, \[P(|X|\geq k)=1-\frac{k}{\sqrt3} =0.0185.\] Chebyshev daria apenas o limite superior \(1/k^2=0.3460\).


  7. Question

    Se \(\phi\) é convexa, como a função x^2 em seu domínio, e as esperanças existem, qual é a desigualdade de Jensen?


    1. \(\phi(E[X])\geq E[\phi(X)]\).
    2. \(\phi(E[X])\leq E[\phi(X)]\).
    3. \(E[\phi(X)]=\phi(X)\) quase certamente.
    4. \(E[\phi(X)]\leq E(X)\).
    5. \(\phi(E[X])=E(X)\phi(1)\).

    Solution

    Para funções convexas, \[\phi(E[X])\leq E[\phi(X)].\] O sentido se inverte para funções côncavas.


  8. Question

    Se \(E(X)=-5\), qual limite inferior Jensen fornece para \(E(X^2)\)? Considere \(\phi(x)=x^2\).


    1. \(0.500\)
    2. \(25.000\)
    3. \(5.000\)
    4. \(25.500\)
    5. \(5.050\)

    Solution

    Como \(x^2\) é convexa, \[[E(X)]^2\leq E(X^2).\] Assim, \(E(X^2)\geq(-5)^2=25.000\). O valor 25.5 foi usado apenas para gerar distratores plausíveis.


  9. Question

    Para os números positivos \((11, 6, 7, 8)\), qual é a média harmônica?


    1. \(0.1314\)
    2. \(8.0000\)
    3. \(7.6128\)
    4. \(32.0000\)
    5. \(7.7971\)

    Solution

    A média harmônica é \[HM=\frac{n}{\sum_i1/x_i}=7.6128.\] Além disso, Jensen permite estabelecer \(HM\leq GM\leq AM\); aqui \(GM=7.7971\) e \(AM=8.0000\).


  10. Question

    Se \(\phi=exp(x)\) é estritamente convexa e \(E[\phi(X)]<\infty\), quando ocorre igualdade em Jensen, \(\phi(E[X])=E[\phi(X)]\)?


    1. Para toda variável contínua.
    2. Somente quando \(X\) é constante quase certamente.
    3. Somente quando \(\operatorname{Var}(X)=1\).
    4. Sempre que \(E(X)=0\).
    5. Sempre que a distribuição de \(X\) é simétrica.

    Solution

    Para uma função estritamente convexa, Jensen é estrita quando \(X\) é não degenerada. A igualdade ocorre precisamente quando toda a massa de \(X\) está concentrada em um único valor, isto é, quando \(X\) é constante quase certamente.
