Exam 1

  1. Question

    Ao modelar uma pesquisa amostral, uma função \(P\) é definida sobre os eventos de um espaço amostral \(\Omega\). Qual condição faz parte dos axiomas de probabilidade?


    1. A probabilidade do espaço amostral pode assumir qualquer constante positiva.
    2. Se \(A\subset B\), então \(P(A)>P(B)\).
    3. Para eventos dois a dois disjuntos \(A_1,A_2,\ldots\), vale \(P(\bigcup_i A_i)=\sum_iP(A_i)\).
    4. Para quaisquer eventos \(A\) e \(B\), vale sempre \(P(A\cup B)=P(A)+P(B)\).
    5. Todo evento não vazio deve ter probabilidade estritamente positiva.

    Solution

    Os axiomas exigem não negatividade, \(P(\Omega)=1\) e aditividade enumerável para eventos dois a dois disjuntos. Para eventos que se sobrepõem, a interseção precisa ser descontada. Eventos não vazios podem ter probabilidade zero.


  2. Question

    Em certo experimento, \(P(A)=0.23\). Qual é \(P(A^c)\)?


    1. \(0.7700\)
    2. \(0.2300\)
    3. \(0.0529\)
    4. \(0.8130\)
    5. \(0.5929\)

    Solution

    Como \(A\) e \(A^c\) são disjuntos e sua união é \(\Omega\), \[P(A^c)=1-P(A)=1-0.23=0.7700.\]


  3. Question

    Em uma população de 140 unidades, 28 satisfazem \(A\), 30 satisfazem \(B\) e 19 satisfazem ambos. Selecionando uma unidade ao acaso, qual é \(P(A\cup B)\)?


    1. \(0.279\)
    2. \(0.136\)
    3. \(0.550\)
    4. \(0.414\)
    5. \(0.721\)

    Solution

    Pela fórmula da adição, \[P(A\cup B)=P(A)+P(B)-P(A\cap B) =\frac{28+30-19}{140} =0.279.\]


  4. Question

    Para três eventos, são conhecidas as probabilidades \[(P(A),P(B),P(C))=(0.45,0.55,0.3),\] \[(P(A\cap B),P(A\cap C),P(B\cap C))=(0.16,0.14,0.15)\] e \(P(A\cap B\cap C)=0.05\). Calcule \(P(A\cup B\cup C)\).


    1. \(1.350\)
    2. \(0.850\)
    3. \(0.900\)
    4. \(1.300\)
    5. \(0.100\)

    Solution

    A inclusão-exclusão fornece \[P(A\cup B\cup C)=\sum P(A_i)-\sum P(A_i\cap A_j)+P(A\cap B\cap C) =0.900.\]


  5. Question

    Uma comissão de 4 pessoas é escolhida uniformemente, sem reposição, entre 8 mulheres e 6 homens. Qual é a probabilidade de a comissão conter exatamente 2 mulheres?


    1. \(0.0280\)
    2. \(0.4196\)
    3. \(0.5804\)
    4. \(0.0659\)
    5. \(0.5000\)

    Solution

    Há \(\binom{14}{4}\) comissões possíveis. As favoráveis escolhem 2 mulheres e 2 homens: \[P=\frac{\binom{8}{2}\binom{6}{2}} {\binom{14}{4}} =0.4196.\]


  6. Question

    Supondo 365 dias equiprováveis e ignorando anos bissextos, qual é a probabilidade de haver ao menos duas pessoas com o mesmo aniversário em um grupo de 32 pessoas?


    1. \(0.0840\)
    2. \(0.2467\)
    3. \(1.3589\)
    4. \(0.0877\)
    5. \(0.7533\)

    Solution

    Pelo complemento, \[P(\text{coincidência})=1-\prod_{j=0}^{31}\frac{365-j}{365} =0.7533.\]


  7. Question

    Para 3 eventos, as probabilidades individuais são 0.04, 0.05, 0.12. Sem informação sobre interseções, qual limite superior é garantido por Boole para \(P(\bigcup_iA_i)\)?


    1. \(0.7900\)
    2. \(0.0002\)
    3. \(0.2100\)
    4. \(0.0700\)
    5. \(0.1200\)

    Solution

    A desigualdade de Boole fornece \[P\left(\bigcup_iA_i\right)\leq\sum_iP(A_i).\] Como uma probabilidade não excede 1, o limite é \(\min\{1,\sum_iP(A_i)\}=0.2100\).


  8. Question

    Considere eventos crescentes \(A_1\subseteq A_2\subseteq\cdots\), por exemplo A_n = {|X| <= n}. Qual propriedade é correta?


    1. \(P(\bigcup_{n=1}^{\infty}A_n)=\prod_{n=1}^{\infty}P(A_n)\).
    2. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}[1-P(A_n)]\).
    3. \(P(\bigcap_{n=1}^{\infty}A_n)=\sum_{n=1}^{\infty}P(A_n)\).
    4. A igualdade com o limite exige que os \(A_n\) sejam disjuntos.
    5. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}P(A_n)\).

    Solution

    A probabilidade é contínua por baixo: \[P\left(\bigcup_{n=1}^{\infty}A_n\right)=\lim_{n\to\infty}P(A_n).\] A condição necessária aqui é a monotonicidade crescente, não a disjunção.


  9. Question

    Um dado honesto de 10 faces, numeradas de 1 a 10, é lançado. Qual é a probabilidade de o resultado ser pelo menos 8?


    1. \(0.1250\)
    2. \(0.3000\)
    3. \(0.7000\)
    4. \(0.8000\)
    5. \(0.2000\)

    Solution

    Há 3 resultados favoráveis entre 10 equiprováveis. Logo, \[P(X\geq 8)=\frac{3}{10} =0.3000.\]


  10. Question

    Sabe-se apenas que \(P(A)=0.55\) e \(P(B)=0.75\). Qual é o limite inferior de Bonferroni para \(P(A\cap B)\)?


    1. \(0.700\)
    2. \(-0.300\)
    3. \(0.550\)
    4. \(0.300\)
    5. \(0.413\)

    Solution

    Como \(P(A\cup B)\leq1\), \[P(A\cap B)\geq P(A)+P(B)-1 =0.300.\]
