  1. Question

    Seja TnT_n um estimador de θ\theta baseado em uma amostra de tamanho nn. Qual das afirmações abaixo define corretamente a convergência em probabilidade Tn→PθT_n \xrightarrow{P} \theta?


    1. P(|Tn−θ|=0)→1P(|T_n - \theta| = 0) \to 1 quando n→∞n \to \infty.
    2. P(|θ−Tn|≤ε)→0P(|\theta - T_n| \leq \varepsilon) \to 0 quando n→∞n \to \infty, para todo ε>0\varepsilon > 0.
    3. Tn→θT_n \to \theta quase certamente (com probabilidade 1).
    4. P(|Tn−θ|≥ε)→0P(|T_n - \theta| \geq \varepsilon) \to 0 quando n→∞n \to \infty, para todo ε>0\varepsilon > 0.
    5. P(|Tn−θ|<ε)→0P(|T_n - \theta| < \varepsilon) \to 0 quando n→∞n \to \infty, para todo ε>0\varepsilon > 0.

    Solution

    A definição correta de convergência em probabilidade é: para todo ε>0\varepsilon > 0, P(|Tn−θ|≥ε)→0P(|T_n - \theta| \geq \varepsilon) \to 0 quando n→∞n \to \infty.

    Equivalentemente, P(|Tn−θ|<ε)→1P(|T_n - \theta| < \varepsilon) \to 1, mas a definição padrão usa a probabilidade do evento {|Tn−θ|≥ε}\{|T_n - \theta| \geq \varepsilon\} ir a zero. As demais alternativas confundem o sentido da desigualdade, trocam por convergência quase certa (mais forte), ou usam P(|Tn−θ|=0)P(|T_n - \theta| = 0), que não é a definição correta.


  2. Question

    Seja X1,X2,…,XnX_1, X_2, \ldots, X_n uma sequência de variáveis aleatórias iid com E(Xi)=9E(X_i) = 9 e Var(Xi)=7\text{Var}(X_i) = 7. Defina a média amostral X‾n=1n∑i=1nXi\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i.

    Pela Lei Fraca dos Grandes Números (LFGN), qual é o limite em probabilidade de X‾n\bar{X}_n quando n→∞n \to \infty?


    1. 0.36
    2. 9
    3. 7
    4. 16
    5. 225

    Solution

    Pela Lei Fraca dos Grandes Números, se E(Xi)=μE(X_i) = \mu e Var(Xi)=σ2<∞\text{Var}(X_i) = \sigma^2 < \infty, então X‾n→Pμ=9\bar{X}_n \xrightarrow{P} \mu = 9.

    Isso segue pois E(X‾n)=μE(\bar{X}_n) = \mu e Var(X‾n)=σ2/n→0\text{Var}(\bar{X}_n) = \sigma^2/n \to 0, portanto pela desigualdade de Chebyshev, P(|X‾n−μ|≥ε)≤σ2/(nε2)→0P(|\bar{X}_n - \mu| \geq \varepsilon) \leq \sigma^2/(n\varepsilon^2) \to 0.

  5. Question

    Seja X1,…,XnX_1, \ldots, X_n uma amostra iid com Var(Xi)=σ2=14\text{Var}(X_i) = \sigma^2 = 14 e tamanho de amostra n=36n = 36. Qual é a variância da média amostral X‾n\bar{X}_n?

    Var(X‾n)=σ2n\text{Var}(\bar{X}_n) = \frac{\sigma^2}{n}


    1. 0.4
    2. 504
    3. 0.3889
    4. 0.6236
    5. 0.3784

    Solution

    Como X1,…,XnX_1, \ldots, X_n são iid, temos: Var(X‾n)=Var(1n∑i=1nXi)=1n2∑i=1nVar(Xi)=nσ2n2=σ2n\text{Var}(\bar{X}_n) = \text{Var}\left(\frac{1}{n}\sum_{i=1}^n X_i\right) = \frac{1}{n^2}\sum_{i=1}^n \text{Var}(X_i) = \frac{n\sigma^2}{n^2} = \frac{\sigma^2}{n}

    Substituindo σ2=14\sigma^2 = 14 e n=36n = 36: Var(X‾n)=1436=0.3889\text{Var}(\bar{X}_n) = \frac{14}{36} = 0.3889

    Note que Var(X‾n)→0\text{Var}(\bar{X}_n) \to 0 conforme n→∞n \to \infty, o que é uma condição suficiente para consistência (quando combinado com não-viés).


  6. Question

    Seja Tn→PθT_n \xrightarrow{P} \theta e g:ℝ→ℝg : \mathbb{R} \to \mathbb{R} uma função contínua em θ\theta. O que podemos concluir sobre g(Tn)g(T_n)?


    1. g(Tn)→a.s.g(θ)g(T_n) \xrightarrow{a.s.} g(\theta) (convergência quase certa, mais forte do que o garantido).
    2. Nada pode ser concluído sobre g(Tn)g(T_n) sem hipóteses adicionais sobre gg.
    3. g(Tn)→Pg(θ)g(T_n) \xrightarrow{P} g(\theta) (convergência em probabilidade é preservada por funções contínuas).
    4. g(Tn)→Dg(θ)g(T_n) \xrightarrow{D} g(\theta), mas não necessariamente em probabilidade.
    5. g(Tn)→g(θ)g(T_n) \to g(\theta) quase certamente, mas não necessariamente em probabilidade.

    Solution

    Este é o Teorema do Mapeamento Contínuo (Continuous Mapping Theorem) para convergência em probabilidade: se Tn→PθT_n \xrightarrow{P} \theta e gg é contínua em θ\theta, então g(Tn)→Pg(θ)g(T_n) \xrightarrow{P} g(\theta).

    A convergência em probabilidade é preservada por transformações contínuas. Note que o resultado não garante convergência quase certa (que é mais forte), e a convergência em distribuição já estaria implícita pela hierarquia das convergências (P⇒DP \Rightarrow D).


  7. Question

    Sejam XnX_n e YnY_n sequências de variáveis aleatórias tais que Xn→P5X_n \xrightarrow{P} 5 e Yn→P3Y_n \xrightarrow{P} 3. Pelo Teorema de Slutsky (ou pelas propriedades da convergência em probabilidade), qual é o limite em probabilidade de Xn+YnX_n + Y_n?


    1. 2
    2. 8
    3. 1.6667
    4. 15
    5. 9

    Solution

    A convergência em probabilidade é preservada por operações aritméticas: se Xn→PaX_n \xrightarrow{P} a e Yn→PbY_n \xrightarrow{P} b, então Xn+Yn→Pa+bX_n + Y_n \xrightarrow{P} a + b.

    Portanto Xn+Yn→P5+3=8X_n + Y_n \xrightarrow{P} 5 + 3 = 8.

    De forma análoga, Xn⋅Yn→P15X_n \cdot Y_n \xrightarrow{P} 15 e Xn/Yn→P1.6667X_n / Y_n \xrightarrow{P} 1.6667 (desde que b≠0b \neq 0).

    Seja X1,…,XnX_1, \ldots, X_n uma amostra iid com E(Xi)=7E(X_i) = 7 e Var(Xi)=σ2=8\text{Var}(X_i) = \sigma^2 = 8, com n=100n = 100. Considere a variância amostral não-viesada: Sn2=1n−1∑i=1n(Xi−X‾n)2S^2_n = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X}_n)^2

    Qual é o limite em probabilidade de Sn2S^2_n quando n→∞n \to \infty?


    1. 2.8284
    2. 7.9208
    3. 8
    4. 0.08
    5. 8.0808

    Solution

    A variância amostral Sn2S^2_n é um estimador consistente de σ2\sigma^2, ou seja, Sn2→Pσ2=8S^2_n \xrightarrow{P} \sigma^2 = 8.

  9. Question

    Seja TnT_n um estimador de θ\theta tal que o erro quadrático médio satisfaz EQM(Tn)=E[(Tn−θ)2]→0\text{EQM}(T_n) = E[(T_n - \theta)^2] \to 0 quando n→∞n \to \infty. O que se pode concluir?


    1. TnT_n tem variância mínima entre todos os estimadores de θ\theta.
    2. TnT_n é não-viesado, ou seja, E(Tn)=θE(T_n) = \theta para todo nn.
    3. Tn→PθT_n \xrightarrow{P} \theta (consistência em probabilidade).
    4. EQM(Tn)→0\text{EQM}(T_n) \to 0 não implica nenhuma forma de convergência estocástica de TnT_n.
    5. Tn→a.s.θT_n \xrightarrow{a.s.} \theta (convergência quase certa).

    Solution

    Se EQM(Tn)=E[(Tn−θ)2]→0\text{EQM}(T_n) = E[(T_n - \theta)^2] \to 0, então Tn→PθT_n \xrightarrow{P} \theta (consistência em probabilidade).

    Isso segue diretamente da desigualdade de Markov: para qualquer ε>0\varepsilon > 0, P(|Tn−θ|≥ε)=P((Tn−θ)2≥ε2)≤E[(Tn−θ)2]ε2=EQM(Tn)ε2→0.P(|T_n - \theta| \geq \varepsilon) = P((T_n - \theta)^2 \geq \varepsilon^2) \leq \frac{E[(T_n-\theta)^2]}{\varepsilon^2} = \frac{\text{EQM}(T_n)}{\varepsilon^2} \to 0.

    Note que: (i) EQM →0\to 0 não implica convergência quase certa; (ii) EQM →0\to 0 não implica não-viés para nn fixo (pode haver viés que vai a zero); (iii) EQM →0\to 0 não diz nada sobre variância mínima.

  11. Question

    Seja (Xn)n≥1(X_n)_{n \geq 1} uma sequência de variáveis aleatórias e XX uma variável aleatória com função de distribuição acumulada FXF_X. Qual afirmação define corretamente a convergência em distribuição Xn→DXX_n \xrightarrow{D} X?


    1. P(Xn=x)→P(X=x)P(X_n = x) \to P(X = x) para todo x∈ℝx \in \mathbb{R}.
    2. FXn(x)→FX(x)F_{X_n}(x) \to F_X(x) para todo x∈ℝx \in \mathbb{R}, incluindo pontos de descontinuidade de FXF_X.
    3. E(Xn)→E(X)E(X_n) \to E(X) e Var(Xn)→Var(X)\text{Var}(X_n) \to \text{Var}(X).
    4. Xn−X→P0X_n - X \xrightarrow{P} 0, ou seja, a diferença converge em probabilidade a zero.
    5. FXn(x)→FX(x)F_{X_n}(x) \to F_X(x) para todo xx que seja ponto de continuidade de FXF_X (conjunto C(FX)C(F_X)).

    Solution

    A definição correta de convergência em distribuição (também chamada de convergência fraca) é: Xn→DXX_n \xrightarrow{D} X se e somente se FXn(x)→FX(x)F_{X_n}(x) \to F_X(x) para todo x∈C(FX)x \in C(F_X), onde C(FX)C(F_X) é o conjunto dos pontos de continuidade de FXF_X.

    A restrição aos pontos de continuidade é fundamental: se FXF_X for descontínua em x0x_0 (como ocorre em distribuições discretas), não se exige convergência pontual em x0x_0. As demais alternativas exigem convergência em todos os pontos (mais forte do que o necessário), confundem com convergência de momentos ou com convergência em probabilidade.

  14. Question

    Considere as diferentes noções de convergência para sequências de variáveis aleatórias. Qual das relações abaixo é verdadeira em geral?


    1. Se Xn→DXX_n \xrightarrow{D} X, então Xn→PXX_n \xrightarrow{P} X (sempre, para qualquer XX).
    2. Se Xn→DXX_n \xrightarrow{D} X, então Xn→a.s.XX_n \xrightarrow{a.s.} X (convergência em distribuição implica quase certa).
    3. Se Xn→PXX_n \xrightarrow{P} X, então Xn→DXX_n \xrightarrow{D} X (convergência em probabilidade implica em distribuição).
    4. As convergências em probabilidade e em distribuição são sempre equivalentes entre si.
    5. Convergência em distribuição implica convergência em probabilidade quando a variância limite é finita.

    Solution

    A hierarquia das convergências é: convergência quase certa (a.s.a.s.) ⇒\Rightarrow convergência em probabilidade (PP) ⇒\Rightarrow convergência em distribuição (DD). Portanto, Xn→PXX_n \xrightarrow{P} X implica Xn→DXX_n \xrightarrow{D} X.

    A recíproca é falsa em geral: convergência em distribuição não implica convergência em probabilidade (exceto quando o limite XX é uma constante). Por exemplo, Xn=−X∼N(0,1)X_n = -X \sim N(0,1) converge em distribuição para N(0,1)N(0,1) mas não em probabilidade para XX se X∼N(0,1)X \sim N(0,1) independente de XnX_n.


  15. Question

    Seja Xn→DXX_n \xrightarrow{D} X e Yn→Pc=4Y_n \xrightarrow{P} c = 4 (uma constante). Pelo Teorema de Slutsky, qual é a distribuição limite de Xn+YnX_n + Y_n?


    1. X⋅4X \cdot 4 (produto, mesmo resultado de escala).
    2. X+4X + 4 (a distribuição de XX deslocada por c=4c = 4).
    3. X+4X + 4, mas apenas se X∼N(0,1)X \sim N(0,1); caso contrário não há resultado geral.
    4. 4X4X (distribuição de XX escalada por cc).
    5. XX (sem alteração na distribuição limite).

    Solution

    Pelo Teorema de Slutsky: se Xn→DXX_n \xrightarrow{D} X e Yn→PcY_n \xrightarrow{P} c (constante), então Xn+Yn→DX+cX_n + Y_n \xrightarrow{D} X + c.

    A chave é que YnY_n deve convergir em probabilidade para uma constante cc. O resultado garante que a distribuição limite de Xn+YnX_n + Y_n é a distribuição de XX deslocada por c=4c = 4. Se cc fosse uma variável aleatória (não constante), o resultado não valeria em geral.


  16. Question

    Seja (Xn)(X_n) uma sequência de variáveis aleatórias tal que Xn→DcX_n \xrightarrow{D} c para alguma constante c∈ℝc \in \mathbb{R}. O que mais podemos concluir sobre a convergência de XnX_n?


    1. Xn→a.s.cX_n \xrightarrow{a.s.} c (convergência quase certa para cc, sempre).
    2. Var(Xn)→0\text{Var}(X_n) \to 0 mas XnX_n não necessariamente converge em probabilidade para cc.
    3. Nada mais pode ser concluído além da própria convergência em distribuição.
    4. Xn→PcX_n \xrightarrow{P} c (convergência em distribuição para constante implica convergência em probabilidade).
    5. E(Xn)→cE(X_n) \to c mas não há convergência em probabilidade para cc.

    Solution

    Quando o limite em distribuição é uma constante cc (variável degenerada), a convergência em distribuição é equivalente à convergência em probabilidade: Xn→Dc⇔Xn→PcX_n \xrightarrow{D} c \Leftrightarrow X_n \xrightarrow{P} c.

    Isso é um caso especial importante: a equivalência entre convergência em probabilidade e em distribuição vale somente quando o limite é uma constante. Em geral (limite não-constante), convergência em distribuição é mais fraca.

    Convergência quase certa não é garantida pela convergência em distribuição, mesmo quando o limite é uma constante.


  17. Question

    Seja Xn→DXX_n \xrightarrow{D} X e g:ℝ→ℝg : \mathbb{R} \to \mathbb{R} uma função contínua. O que podemos concluir sobre g(Xn)g(X_n)?


    1. g(Xn)→Dg(X)g(X_n) \xrightarrow{D} g(X) (o Teorema do Mapeamento Contínuo vale para convergência em distribuição).
    2. g(Xn)→a.s.g(X)g(X_n) \xrightarrow{a.s.} g(X) (convergência quase certa).
    3. Nenhuma conclusão pode ser tirada sobre g(Xn)g(X_n) sem hipóteses adicionais sobre a derivada de gg.
    4. g(Xn)→Pg(X)g(X_n) \xrightarrow{P} g(X) (convergência em probabilidade, que é mais forte).
    5. g(Xn)→Dg(X)g(X_n) \xrightarrow{D} g(X) apenas se gg for linear ou afim.

    Solution

    O Teorema do Mapeamento Contínuo (CMT) para convergência em distribuição afirma: se Xn→DXX_n \xrightarrow{D} X e gg é contínua (exceto possivelmente em um conjunto de probabilidade zero sob XX), então g(Xn)→Dg(X)g(X_n) \xrightarrow{D} g(X).

    Este resultado vale para qualquer função contínua gg, não apenas para funções lineares. Contudo, a convergência em distribuição de g(Xn)g(X_n) não garante convergência em probabilidade de g(Xn)g(X_n) para g(X)g(X) (que é mais forte).

  20. Question

    Seja X∼N(0,1)X \sim N(0,1) e defina a sequência Xn=(−1)nXX_n = (-1)^n X para n=1,2,3,…n = 1, 2, 3, \ldots. Qual afirmação sobre a convergência de XnX_n é verdadeira?


    1. Xn→a.s.XX_n \xrightarrow{a.s.} X pois cada realização alterna entre xx e −x-x de forma controlada.
    2. Xn→P0X_n \xrightarrow{P} 0 pois as oscilações entre +x+x e −x-x se cancelam em média.
    3. Xn→PXX_n \xrightarrow{P} X pois a distribuição é simétrica e o valor médio não muda.
    4. Xn→DN(0,1)X_n \xrightarrow{D} N(0,1) pois XX e −X-X têm a mesma distribuição N(0,1)N(0,1), mas XnX_n não converge em probabilidade para XX.
    5. XnX_n não tem limite em nenhum sentido de convergência estocástica.

    Solution

    Como X∼N(0,1)X \sim N(0,1), temos −X∼N(0,1)-X \sim N(0,1) (a normal padrão é simétrica). Portanto, Xn=(−1)nXX_n = (-1)^n X tem distribuição N(0,1)N(0,1) para todo nn, e trivialmente Xn→DN(0,1)=XX_n \xrightarrow{D} N(0,1) = X em distribuição.

    No entanto, XnX_n não converge em probabilidade para XX: para nn par, Xn=XX_n = X, mas para nn ímpar, Xn=−XX_n = -X. Temos P(|Xn−X|>ε)=P(2|X|>ε)↛0P(|X_n - X| > \varepsilon) = P(2|X| > \varepsilon) \not \to 0.

    Este é o exemplo clássico que mostra que convergência em distribuição ⇏\not \Rightarrow convergência em probabilidade quando o limite não é uma constante.
