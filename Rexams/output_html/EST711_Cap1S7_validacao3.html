Exam 3

  1. Question

    Uma função é dada por \(f(x)=c x^{2}\) para \(0<x<4\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.25000\)
    2. \(1.33333\)
    3. \(0.04688\)
    4. \(0.18750\)
    5. \(0.01562\)

    Solution

    A normalização exige \[1=c\int_0^{4}x^{2}dx =c\frac{4^{3}}{3}.\] Logo, \(c=(3)/4^{3}=0.04688\).


  2. Question

    Se $f(x)=2x/4 para \(0<x<2\), qual é \(P(0.63<X<1.40)\)?


    1. \(0.3948\)
    2. \(0.6052\)
    3. \(0.1942\)
    4. \(0.1974\)
    5. \(0.3884\)

    Solution

    \[P(u<X<v)=\int_u^v\frac{2x}{a^2}dx =\frac{v^2-u^2}{a^2} =0.3948.\]


  3. Question

    Uma variável tem densidade \(f(x)=(5)x^{4}/1024\) em \((0,4)\). Qual é \(F_X(1.70)\)?


    1. \(0.0138\)
    2. \(0.0406\)
    3. \(0.9862\)
    4. \(0.0325\)
    5. \(0.4247\)

    Solution

    Para \(0<x<a\), \[F_X(x)=\int_0^x\frac{(k+1)t^k}{a^{k+1}}dt =\left(\frac{x}{a}\right)^{k+1}.\] No ponto informado, isso vale 0.0138.


  4. Question

    Se \(F_X(x)=(x/5)^{2}\) para \(0<x<5\), qual é o quantil de ordem \(p=0.6\)?


    1. \(1.8000\)
    2. \(0.1200\)
    3. \(3.0000\)
    4. \(3.8730\)
    5. \(3.1623\)

    Solution

    Resolve-se \(F_X(q_p)=p\): \[\left(\frac{q_p}{5}\right)^{2}=0.6, \qquad q_p=5\,p^{1/2} =3.8730.\]


  5. Question

    Se \(X\) tem densidade \(f_X\) e \(Y=2X+-3\), qual é a densidade de \(Y\)?


    1. \(f_Y(y)=2f_X(2y+-3)\).
    2. \(f_Y(y)=\dfrac1{4}f_X(y/2--3)\).
    3. \(f_Y(y)=f_X(y--3)\).
    4. \(f_Y(y)=\dfrac1{2}f_X\!\left(\dfrac{y--3}{2}\right)\).
    5. \(f_Y(y)=f_X((y+-3)/2)\).

    Solution

    A inversa é \(x=(y--3)/2\) e seu jacobiano absoluto é \(1/2\). Portanto, \[f_Y(y)=\frac1{2}f_X\left(\frac{y--3}{2}\right).\]


  6. Question

    Se \(X\sim U(0,2)\) e \(Y=X^2\), qual é \(f_Y(2.388)\)?


    1. \(0.16177\)
    2. \(0.10467\)
    3. \(1.54545\)
    4. \(0.32353\)
    5. \(0.77272\)

    Solution

    A inversa é \(x=\sqrt y\) e \(|dx/dy|=1/(2\sqrt y)\). Assim, \[f_Y(y)=\frac1{2}\frac1{2\sqrt y}, \quad 0<y<4,\] que no ponto dado vale 0.16177.


  7. Question

    Se \(X\sim U(-4,4)\) e \(Y=X^2\), qual é \(F_Y(4.475)\)?


    1. \(0.4711\)
    2. \(0.0591\)
    3. \(0.2797\)
    4. \(1.0577\)
    5. \(0.5289\)

    Solution

    Para \(0<y<a^2\), \[F_Y(y)=P(-\sqrt y\leq X\leq\sqrt y) =\frac{2\sqrt y}{2a}=\frac{\sqrt y}{a}.\] No ponto indicado, o valor é 0.5289.


  8. Question

    Uma distribuição é mistura de \(f_1\) e \(f_2\) com pesos 0.55 e 0.45. Para certo conjunto \(A\), \(P_1(A)=0.45\) e \(P_2(A)=0.2\). Qual é \(P(A)\) na mistura?


    1. \(0.3375\)
    2. \(0.0900\)
    3. \(0.2475\)
    4. \(0.3250\)
    5. \(0.6625\)

    Solution

    Probabilidades também se combinam pelos pesos da mistura: \[P(A)=0.55(0.45)+0.45(0.2) =0.3375.\]


  9. Question

    Se \(U\sim U(0,1)\) e \(Y=-\log(1-U)/1\), qual é a distribuição de \(Y\)?


    1. Normal com média \(1/1\).
    2. Exponencial com taxa 1, pois \(F_Y(y)=1-e^{-1y}\) para \(y>0\).
    3. Uniforme em \((0,1/1)\).
    4. Exponencial com taxa \(1/1\), pois \(F_Y(y)=e^{-y/1}\).
    5. Gama com forma 1 e taxa 1.

    Solution

    Para \(y>0\), \[P(Y\leq y)=P\{U\leq1-e^{-1y}\} =1-e^{-1y},\] que é a FDA exponencial de taxa 1.


  10. Question

    Uma variável \(X\) é absolutamente contínua e sua densidade em \(x_0=1.5\) vale \(f_X(x_0)=1.4\). Qual é \(P(X=x_0)\)?


    1. \(0\).
    2. \(1-1.4\).
    3. 1.4.
    4. \(F_X(x_0)\).
    5. Não pode ser determinada sem integrar sobre toda a reta.

    Solution

    Em distribuições absolutamente contínuas, conjuntos com um único ponto têm probabilidade zero. O valor da densidade não é uma probabilidade pontual: \[P(X=x_0)=\int_{x_0}^{x_0}f_X(x)\,dx=0.\]
