Exam 2

  1. Question

    Em 90 registros, 61 apresentam \(B\) e 51 apresentam simultaneamente \(A\) e \(B\). Qual é \(P(A\mid B)\)?


    1. \(0.8361\)
    2. \(0.6778\)
    3. \(0.1639\)
    4. \(0.5667\)
    5. \(0.3222\)

    Solution

    \[P(A\mid B)=\frac{P(A\cap B)}{P(B)} =\frac{51}{61}=0.8361.\]


  2. Question

    Se \(P(A)=0.55\) e \(P(B\mid A)=0.75\), qual é \(P(A\cap B)\)?


    1. \(0.4125\)
    2. \(0.7500\)
    3. \(1.3636\)
    4. \(1.3000\)
    5. \(0.1375\)

    Solution

    Pela regra da multiplicação, \[P(A\cap B)=P(A)P(B\mid A) =(0.55)(0.75)=0.4125.\]


  3. Question

    Três máquinas produzem proporções 0.15, 0.35, 0.50 dos itens, com probabilidades de defeito 0.08, 0.09, 0.04. Qual é a probabilidade de um item escolhido ao acaso ser defeituoso?


    1. \(0.0700\)
    2. \(0.0228\)
    3. \(0.2100\)
    4. \(0.9365\)
    5. \(0.0635\)

    Solution

    Pela probabilidade total, \[P(D)=\sum_{j=1}^3P(M_j)P(D\mid M_j) =0.0635.\]


  4. Question

    Uma condição tem prevalência 0.07. Um teste possui sensibilidade 0.98 e especificidade 0.95. Qual é \(P(\text{condição}\mid\text{positivo})\)?


    1. \(0.9500\)
    2. \(0.4040\)
    3. \(0.5960\)
    4. \(0.0700\)
    5. \(0.9800\)

    Solution

    Pelo teorema de Bayes, \[P(C\mid+)=\frac{P(+\mid C)P(C)} {P(+\mid C)P(C)+P(+\mid C^c)P(C^c)} =0.5960.\]


  5. Question

    Eventos \(A\) e \(B\) são independentes, com \(P(A)=0.8\) e \(P(B)=0.35\). Quanto vale \(P(A\cap B)\)?


    1. \(0.3500\)
    2. \(0.7200\)
    3. \(2.2857\)
    4. \(0.2800\)
    5. \(1.1500\)

    Solution

    A independência implica \[P(A\cap B)=P(A)P(B)=(0.8)(0.35) =0.2800.\]


  6. Question

    Se \(A\) e \(B\) são eventos independentes em um estudo de resultados de ensaios, qual afirmação é necessariamente verdadeira?


    1. Exatamente um dos eventos deve ocorrer.
    2. \(A^c\) e \(B^c\) também são independentes.
    3. \(P(A\mid B)=P(B\mid A)\) para quaisquer marginais.
    4. \(A\) e \(B\) são necessariamente disjuntos.
    5. \(P(A\cup B)=P(A)+P(B)\).

    Solution

    Temos \[P(A^c\cap B^c)=1-P(A)-P(B)+P(A)P(B) =P(A^c)P(B^c),\] logo os complementos são independentes. Independência não significa disjunção.


  7. Question

    Em duas moedas honestas lançadas uma vez, seja \(A=\) “cara no primeiro resultado”, \(B=\) “cara no segundo” e \(C=\) “os resultados são iguais”. Como se classificam esses eventos?


    1. São mutuamente independentes.
    2. Nenhum par é independente.
    3. Apenas \(A\) e \(B\) são independentes.
    4. São independentes dois a dois, mas não mutuamente independentes.
    5. São mutuamente exclusivos.

    Solution

    Cada evento tem probabilidade \(1/2\) e cada interseção de dois tem probabilidade \(1/4\). Porém, \[P(A\cap B\cap C)=\frac14\neq\frac18=P(A)P(B)P(C).\] Logo, independência aos pares não implica independência mútua.


  8. Question

    Uma urna contém 6 bolas vermelhas e 4 azuis. Duas são retiradas sem reposição. Dado que a primeira foi vermelha, qual é a probabilidade de a segunda também ser vermelha?


    1. \(0.5556\)
    2. \(0.6667\)
    3. \(0.4444\)
    4. \(0.5000\)
    5. \(0.6000\)

    Solution

    Restam 5 vermelhas entre 9 bolas. Assim, \[P(V_2\mid V_1)=\frac{5}{9} =0.5556.\]


  9. Question

    Dois componentes independentes têm confiabilidades 0.84 e 0.94. O sistema é do tipo paralelo. Qual é sua confiabilidade?


    1. \(0.9904\)
    2. \(0.8400\)
    3. \(0.0096\)
    4. \(0.9400\)
    5. \(0.2104\)

    Solution

    Em paralelo, o sistema falha apenas quando ambos os componentes falham. A confiabilidade é 0.9904.


  10. Question

    Ensaios de Bernoulli independentes têm probabilidade de sucesso \(p=0.4\). Qual é a probabilidade de o primeiro sucesso ocorrer exatamente no ensaio 2?


    1. \(0.1600\)
    2. \(0.3600\)
    3. \(0.8000\)
    4. \(0.2400\)
    5. \(0.6400\)

    Solution

    São necessários 1 fracassos seguidos de um sucesso: \[P(X=2)=(1-p)^{1}p =0.2400.\]
