Exam 1

  1. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,5\). Qual valor normaliza a pmf?


    1. \(0.20000\)
    2. \(0.03333\)
    3. \(0.16667\)
    4. \(0.83333\)
    5. \(0.06667\)

    Solution

    Deve valer \[1=\sum_{x=1}^{5}cx=c\frac{5(6)}2,\] portanto \(c=2/[5(6)]=0.06667\).


  2. Question

    Em 9 ensaios de Bernoulli independentes, com probabilidade de sucesso 0.45, qual é a probabilidade de exatamente 1 sucessos?


    1. \(0.00377\)
    2. \(0.00832\)
    3. \(4.05000\)
    4. \(0.96609\)
    5. \(0.03391\)

    Solution

    Para \(X\sim\operatorname{Binomial}(9,0.45)\), \[P(X=1)=\binom{9}{1}p^{1}(1-p)^{8} =0.03391.\]


  3. Question

    Se \(X\) é o número de tentativas independentes até o primeiro sucesso, com probabilidade de sucesso \(p=0.65\), calcule \(P(X=6)\).


    1. \(0.00341\)
    2. \(0.00184\)
    3. \(0.07542\)
    4. \(0.04061\)
    5. \(0.99816\)

    Solution

    Para o primeiro sucesso ocorrer na tentativa 6, devem ocorrer 5 fracassos e então um sucesso: \[P(X=6)=(1-p)^{5}p =0.00341.\]


  4. Question

    Uma variável tem suporte \((0,1,2,3,4)\) e pmf correspondente \((0.143, 0.381, 0.095, 0.238, 0.143)\). Qual é \(F_X(2)\)?


    1. \(0.6190\)
    2. \(0.4762\)
    3. \(0.3810\)
    4. \(0.0952\)
    5. \(0.5238\)

    Solution

    A FDA acumula todas as massas até 2: \[F_X(2)=\sum_{x\leq2}p_X(x) =0.6190.\]


  5. Question

    Uma variável \(X\) assume $-5, \(0\) e $5 com probabilidades 0.4, 0.15 e 0.45. Para \(Y=X^2\), qual é \(P(Y=25)\)?


    1. \(0.400\)
    2. \(0.450\)
    3. \(0.850\)
    4. \(0.150\)
    5. \(0.050\)

    Solution

    Os valores \(X=-5\) e \(X=5\) produzem o mesmo valor \(Y=25\). Portanto, somam-se as massas: \[P(Y=25)=0.4+0.45 =0.850.\]


  6. Question

    Uma variável tem suporte \((-2,-1,0,1,2)\) e probabilidades \((0.043, 0.304, 0.261, 0.087, 0.304)\). Se \(Y=|X|\), qual é \(P(Y=1)\)?


    1. \(0.6522\)
    2. \(0.3913\)
    3. \(0.6087\)
    4. \(0.3043\)
    5. \(0.0870\)

    Solution

    As pré-imagens de \(Y=1\) são \(X=-1\) e \(X=1\). Logo, \[P(Y=1)=P(X=-1)+P(X=1) =0.3913.\]


  7. Question

    Para a transformação injetiva \(Y=4X+0\), sabe-se que \(P(X=0)=0.3\). Qual é \(P(Y=0)\)?


    1. \(1.200\)
    2. \(0.700\)
    3. \(0.075\)
    4. \(0.060\)
    5. \(0.300\)

    Solution

    Como a transformação é injetiva, \(Y=0\) ocorre exatamente quando \(X=0\). Em transformações discretas não aparece fator jacobiano: \[P(Y=0)=0.3.\]


  8. Question

    De 18 itens, 5 são especiais. Uma amostra de 6 itens é retirada sem reposição. Qual é a probabilidade de obter exatamente 4 itens especiais?


    1. \(0.04658\)
    2. \(0.00027\)
    3. \(390.00000\)
    4. \(0.02101\)
    5. \(0.97899\)

    Solution

    A contagem é hipergeométrica: \[P(X=4)= \frac{\binom{5}{4}\binom{13}{2}} {\binom{18}{6}} =0.02101.\]


  9. Question

    Se \(X\sim\operatorname{Poisson}(\lambda)\), com \(\lambda=2\), qual é a razão \(p_X(6)/p_X(5)\)?


    1. \(2.0000\)
    2. \(0.3333\)
    3. \(0.2857\)
    4. \(3.0000\)
    5. \(6.0000\)

    Solution

    Pela pmf de Poisson, \[\frac{p(x+1)}{p(x)} =\frac{\lambda^{x+1}/(x+1)!}{\lambda^x/x!} =\frac{\lambda}{x+1} =0.3333.\]


  10. Question

    Para \(X=\mathbf{1}_A\) com \(P(A)=0.7\), qual é \(p_X(1)\)?


    1. \(0.700\)
    2. \(0.090\)
    3. \(0.490\)
    4. \(0.300\)
    5. \(0.210\)

    Solution

    A indicadora tem distribuição Bernoulli: \(P(X=1)=p\) e \(P(X=0)=1-p\). Assim, \(p_X(1)=0.700\).
