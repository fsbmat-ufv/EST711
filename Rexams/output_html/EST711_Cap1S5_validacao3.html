Exam 3

  1. Question

    Qual conjunto de propriedades é necessário para que \(G\) seja uma função de distribuição acumulada em \(\mathbb{R}\)?


    1. Ser estritamente crescente, contínua e possuir derivada positiva em todos os pontos.
    2. Ser não crescente, contínua à esquerda e integrar 1.
    3. Ser simétrica em torno de zero e limitada por uma densidade.
    4. Ser não decrescente, contínua à direita, com limites 0 em \(-\infty\) e 1 em \(+\infty\).
    5. Assumir apenas os valores 0 e 1.

    Solution

    Uma FDA é não decrescente e contínua à direita, com \(\lim_{x\to-\infty}F(x)=0\) e \(\lim_{x\to\infty}F(x)=1\). Ela pode ter saltos, trechos constantes e não precisa possuir densidade.


  2. Question

    Para uma variável aleatória contínua, \(F_X(0)=0.1\) e \(F_X(4)=0.65\). Qual é \(P(0<X\leq4)\)?


    1. \(0.750\)
    2. \(0.450\)
    3. \(0.100\)
    4. \(0.550\)
    5. \(0.650\)

    Solution

    \[P(a<X\leq b)=F_X(b)-F_X(a) =0.65-0.1=0.550.\]


  3. Question

    Uma FDA satisfaz \(F_X(4^-)=0.6\) e \(F_X(4)=0.9\). Qual é \(P(X=4)\)?


    1. \(0.900\)
    2. \(0.100\)
    3. \(0.600\)
    4. \(0.300\)
    5. \(1.500\)

    Solution

    A probabilidade pontual é o tamanho do salto: \[P(X=x_0)=F_X(x_0)-F_X(x_0^-) =0.9-0.6=0.3.\]


  4. Question

    Uma variável discreta tem suporte \((-1, 0, 1)\) e probabilidades correspondentes \((0.133, 0.533, 0.333)\). Qual é o tamanho do salto de \(F_X\) em \(x=0\)?


    1. \(0.333\)
    2. \(0.467\)
    3. \(0.133\)
    4. \(0.667\)
    5. \(0.533\)

    Solution

    O salto da FDA em um ponto é exatamente sua massa: \[F_X(x)-F_X(x^-)=P(X=x)=0.533.\]


  5. Question

    Na definição formal de variável aleatória, como em X é o número de sucessos, qual condição conecta a aplicação \(X:\Omega\to\mathbb{R}\) aos eventos?


    1. Cada evento deve ser levado por \(X\) a um único valor com probabilidade positiva.
    2. A aplicação \(X\) deve ser injetiva em todo espaço amostral.
    3. A imagem de \(X\) deve conter todos os números reais.
    4. Para todo \(\omega\), o valor \(X(\omega)\) deve ser uma probabilidade entre 0 e 1.
    5. Para todo conjunto boreliano \(B\), a pré-imagem \(\{\omega:X(\omega)\in B\}\) deve ser um evento.

    Solution

    A mensurabilidade exige que a pré-imagem de qualquer conjunto boreliano seja um evento ao qual \(P\) possa ser aplicado. Os valores de uma variável aleatória são números reais, não probabilidades.


  6. Question

    Uma distribuição mista possui um salto da FDA em \(x=1\), de \(F_X(x^-)=0.15\) para \(F_X(x)=0.35\). Qual massa discreta está concentrada nesse ponto?


    1. \(0.500\)
    2. \(0.350\)
    3. \(0.200\)
    4. \(0.150\)
    5. \(0.800\)

    Solution

    Mesmo em uma distribuição mista, a massa pontual é o salto: \[P(X=1)=0.35-0.15=0.2.\]


  7. Question

    Sabe-se que \(P(-5\leq X\leq-3)=1\). Qual afirmação sobre a FDA é necessariamente verdadeira?


    1. \(F_X(x)=0\) para \(x<-5\) e \(F_X(x)=1\) para \(x\geq-3\).
    2. \(F_X(x)=1\) para \(x<-5\) e \(F_X(x)=0\) para \(x>-3\).
    3. \(X\) deve ser uniformemente distribuída nesse intervalo.
    4. \(F_X(-5)=0\) e \(F_X(-3)=0\).
    5. \(F_X\) deve ser linear entre -5 e -3.

    Solution

    Não há massa abaixo de -5, logo \(F_X(x)=0\) para \(x<-5\). Toda a massa já foi acumulada em -3, então \(F_X(x)=1\) para \(x\geq-3\). Isso não determina a forma da distribuição dentro do intervalo.


  8. Question

    Para uma FDA \(F_X\) e um ponto \(x_0=0\), qual relação de continuidade é sempre válida?


    1. \(\lim_{h\downarrow0}F_X(x_0-h)=F_X(x_0)\).
    2. \(F_X\) é derivável em \(x_0\).
    3. \(\lim_{x\to x_0}F_X(x)=0\).
    4. \(\lim_{h\downarrow0}F_X(x_0+h)=F_X(x_0)\).
    5. \(F_X(x_0^-)=F_X(x_0)\), mesmo quando há massa em \(x_0\).

    Solution

    Toda FDA é contínua à direita. A continuidade à esquerda pode falhar quando existe massa pontual, pois então \(F_X(x_0)-F_X(x_0^-)=P(X=x_0)>0\).


  9. Question

    Se \(X=\mathbf{1}_A\), com \(P(A)=0.6\), qual é \(F_X(-2)\)?


    1. \(0.500\)
    2. \(0.400\)
    3. \(0.600\)
    4. \(0.240\)
    5. \(0.000\)

    Solution

    A indicadora vale 0 com probabilidade \(1-p\) e 1 com probabilidade \(p\). Portanto, \(F_X(x)=0\) se \(x<0\), \(F_X(x)=1-p\) se \(0\leq x<1\), e \(F_X(x)=1\) se \(x\geq1\). Para o valor fornecido, o resultado é 0.000.


  10. Question

    Para certo \(x=1\), sabe-se que \(F_X(x)=0.7\). Qual é \(P(X>x)\)?


    1. \(0.510\)
    2. \(0.588\)
    3. \(0.210\)
    4. \(0.300\)
    5. \(0.700\)

    Solution

    Como \(F_X(x)=P(X\leq x)\), \[P(X>x)=1-F_X(x)=1-0.7=0.300.\]
