Exam 3

  1. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,3\). Qual valor normaliza a pmf?


    1. \(0.75000\)
    2. \(0.08333\)
    3. \(0.25000\)
    4. \(0.16667\)
    5. \(0.33333\)

    Solution

    Deve valer \[1=\sum_{x=1}^{3}cx=c\frac{3(4)}2,\] portanto \(c=2/[3(4)]=0.16667\).


  2. Question

    Em 7 ensaios de Bernoulli independentes, com probabilidade de sucesso 0.35, qual é a probabilidade de exatamente 1 sucessos?


    1. \(0.02640\)
    2. \(0.18478\)
    3. \(2.45000\)
    4. \(0.00836\)
    5. \(0.81522\)

    Solution

    Para \(X\sim\operatorname{Binomial}(7,0.35)\), \[P(X=1)=\binom{7}{1}p^{1}(1-p)^{6} =0.18478.\]


  3. Question

    Se \(X\) é o número de tentativas independentes até o primeiro sucesso, com probabilidade de sucesso \(p=0.3\), calcule \(P(X=4)\).


    1. \(0.24010\)
    2. \(0.01890\)
    3. \(0.75990\)
    4. \(0.00810\)
    5. \(0.10290\)

    Solution

    Para o primeiro sucesso ocorrer na tentativa 4, devem ocorrer 3 fracassos e então um sucesso: \[P(X=4)=(1-p)^{3}p =0.10290.\]


  4. Question

    Uma variável tem suporte \((0,1,2,3,4)\) e pmf correspondente \((0.067, 0.200, 0.400, 0.200, 0.133)\). Qual é \(F_X(1)\)?


    1. \(0.9333\)
    2. \(0.2667\)
    3. \(0.7333\)
    4. \(0.0667\)
    5. \(0.2000\)

    Solution

    A FDA acumula todas as massas até 1: \[F_X(1)=\sum_{x\leq1}p_X(x) =0.2667.\]


  5. Question

    Uma variável \(X\) assume $-2, \(0\) e $2 com probabilidades 0.25, 0.1 e 0.65. Para \(Y=X^2\), qual é \(P(Y=4)\)?


    1. \(0.650\)
    2. \(0.250\)
    3. \(0.900\)
    4. \(0.400\)
    5. \(0.100\)

    Solution

    Os valores \(X=-2\) e \(X=2\) produzem o mesmo valor \(Y=4\). Portanto, somam-se as massas: \[P(Y=4)=0.25+0.65 =0.900.\]


  6. Question

    Uma variável tem suporte \((-2,-1,0,1,2)\) e probabilidades \((0.231, 0.192, 0.154, 0.269, 0.154)\). Se \(Y=|X|\), qual é \(P(Y=2)\)?


    1. \(0.1538\)
    2. \(1.0000\)
    3. \(0.3846\)
    4. \(0.2308\)
    5. \(0.6154\)

    Solution

    As pré-imagens de \(Y=2\) são \(X=-2\) e \(X=2\). Logo, \[P(Y=2)=P(X=-2)+P(X=2) =0.3846.\]


  7. Question

    Para a transformação injetiva \(Y=3X+-1\), sabe-se que \(P(X=-2)=0.7\). Qual é \(P(Y=-7)\)?


    1. \(0.175\)
    2. \(0.300\)
    3. \(0.700\)
    4. \(0.233\)
    5. \(2.100\)

    Solution

    Como a transformação é injetiva, \(Y=-7\) ocorre exatamente quando \(X=-2\). Em transformações discretas não aparece fator jacobiano: \[P(Y=-7)=0.7.\]


  8. Question

    De 24 itens, 10 são especiais. Uma amostra de 6 itens é retirada sem reposição. Qual é a probabilidade de obter exatamente 2 itens especiais?


    1. \(0.30153\)
    2. \(0.66533\)
    3. \(45045.00000\)
    4. \(0.33467\)
    5. \(0.00033\)

    Solution

    A contagem é hipergeométrica: \[P(X=2)= \frac{\binom{10}{2}\binom{14}{4}} {\binom{24}{6}} =0.33467.\]


  9. Question

    Se \(X\sim\operatorname{Poisson}(\lambda)\), com \(\lambda=2\), qual é a razão \(p_X(3)/p_X(2)\)?


    1. \(0.5000\)
    2. \(3.0000\)
    3. \(0.6667\)
    4. \(2.0000\)
    5. \(1.5000\)

    Solution

    Pela pmf de Poisson, \[\frac{p(x+1)}{p(x)} =\frac{\lambda^{x+1}/(x+1)!}{\lambda^x/x!} =\frac{\lambda}{x+1} =0.6667.\]


  10. Question

    Para \(X=\mathbf{1}_A\) com \(P(A)=0.15\), qual é \(p_X(0)\)?


    1. \(0.128\)
    2. \(0.023\)
    3. \(0.722\)
    4. \(0.850\)
    5. \(0.150\)

    Solution

    A indicadora tem distribuição Bernoulli: \(P(X=1)=p\) e \(P(X=0)=1-p\). Assim, \(p_X(0)=0.850\).
