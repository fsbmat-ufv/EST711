Exam 5

  1. Question

    Em 80 registros, 66 apresentam \(B\) e 51 apresentam simultaneamente \(A\) e \(B\). Qual é \(P(A\mid B)\)?


    1. \(0.2273\)
    2. \(0.1750\)
    3. \(0.8250\)
    4. \(0.6375\)
    5. \(0.7727\)

    Solution

    \[P(A\mid B)=\frac{P(A\cap B)}{P(B)} =\frac{51}{66}=0.7727.\]


  2. Question

    Se \(P(A)=0.7\) e \(P(B\mid A)=0.2\), qual é \(P(A\cap B)\)?


    1. \(0.1400\)
    2. \(0.2857\)
    3. \(0.2000\)
    4. \(0.9000\)
    5. \(0.5600\)

    Solution

    Pela regra da multiplicação, \[P(A\cap B)=P(A)P(B\mid A) =(0.7)(0.2)=0.1400.\]


  3. Question

    Três máquinas produzem proporções 0.30, 0.30, 0.40 dos itens, com probabilidades de defeito 0.05, 0.03, 0.12. Qual é a probabilidade de um item escolhido ao acaso ser defeituoso?


    1. \(0.0264\)
    2. \(0.9280\)
    3. \(0.2000\)
    4. \(0.0720\)
    5. \(0.0667\)

    Solution

    Pela probabilidade total, \[P(D)=\sum_{j=1}^3P(M_j)P(D\mid M_j) =0.0720.\]


  4. Question

    Uma condição tem prevalência 0.06. Um teste possui sensibilidade 0.90 e especificidade 0.99. Qual é \(P(\text{condição}\mid\text{positivo})\)?


    1. \(0.8517\)
    2. \(0.0600\)
    3. \(0.1483\)
    4. \(0.9900\)
    5. \(0.9000\)

    Solution

    Pelo teorema de Bayes, \[P(C\mid+)=\frac{P(+\mid C)P(C)} {P(+\mid C)P(C)+P(+\mid C^c)P(C^c)} =0.8517.\]


  5. Question

    Eventos \(A\) e \(B\) são independentes, com \(P(A)=0.35\) e \(P(B)=0.5\). Quanto vale \(P(A\cap B)\)?


    1. \(0.3500\)
    2. \(0.7000\)
    3. \(0.8500\)
    4. \(0.8250\)
    5. \(0.1750\)

    Solution

    A independência implica \[P(A\cap B)=P(A)P(B)=(0.35)(0.5) =0.1750.\]


  6. Question

    Se \(A\) e \(B\) são eventos independentes em um estudo de respostas de indivíduos, qual afirmação é necessariamente verdadeira?


    1. \(A^c\) e \(B^c\) também são independentes.
    2. \(P(A\cup B)=P(A)+P(B)\).
    3. \(P(A\mid B)=P(B\mid A)\) para quaisquer marginais.
    4. \(A\) e \(B\) são necessariamente disjuntos.
    5. Exatamente um dos eventos deve ocorrer.

    Solution

    Temos \[P(A^c\cap B^c)=1-P(A)-P(B)+P(A)P(B) =P(A^c)P(B^c),\] logo os complementos são independentes. Independência não significa disjunção.


  7. Question

    Em duas moedas honestas lançadas uma vez, seja \(A=\) “cara no primeiro resultado”, \(B=\) “cara no segundo” e \(C=\) “os resultados são iguais”. Como se classificam esses eventos?


    1. São independentes dois a dois, mas não mutuamente independentes.
    2. Apenas \(A\) e \(B\) são independentes.
    3. Nenhum par é independente.
    4. São mutuamente exclusivos.
    5. São mutuamente independentes.

    Solution

    Cada evento tem probabilidade \(1/2\) e cada interseção de dois tem probabilidade \(1/4\). Porém, \[P(A\cap B\cap C)=\frac14\neq\frac18=P(A)P(B)P(C).\] Logo, independência aos pares não implica independência mútua.


  8. Question

    Uma urna contém 4 bolas vermelhas e 7 azuis. Duas são retiradas sem reposição. Dado que a primeira foi vermelha, qual é a probabilidade de a segunda também ser vermelha?


    1. \(0.7000\)
    2. \(0.3000\)
    3. \(0.3636\)
    4. \(0.2727\)
    5. \(0.4000\)

    Solution

    Restam 3 vermelhas entre 10 bolas. Assim, \[P(V_2\mid V_1)=\frac{3}{10} =0.3000.\]


  9. Question

    Dois componentes independentes têm confiabilidades 0.98 e 0.9. O sistema é do tipo paralelo. Qual é sua confiabilidade?


    1. \(0.1180\)
    2. \(0.9980\)
    3. \(0.9000\)
    4. \(0.9800\)
    5. \(0.0020\)

    Solution

    Em paralelo, o sistema falha apenas quando ambos os componentes falham. A confiabilidade é 0.9980.


  10. Question

    Ensaios de Bernoulli independentes têm probabilidade de sucesso \(p=0.35\). Qual é a probabilidade de o primeiro sucesso ocorrer exatamente no ensaio 6?


    1. \(0.0018\)
    2. \(0.0754\)
    3. \(2.1000\)
    4. \(0.0406\)
    5. \(0.9246\)

    Solution

    São necessários 5 fracassos seguidos de um sucesso: \[P(X=6)=(1-p)^{5}p =0.0406.\]
