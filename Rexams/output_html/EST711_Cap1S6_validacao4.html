Exam 4

  1. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,4\). Qual valor normaliza a pmf?


    1. \(0.10000\)
    2. \(0.20000\)
    3. \(0.25000\)
    4. \(0.05000\)
    5. \(0.80000\)

    Solution

    Deve valer \[1=\sum_{x=1}^{4}cx=c\frac{4(5)}2,\] portanto \(c=2/[4(5)]=0.10000\).


  2. Question

    Em 7 ensaios de Bernoulli independentes, com probabilidade de sucesso 0.35, qual é a probabilidade de exatamente 3 sucessos?


    1. \(1.50063\)
    2. \(0.00765\)
    3. \(0.26787\)
    4. \(0.14424\)
    5. \(0.73213\)

    Solution

    Para \(X\sim\operatorname{Binomial}(7,0.35)\), \[P(X=3)=\binom{7}{3}p^{3}(1-p)^{4} =0.26787.\]


  3. Question

    Se \(X\) é o número de tentativas independentes até o primeiro sucesso, com probabilidade de sucesso \(p=0.3\), calcule \(P(X=5)\).


    1. \(0.00243\)
    2. \(0.83193\)
    3. \(0.16807\)
    4. \(0.07203\)
    5. \(0.00567\)

    Solution

    Para o primeiro sucesso ocorrer na tentativa 5, devem ocorrer 4 fracassos e então um sucesso: \[P(X=5)=(1-p)^{4}p =0.07203.\]


  4. Question

    Uma variável tem suporte \((0,1,2,3,4)\) e pmf correspondente \((0.250, 0.400, 0.200, 0.050, 0.100)\). Qual é \(F_X(3)\)?


    1. \(0.0500\)
    2. \(0.8500\)
    3. \(0.1500\)
    4. \(0.9000\)
    5. \(0.1000\)

    Solution

    A FDA acumula todas as massas até 3: \[F_X(3)=\sum_{x\leq3}p_X(x) =0.9000.\]


  5. Question

    Uma variável \(X\) assume $-3, \(0\) e $3 com probabilidades 0.25, 0.3 e 0.45. Para \(Y=X^2\), qual é \(P(Y=9)\)?


    1. \(0.300\)
    2. \(0.700\)
    3. \(0.200\)
    4. \(0.250\)
    5. \(0.450\)

    Solution

    Os valores \(X=-3\) e \(X=3\) produzem o mesmo valor \(Y=9\). Portanto, somam-se as massas: \[P(Y=9)=0.25+0.45 =0.700.\]


  6. Question

    Uma variável tem suporte \((-2,-1,0,1,2)\) e probabilidades \((0.067, 0.300, 0.200, 0.200, 0.233)\). Se \(Y=|X|\), qual é \(P(Y=2)\)?


    1. \(0.0667\)
    2. \(0.3000\)
    3. \(0.7000\)
    4. \(1.0000\)
    5. \(0.2333\)

    Solution

    As pré-imagens de \(Y=2\) são \(X=-2\) e \(X=2\). Logo, \[P(Y=2)=P(X=-2)+P(X=2) =0.3000.\]


  7. Question

    Para a transformação injetiva \(Y=2X+1\), sabe-se que \(P(X=2)=0.45\). Qual é \(P(Y=5)\)?


    1. \(0.550\)
    2. \(0.150\)
    3. \(0.450\)
    4. \(0.225\)
    5. \(0.900\)

    Solution

    Como a transformação é injetiva, \(Y=5\) ocorre exatamente quando \(X=2\). Em transformações discretas não aparece fator jacobiano: \[P(Y=5)=0.45.\]


  8. Question

    De 26 itens, 7 são especiais. Uma amostra de 5 itens é retirada sem reposição. Qual é a probabilidade de obter exatamente 4 itens especiais?


    1. \(0.01011\)
    2. \(665.00000\)
    3. \(0.01920\)
    4. \(0.00053\)
    5. \(0.98989\)

    Solution

    A contagem é hipergeométrica: \[P(X=4)= \frac{\binom{7}{4}\binom{19}{1}} {\binom{26}{5}} =0.01011.\]


  9. Question

    Se \(X\sim\operatorname{Poisson}(\lambda)\), com \(\lambda=2.5\), qual é a razão \(p_X(8)/p_X(7)\)?


    1. \(0.3125\)
    2. \(3.2000\)
    3. \(2.5000\)
    4. \(8.0000\)
    5. \(0.2778\)

    Solution

    Pela pmf de Poisson, \[\frac{p(x+1)}{p(x)} =\frac{\lambda^{x+1}/(x+1)!}{\lambda^x/x!} =\frac{\lambda}{x+1} =0.3125.\]


  10. Question

    Para \(X=\mathbf{1}_A\) com \(P(A)=0.65\), qual é \(p_X(0)\)?


    1. \(0.227\)
    2. \(0.350\)
    3. \(0.650\)
    4. \(0.122\)
    5. \(0.423\)

    Solution

    A indicadora tem distribuição Bernoulli: \(P(X=1)=p\) e \(P(X=0)=1-p\). Assim, \(p_X(0)=0.350\).
