Exam 5

  1. Question

    Uma variável discreta tem \(p_X(x)=c\,x\) para \(x=1,\ldots,7\). Qual valor normaliza a pmf?


    1. \(0.03571\)
    2. \(0.87500\)
    3. \(0.12500\)
    4. \(0.14286\)
    5. \(0.01786\)

    Solution

    Deve valer \[1=\sum_{x=1}^{7}cx=c\frac{7(8)}2,\] portanto \(c=2/[7(8)]=0.03571\).


  2. Question

    Em 12 ensaios de Bernoulli independentes, com probabilidade de sucesso 0.7, qual é a probabilidade de exatamente 3 sucessos?


    1. \(0.00149\)
    2. \(0.00001\)
    3. \(0.23970\)
    4. \(0.99851\)
    5. \(75.46000\)

    Solution

    Para \(X\sim\operatorname{Binomial}(12,0.7)\), \[P(X=3)=\binom{12}{3}p^{3}(1-p)^{9} =0.00149.\]


  3. Question

    Se \(X\) é o número de tentativas independentes até o primeiro sucesso, com probabilidade de sucesso \(p=0.2\), calcule \(P(X=7)\).


    1. \(0.20972\)
    2. \(0.05243\)
    3. \(0.00005\)
    4. \(0.79028\)
    5. \(0.00001\)

    Solution

    Para o primeiro sucesso ocorrer na tentativa 7, devem ocorrer 6 fracassos e então um sucesso: \[P(X=7)=(1-p)^{6}p =0.05243.\]


  4. Question

    Uma variável tem suporte \((0,1,2,3,4)\) e pmf correspondente \((0.182, 0.364, 0.091, 0.136, 0.227)\). Qual é \(F_X(3)\)?


    1. \(0.3636\)
    2. \(0.7727\)
    3. \(0.1364\)
    4. \(0.2273\)
    5. \(0.6364\)

    Solution

    A FDA acumula todas as massas até 3: \[F_X(3)=\sum_{x\leq3}p_X(x) =0.7727.\]


  5. Question

    Uma variável \(X\) assume $-5, \(0\) e $5 com probabilidades 0.45, 0.2 e 0.35. Para \(Y=X^2\), qual é \(P(Y=25)\)?


    1. \(0.450\)
    2. \(0.800\)
    3. \(-0.100\)
    4. \(0.200\)
    5. \(0.350\)

    Solution

    Os valores \(X=-5\) e \(X=5\) produzem o mesmo valor \(Y=25\). Portanto, somam-se as massas: \[P(Y=25)=0.45+0.35 =0.800.\]


  6. Question

    Uma variável tem suporte \((-2,-1,0,1,2)\) e probabilidades \((0.125, 0.167, 0.250, 0.167, 0.292)\). Se \(Y=|X|\), qual é \(P(Y=2)\)?


    1. \(0.4167\)
    2. \(0.1250\)
    3. \(0.5833\)
    4. \(1.0000\)
    5. \(0.2917\)

    Solution

    As pré-imagens de \(Y=2\) são \(X=-2\) e \(X=2\). Logo, \[P(Y=2)=P(X=-2)+P(X=2) =0.4167.\]


  7. Question

    Para a transformação injetiva \(Y=2X+1\), sabe-se que \(P(X=3)=0.6\). Qual é \(P(Y=7)\)?


    1. \(0.400\)
    2. \(0.600\)
    3. \(0.300\)
    4. \(0.200\)
    5. \(1.200\)

    Solution

    Como a transformação é injetiva, \(Y=7\) ocorre exatamente quando \(X=3\). Em transformações discretas não aparece fator jacobiano: \[P(Y=7)=0.6.\]


  8. Question

    De 30 itens, 9 são especiais. Uma amostra de 6 itens é retirada sem reposição. Qual é a probabilidade de obter exatamente 5 itens especiais?


    1. \(0.00021\)
    2. \(0.01021\)
    3. \(0.99554\)
    4. \(0.00446\)
    5. \(2646.00000\)

    Solution

    A contagem é hipergeométrica: \[P(X=5)= \frac{\binom{9}{5}\binom{21}{1}} {\binom{30}{6}} =0.00446.\]


  9. Question

    Se \(X\sim\operatorname{Poisson}(\lambda)\), com \(\lambda=4.5\), qual é a razão \(p_X(6)/p_X(5)\)?


    1. \(6.0000\)
    2. \(0.6429\)
    3. \(4.5000\)
    4. \(1.3333\)
    5. \(0.7500\)

    Solution

    Pela pmf de Poisson, \[\frac{p(x+1)}{p(x)} =\frac{\lambda^{x+1}/(x+1)!}{\lambda^x/x!} =\frac{\lambda}{x+1} =0.7500.\]


  10. Question

    Para \(X=\mathbf{1}_A\) com \(P(A)=0.65\), qual é \(p_X(0)\)?


    1. \(0.423\)
    2. \(0.650\)
    3. \(0.350\)
    4. \(0.122\)
    5. \(0.227\)

    Solution

    A indicadora tem distribuição Bernoulli: \(P(X=1)=p\) e \(P(X=0)=1-p\). Assim, \(p_X(0)=0.350\).
