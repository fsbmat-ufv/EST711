Exam 4

  1. Question

    Uma função é dada por \(f(x)=c x^{2}\) para \(0<x<4\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.04688\)
    2. \(0.18750\)
    3. \(0.01562\)
    4. \(1.33333\)
    5. \(0.25000\)

    Solution

    A normalização exige \[1=c\int_0^{4}x^{2}dx =c\frac{4^{3}}{3}.\] Logo, \(c=(3)/4^{3}=0.04688\).


  2. Question

    Se $f(x)=2x/9 para \(0<x<3\), qual é \(P(0.52<X<2.21)\)?


    1. \(0.4858\)
    2. \(0.5142\)
    3. \(0.5645\)
    4. \(0.1882\)
    5. \(0.2571\)

    Solution

    \[P(u<X<v)=\int_u^v\frac{2x}{a^2}dx =\frac{v^2-u^2}{a^2} =0.5142.\]


  3. Question

    Uma variável tem densidade \(f(x)=(4)x^{3}/1296\) em \((0,6)\). Qual é \(F_X(3.24)\)?


    1. \(0.9154\)
    2. \(0.1046\)
    3. \(0.1569\)
    4. \(0.5393\)
    5. \(0.0846\)

    Solution

    Para \(0<x<a\), \[F_X(x)=\int_0^x\frac{(k+1)t^k}{a^{k+1}}dt =\left(\frac{x}{a}\right)^{k+1}.\] No ponto informado, isso vale 0.0846.


  4. Question

    Se \(F_X(x)=(x/8)^{2}\) para \(0<x<8\), qual é o quantil de ordem \(p=0.8\)?


    1. \(0.1000\)
    2. \(3.5777\)
    3. \(5.1200\)
    4. \(7.1554\)
    5. \(6.4000\)

    Solution

    Resolve-se \(F_X(q_p)=p\): \[\left(\frac{q_p}{8}\right)^{2}=0.8, \qquad q_p=8\,p^{1/2} =7.1554.\]


  5. Question

    Se \(X\) tem densidade \(f_X\) e \(Y=4X+-2\), qual é a densidade de \(Y\)?


    1. \(f_Y(y)=\dfrac1{4}f_X\!\left(\dfrac{y--2}{4}\right)\).
    2. \(f_Y(y)=f_X((y+-2)/4)\).
    3. \(f_Y(y)=4f_X(4y+-2)\).
    4. \(f_Y(y)=f_X(y--2)\).
    5. \(f_Y(y)=\dfrac1{16}f_X(y/4--2)\).

    Solution

    A inversa é \(x=(y--2)/4\) e seu jacobiano absoluto é \(1/4\). Portanto, \[f_Y(y)=\frac1{4}f_X\left(\frac{y--2}{4}\right).\]


  6. Question

    Se \(X\sim U(0,3)\) e \(Y=X^2\), qual é \(f_Y(2.317)\)?


    1. \(0.10948\)
    2. \(0.50744\)
    3. \(0.21897\)
    4. \(0.07192\)
    5. \(1.01487\)

    Solution

    A inversa é \(x=\sqrt y\) e \(|dx/dy|=1/(2\sqrt y)\). Assim, \[f_Y(y)=\frac1{3}\frac1{2\sqrt y}, \quad 0<y<9,\] que no ponto dado vale 0.10948.


  7. Question

    Se \(X\sim U(-2,2)\) e \(Y=X^2\), qual é \(F_Y(0.935)\)?


    1. \(0.4836\)
    2. \(0.2338\)
    3. \(0.2585\)
    4. \(0.9671\)
    5. \(0.5164\)

    Solution

    Para \(0<y<a^2\), \[F_Y(y)=P(-\sqrt y\leq X\leq\sqrt y) =\frac{2\sqrt y}{2a}=\frac{\sqrt y}{a}.\] No ponto indicado, o valor é 0.4836.


  8. Question

    Uma distribuição é mistura de \(f_1\) e \(f_2\) com pesos 0.6 e 0.4. Para certo conjunto \(A\), \(P_1(A)=0.1\) e \(P_2(A)=0.45\). Qual é \(P(A)\) na mistura?


    1. \(0.7600\)
    2. \(0.2750\)
    3. \(0.2400\)
    4. \(0.0600\)
    5. \(0.0450\)

    Solution

    Probabilidades também se combinam pelos pesos da mistura: \[P(A)=0.6(0.1)+0.4(0.45) =0.2400.\]


  9. Question

    Se \(U\sim U(0,1)\) e \(Y=-\log(1-U)/1\), qual é a distribuição de \(Y\)?


    1. Exponencial com taxa \(1/1\), pois \(F_Y(y)=e^{-y/1}\).
    2. Exponencial com taxa 1, pois \(F_Y(y)=1-e^{-1y}\) para \(y>0\).
    3. Gama com forma 1 e taxa 1.
    4. Uniforme em \((0,1/1)\).
    5. Normal com média \(1/1\).

    Solution

    Para \(y>0\), \[P(Y\leq y)=P\{U\leq1-e^{-1y}\} =1-e^{-1y},\] que é a FDA exponencial de taxa 1.


  10. Question

    Uma variável \(X\) é absolutamente contínua e sua densidade em \(x_0=3\) vale \(f_X(x_0)=0.8\). Qual é \(P(X=x_0)\)?


    1. \(F_X(x_0)\).
    2. Não pode ser determinada sem integrar sobre toda a reta.
    3. 0.8.
    4. \(1-0.8\).
    5. \(0\).

    Solution

    Em distribuições absolutamente contínuas, conjuntos com um único ponto têm probabilidade zero. O valor da densidade não é uma probabilidade pontual: \[P(X=x_0)=\int_{x_0}^{x_0}f_X(x)\,dx=0.\]
