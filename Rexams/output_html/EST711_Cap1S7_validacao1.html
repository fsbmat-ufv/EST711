Exam 1

  1. Question

    Uma função é dada por \(f(x)=c x^{4}\) para \(0<x<5\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.00800\)
    2. \(0.20000\)
    3. \(1.00000\)
    4. \(0.00160\)
    5. \(0.00032\)

    Solution

    A normalização exige \[1=c\int_0^{5}x^{4}dx =c\frac{5^{5}}{5}.\] Logo, \(c=(5)/5^{5}=0.00160\).


  2. Question

    Se $f(x)=2x/36 para \(0<x<6\), qual é \(P(1.45<X<5.24)\)?


    1. \(0.3517\)
    2. \(0.1052\)
    3. \(0.7035\)
    4. \(0.2965\)
    5. \(0.6309\)

    Solution

    \[P(u<X<v)=\int_u^v\frac{2x}{a^2}dx =\frac{v^2-u^2}{a^2} =0.7035.\]


  3. Question

    Uma variável tem densidade \(f(x)=(3)x^{2}/8\) em \((0,2)\). Qual é \(F_X(1.06)\)?


    1. \(0.8519\)
    2. \(0.1481\)
    3. \(0.2799\)
    4. \(0.4199\)
    5. \(0.5291\)

    Solution

    Para \(0<x<a\), \[F_X(x)=\int_0^x\frac{(k+1)t^k}{a^{k+1}}dt =\left(\frac{x}{a}\right)^{k+1}.\] No ponto informado, isso vale 0.1481.


  4. Question

    Se \(F_X(x)=(x/7)^{3}\) para \(0<x<7\), qual é o quantil de ordem \(p=0.85\)?


    1. \(3.7193\)
    2. \(5.9500\)
    3. \(4.2989\)
    4. \(0.1214\)
    5. \(6.6309\)

    Solution

    Resolve-se \(F_X(q_p)=p\): \[\left(\frac{q_p}{7}\right)^{3}=0.85, \qquad q_p=7\,p^{1/3} =6.6309.\]


  5. Question

    Se \(X\) tem densidade \(f_X\) e \(Y=3X+-3\), qual é a densidade de \(Y\)?


    1. \(f_Y(y)=3f_X(3y+-3)\).
    2. \(f_Y(y)=\dfrac1{3}f_X\!\left(\dfrac{y--3}{3}\right)\).
    3. \(f_Y(y)=f_X(y--3)\).
    4. \(f_Y(y)=f_X((y+-3)/3)\).
    5. \(f_Y(y)=\dfrac1{9}f_X(y/3--3)\).

    Solution

    A inversa é \(x=(y--3)/3\) e seu jacobiano absoluto é \(1/3\). Portanto, \[f_Y(y)=\frac1{3}f_X\left(\frac{y--3}{3}\right).\]


  6. Question

    Se \(X\sim U(0,2)\) e \(Y=X^2\), qual é \(f_Y(1.685)\)?


    1. \(0.14841\)
    2. \(0.38524\)
    3. \(0.19262\)
    4. \(0.64895\)
    5. \(1.29789\)

    Solution

    A inversa é \(x=\sqrt y\) e \(|dx/dy|=1/(2\sqrt y)\). Assim, \[f_Y(y)=\frac1{2}\frac1{2\sqrt y}, \quad 0<y<4,\] que no ponto dado vale 0.19262.


  7. Question

    Se \(X\sim U(-6,6)\) e \(Y=X^2\), qual é \(F_Y(30.052)\)?


    1. \(0.0152\)
    2. \(1.8273\)
    3. \(0.9137\)
    4. \(0.8348\)
    5. \(0.0863\)

    Solution

    Para \(0<y<a^2\), \[F_Y(y)=P(-\sqrt y\leq X\leq\sqrt y) =\frac{2\sqrt y}{2a}=\frac{\sqrt y}{a}.\] No ponto indicado, o valor é 0.9137.


  8. Question

    Uma distribuição é mistura de \(f_1\) e \(f_2\) com pesos 0.35 e 0.65. Para certo conjunto \(A\), \(P_1(A)=0.65\) e \(P_2(A)=0.5\). Qual é \(P(A)\) na mistura?


    1. \(0.4475\)
    2. \(0.3250\)
    3. \(0.5750\)
    4. \(0.2275\)
    5. \(0.5525\)

    Solution

    Probabilidades também se combinam pelos pesos da mistura: \[P(A)=0.35(0.65)+0.65(0.5) =0.5525.\]


  9. Question

    Se \(U\sim U(0,1)\) e \(Y=-\log(1-U)/3\), qual é a distribuição de \(Y\)?


    1. Normal com média \(1/3\).
    2. Exponencial com taxa \(1/3\), pois \(F_Y(y)=e^{-y/3}\).
    3. Gama com forma 3 e taxa 1.
    4. Uniforme em \((0,1/3)\).
    5. Exponencial com taxa 3, pois \(F_Y(y)=1-e^{-3y}\) para \(y>0\).

    Solution

    Para \(y>0\), \[P(Y\leq y)=P\{U\leq1-e^{-3y}\} =1-e^{-3y},\] que é a FDA exponencial de taxa 3.


  10. Question

    Uma variável \(X\) é absolutamente contínua e sua densidade em \(x_0=-2\) vale \(f_X(x_0)=1.3\). Qual é \(P(X=x_0)\)?


    1. \(F_X(x_0)\).
    2. Não pode ser determinada sem integrar sobre toda a reta.
    3. 1.3.
    4. \(0\).
    5. \(1-1.3\).

    Solution

    Em distribuições absolutamente contínuas, conjuntos com um único ponto têm probabilidade zero. O valor da densidade não é uma probabilidade pontual: \[P(X=x_0)=\int_{x_0}^{x_0}f_X(x)\,dx=0.\]
