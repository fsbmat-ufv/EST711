Exam 2

  1. Question

    Uma função é dada por \(f(x)=c x^{4}\) para \(0<x<5\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.00800\)
    2. \(0.00032\)
    3. \(0.00160\)
    4. \(1.00000\)
    5. \(0.20000\)

    Solution

    A normalização exige \[1=c\int_0^{5}x^{4}dx =c\frac{5^{5}}{5}.\] Logo, \(c=(5)/5^{5}=0.00160\).


  2. Question

    Se $f(x)=2x/4 para \(0<x<2\), qual é \(P(0.68<X<1.92)\)?


    1. \(0.6180\)
    2. \(0.8042\)
    3. \(0.1958\)
    4. \(0.3090\)
    5. \(0.4021\)

    Solution

    \[P(u<X<v)=\int_u^v\frac{2x}{a^2}dx =\frac{v^2-u^2}{a^2} =0.8042.\]


  3. Question

    Uma variável tem densidade \(f(x)=(4)x^{3}/625\) em \((0,5)\). Qual é \(F_X(1.51)\)?


    1. \(0.0222\)
    2. \(0.0084\)
    3. \(0.0277\)
    4. \(0.3026\)
    5. \(0.9916\)

    Solution

    Para \(0<x<a\), \[F_X(x)=\int_0^x\frac{(k+1)t^k}{a^{k+1}}dt =\left(\frac{x}{a}\right)^{k+1}.\] No ponto informado, isso vale 0.0084.


  4. Question

    Se \(F_X(x)=(x/7)^{3}\) para \(0<x<7\), qual é o quantil de ordem \(p=0.35\)?


    1. \(2.4500\)
    2. \(6.0637\)
    3. \(0.0500\)
    4. \(0.3001\)
    5. \(4.9331\)

    Solution

    Resolve-se \(F_X(q_p)=p\): \[\left(\frac{q_p}{7}\right)^{3}=0.35, \qquad q_p=7\,p^{1/3} =4.9331.\]


  5. Question

    Se \(X\) tem densidade \(f_X\) e \(Y=2X+2\), qual é a densidade de \(Y\)?


    1. \(f_Y(y)=f_X(y-2)\).
    2. \(f_Y(y)=f_X((y+2)/2)\).
    3. \(f_Y(y)=\dfrac1{2}f_X\!\left(\dfrac{y-2}{2}\right)\).
    4. \(f_Y(y)=2f_X(2y+2)\).
    5. \(f_Y(y)=\dfrac1{4}f_X(y/2-2)\).

    Solution

    A inversa é \(x=(y-2)/2\) e seu jacobiano absoluto é \(1/2\). Portanto, \[f_Y(y)=\frac1{2}f_X\left(\frac{y-2}{2}\right).\]


  6. Question

    Se \(X\sim U(0,6)\) e \(Y=X^2\), qual é \(f_Y(7.724)\)?


    1. \(0.46319\)
    2. \(0.92639\)
    3. \(0.05997\)
    4. \(0.01079\)
    5. \(0.02999\)

    Solution

    A inversa é \(x=\sqrt y\) e \(|dx/dy|=1/(2\sqrt y)\). Assim, \[f_Y(y)=\frac1{6}\frac1{2\sqrt y}, \quad 0<y<36,\] que no ponto dado vale 0.02999.


  7. Question

    Se \(X\sim U(-5,5)\) e \(Y=X^2\), qual é \(F_Y(21.792)\)?


    1. \(0.8717\)
    2. \(1.8673\)
    3. \(0.0214\)
    4. \(0.0664\)
    5. \(0.9336\)

    Solution

    Para \(0<y<a^2\), \[F_Y(y)=P(-\sqrt y\leq X\leq\sqrt y) =\frac{2\sqrt y}{2a}=\frac{\sqrt y}{a}.\] No ponto indicado, o valor é 0.9336.


  8. Question

    Uma distribuição é mistura de \(f_1\) e \(f_2\) com pesos 0.35 e 0.65. Para certo conjunto \(A\), \(P_1(A)=0.3\) e \(P_2(A)=0.55\). Qual é \(P(A)\) na mistura?


    1. \(0.4625\)
    2. \(0.4250\)
    3. \(0.5375\)
    4. \(0.1050\)
    5. \(0.1650\)

    Solution

    Probabilidades também se combinam pelos pesos da mistura: \[P(A)=0.35(0.3)+0.65(0.55) =0.4625.\]


  9. Question

    Se \(U\sim U(0,1)\) e \(Y=-\log(1-U)/1.5\), qual é a distribuição de \(Y\)?


    1. Exponencial com taxa \(1/1.5\), pois \(F_Y(y)=e^{-y/1.5}\).
    2. Uniforme em \((0,1/1.5)\).
    3. Exponencial com taxa 1.5, pois \(F_Y(y)=1-e^{-1.5y}\) para \(y>0\).
    4. Normal com média \(1/1.5\).
    5. Gama com forma 1.5 e taxa 1.

    Solution

    Para \(y>0\), \[P(Y\leq y)=P\{U\leq1-e^{-1.5y}\} =1-e^{-1.5y},\] que é a FDA exponencial de taxa 1.5.


  10. Question

    Uma variável \(X\) é absolutamente contínua e sua densidade em \(x_0=-0.5\) vale \(f_X(x_0)=0.7\). Qual é \(P(X=x_0)\)?


    1. 0.7.
    2. Não pode ser determinada sem integrar sobre toda a reta.
    3. \(1-0.7\).
    4. \(F_X(x_0)\).
    5. \(0\).

    Solution

    Em distribuições absolutamente contínuas, conjuntos com um único ponto têm probabilidade zero. O valor da densidade não é uma probabilidade pontual: \[P(X=x_0)=\int_{x_0}^{x_0}f_X(x)\,dx=0.\]
