Exam 5

  1. Question

    Ao modelar um experimento físico, uma função \(P\) é definida sobre os eventos de um espaço amostral \(\Omega\). Qual condição faz parte dos axiomas de probabilidade?


    1. Para eventos dois a dois disjuntos \(A_1,A_2,\ldots\), vale \(P(\bigcup_i A_i)=\sum_iP(A_i)\).
    2. Se \(A\subset B\), então \(P(A)>P(B)\).
    3. A probabilidade do espaço amostral pode assumir qualquer constante positiva.
    4. Para quaisquer eventos \(A\) e \(B\), vale sempre \(P(A\cup B)=P(A)+P(B)\).
    5. Todo evento não vazio deve ter probabilidade estritamente positiva.

    Solution

    Os axiomas exigem não negatividade, \(P(\Omega)=1\) e aditividade enumerável para eventos dois a dois disjuntos. Para eventos que se sobrepõem, a interseção precisa ser descontada. Eventos não vazios podem ter probabilidade zero.


  2. Question

    Em certo experimento, \(P(A)=0.12\). Qual é \(P(A^c)\)?


    1. \(0.1200\)
    2. \(0.8800\)
    3. \(0.8929\)
    4. \(0.7744\)
    5. \(0.0144\)

    Solution

    Como \(A\) e \(A^c\) são disjuntos e sua união é \(\Omega\), \[P(A^c)=1-P(A)=1-0.12=0.8800.\]


  3. Question

    Em uma população de 150 unidades, 60 satisfazem \(A\), 51 satisfazem \(B\) e 40 satisfazem ambos. Selecionando uma unidade ao acaso, qual é \(P(A\cup B)\)?


    1. \(0.473\)
    2. \(0.527\)
    3. \(0.740\)
    4. \(1.007\)
    5. \(0.267\)

    Solution

    Pela fórmula da adição, \[P(A\cup B)=P(A)+P(B)-P(A\cap B) =\frac{60+51-40}{150} =0.473.\]


  4. Question

    Para três eventos, são conhecidas as probabilidades \[(P(A),P(B),P(C))=(0.55,0.3,0.4),\] \[(P(A\cap B),P(A\cap C),P(B\cap C))=(0.14,0.06,0.15)\] e \(P(A\cap B\cap C)=0.04\). Calcule \(P(A\cup B\cup C)\).


    1. \(1.290\)
    2. \(1.250\)
    3. \(0.940\)
    4. \(0.060\)
    5. \(0.900\)

    Solution

    A inclusão-exclusão fornece \[P(A\cup B\cup C)=\sum P(A_i)-\sum P(A_i\cap A_j)+P(A\cap B\cap C) =0.940.\]


  5. Question

    Uma comissão de 5 pessoas é escolhida uniformemente, sem reposição, entre 7 mulheres e 9 homens. Qual é a probabilidade de a comissão conter exatamente 3 mulheres?


    1. \(0.6000\)
    2. \(0.0080\)
    3. \(0.0179\)
    4. \(0.2885\)
    5. \(0.7115\)

    Solution

    Há \(\binom{16}{5}\) comissões possíveis. As favoráveis escolhem 3 mulheres e 2 homens: \[P=\frac{\binom{7}{3}\binom{9}{2}} {\binom{16}{5}} =0.2885.\]


  6. Question

    Supondo 365 dias equiprováveis e ignorando anos bissextos, qual é a probabilidade de haver ao menos duas pessoas com o mesmo aniversário em um grupo de 25 pessoas?


    1. \(0.4313\)
    2. \(0.8219\)
    3. \(0.0663\)
    4. \(0.5687\)
    5. \(0.0685\)

    Solution

    Pelo complemento, \[P(\text{coincidência})=1-\prod_{j=0}^{24}\frac{365-j}{365} =0.5687.\]


  7. Question

    Para 4 eventos, as probabilidades individuais são 0.13, 0.14, 0.18, 0.11. Sem informação sobre interseções, qual limite superior é garantido por Boole para \(P(\bigcup_iA_i)\)?


    1. \(0.1400\)
    2. \(0.5600\)
    3. \(0.1800\)
    4. \(0.0004\)
    5. \(0.4400\)

    Solution

    A desigualdade de Boole fornece \[P\left(\bigcup_iA_i\right)\leq\sum_iP(A_i).\] Como uma probabilidade não excede 1, o limite é \(\min\{1,\sum_iP(A_i)\}=0.5600\).


  8. Question

    Considere eventos crescentes \(A_1\subseteq A_2\subseteq\cdots\), por exemplo A_n = {|X| <= n}. Qual propriedade é correta?


    1. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}P(A_n)\).
    2. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}[1-P(A_n)]\).
    3. \(P(\bigcup_{n=1}^{\infty}A_n)=\prod_{n=1}^{\infty}P(A_n)\).
    4. A igualdade com o limite exige que os \(A_n\) sejam disjuntos.
    5. \(P(\bigcap_{n=1}^{\infty}A_n)=\sum_{n=1}^{\infty}P(A_n)\).

    Solution

    A probabilidade é contínua por baixo: \[P\left(\bigcup_{n=1}^{\infty}A_n\right)=\lim_{n\to\infty}P(A_n).\] A condição necessária aqui é a monotonicidade crescente, não a disjunção.


  9. Question

    Um dado honesto de 8 faces, numeradas de 1 a 8, é lançado. Qual é a probabilidade de o resultado ser pelo menos 6?


    1. \(0.3750\)
    2. \(0.6250\)
    3. \(0.2500\)
    4. \(0.7500\)
    5. \(0.1667\)

    Solution

    Há 3 resultados favoráveis entre 8 equiprováveis. Logo, \[P(X\geq 6)=\frac{3}{8} =0.3750.\]


  10. Question

    Sabe-se apenas que \(P(A)=0.9\) e \(P(B)=0.75\). Qual é o limite inferior de Bonferroni para \(P(A\cap B)\)?


    1. \(0.650\)
    2. \(0.750\)
    3. \(0.350\)
    4. \(-0.650\)
    5. \(0.675\)

    Solution

    Como \(P(A\cup B)\leq1\), \[P(A\cap B)\geq P(A)+P(B)-1 =0.650.\]
