Exam 4

  1. Question

    Ao modelar uma pesquisa amostral, uma função \(P\) é definida sobre os eventos de um espaço amostral \(\Omega\). Qual condição faz parte dos axiomas de probabilidade?


    1. Para eventos dois a dois disjuntos \(A_1,A_2,\ldots\), vale \(P(\bigcup_i A_i)=\sum_iP(A_i)\).
    2. Para quaisquer eventos \(A\) e \(B\), vale sempre \(P(A\cup B)=P(A)+P(B)\).
    3. A probabilidade do espaço amostral pode assumir qualquer constante positiva.
    4. Se \(A\subset B\), então \(P(A)>P(B)\).
    5. Todo evento não vazio deve ter probabilidade estritamente positiva.

    Solution

    Os axiomas exigem não negatividade, \(P(\Omega)=1\) e aditividade enumerável para eventos dois a dois disjuntos. Para eventos que se sobrepõem, a interseção precisa ser descontada. Eventos não vazios podem ter probabilidade zero.


  2. Question

    Em certo experimento, \(P(A)=0.48\). Qual é \(P(A^c)\)?


    1. \(0.4800\)
    2. \(0.2704\)
    3. \(0.2304\)
    4. \(0.6757\)
    5. \(0.5200\)

    Solution

    Como \(A\) e \(A^c\) são disjuntos e sua união é \(\Omega\), \[P(A^c)=1-P(A)=1-0.48=0.5200.\]


  3. Question

    Em uma população de 80 unidades, 38 satisfazem \(A\), 53 satisfazem \(B\) e 32 satisfazem ambos. Selecionando uma unidade ao acaso, qual é \(P(A\cup B)\)?


    1. \(1.138\)
    2. \(1.538\)
    3. \(0.262\)
    4. \(0.400\)
    5. \(0.738\)

    Solution

    Pela fórmula da adição, \[P(A\cup B)=P(A)+P(B)-P(A\cap B) =\frac{38+53-32}{80} =0.738.\]


  4. Question

    Para três eventos, são conhecidas as probabilidades \[(P(A),P(B),P(C))=(0.45,0.3,0.3),\] \[(P(A\cap B),P(A\cap C),P(B\cap C))=(0.12,0.06,0.05)\] e \(P(A\cap B\cap C)=0.04\). Calcule \(P(A\cup B\cup C)\).


    1. \(0.140\)
    2. \(0.860\)
    3. \(1.050\)
    4. \(1.090\)
    5. \(0.820\)

    Solution

    A inclusão-exclusão fornece \[P(A\cup B\cup C)=\sum P(A_i)-\sum P(A_i\cap A_j)+P(A\cap B\cap C) =0.860.\]


  5. Question

    Uma comissão de 5 pessoas é escolhida uniformemente, sem reposição, entre 6 mulheres e 9 homens. Qual é a probabilidade de a comissão conter exatamente 2 mulheres?


    1. \(0.0050\)
    2. \(0.0952\)
    3. \(0.4000\)
    4. \(0.4196\)
    5. \(0.5804\)

    Solution

    Há \(\binom{15}{5}\) comissões possíveis. As favoráveis escolhem 2 mulheres e 3 homens: \[P=\frac{\binom{6}{2}\binom{9}{3}} {\binom{15}{5}} =0.4196.\]


  6. Question

    Supondo 365 dias equiprováveis e ignorando anos bissextos, qual é a probabilidade de haver ao menos duas pessoas com o mesmo aniversário em um grupo de 24 pessoas?


    1. \(0.5383\)
    2. \(0.7562\)
    3. \(0.0637\)
    4. \(0.4617\)
    5. \(0.0658\)

    Solution

    Pelo complemento, \[P(\text{coincidência})=1-\prod_{j=0}^{23}\frac{365-j}{365} =0.5383.\]


  7. Question

    Para 3 eventos, as probabilidades individuais são 0.10, 0.06, 0.15. Sem informação sobre interseções, qual limite superior é garantido por Boole para \(P(\bigcup_iA_i)\)?


    1. \(0.1500\)
    2. \(0.1033\)
    3. \(0.0009\)
    4. \(0.6900\)
    5. \(0.3100\)

    Solution

    A desigualdade de Boole fornece \[P\left(\bigcup_iA_i\right)\leq\sum_iP(A_i).\] Como uma probabilidade não excede 1, o limite é \(\min\{1,\sum_iP(A_i)\}=0.3100\).


  8. Question

    Considere eventos crescentes \(A_1\subseteq A_2\subseteq\cdots\), por exemplo A_n = {X <= n}. Qual propriedade é correta?


    1. \(P(\bigcap_{n=1}^{\infty}A_n)=\sum_{n=1}^{\infty}P(A_n)\).
    2. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}P(A_n)\).
    3. \(P(\bigcup_{n=1}^{\infty}A_n)=\lim_{n\to\infty}[1-P(A_n)]\).
    4. \(P(\bigcup_{n=1}^{\infty}A_n)=\prod_{n=1}^{\infty}P(A_n)\).
    5. A igualdade com o limite exige que os \(A_n\) sejam disjuntos.

    Solution

    A probabilidade é contínua por baixo: \[P\left(\bigcup_{n=1}^{\infty}A_n\right)=\lim_{n\to\infty}P(A_n).\] A condição necessária aqui é a monotonicidade crescente, não a disjunção.


  9. Question

    Um dado honesto de 12 faces, numeradas de 1 a 12, é lançado. Qual é a probabilidade de o resultado ser pelo menos 10?


    1. \(0.2500\)
    2. \(0.8333\)
    3. \(0.7500\)
    4. \(0.1000\)
    5. \(0.1667\)

    Solution

    Há 3 resultados favoráveis entre 12 equiprováveis. Logo, \[P(X\geq 10)=\frac{3}{12} =0.2500.\]


  10. Question

    Sabe-se apenas que \(P(A)=0.85\) e \(P(B)=0.85\). Qual é o limite inferior de Bonferroni para \(P(A\cap B)\)?


    1. \(0.850\)
    2. \(0.723\)
    3. \(-0.700\)
    4. \(0.300\)
    5. \(0.700\)

    Solution

    Como \(P(A\cup B)\leq1\), \[P(A\cap B)\geq P(A)+P(B)-1 =0.700.\]
