Exam 1

  1. Question

    Sabe-se que \(E(|X|^{7})<\infty\). Qual conclusão é garantida?


    1. \(E(|X|^{6})<\infty\).
    2. \(X\) possui suporte limitado.
    3. A fgm de \(X\) existe para todo \(t\).
    4. \(X\) tem distribuição simétrica.
    5. \(E(|X|^{8})<\infty\).

    Solution

    A existência de um momento absoluto de ordem \(m\) implica a existência de todos os momentos absolutos de ordens menores. Isso decorre de \(|x|^k\leq1+|x|^m\) para \(k\leq m\).


  2. Question

    Uma variável não negativa tem \(E(X)=19\). Pelo limite de Markov, qual é o maior valor garantido para \(P(X\geq36)\)?


    1. \(1.8947\)
    2. \(0.0147\)
    3. \(0.5278\)
    4. \(19.0000\)
    5. \(0.4722\)

    Solution

    A desigualdade de Markov dá \[P(X\geq c)\leq\frac{E(X)}c =\frac{19}{36}=0.5278.\]


  3. Question

    Sabe-se que \(E(X^2)=10\). Aplicando Markov a \(X^2\), qual limite superior se obtém para \(P(|X|\geq9)\)?


    1. \(0.1235\)
    2. \(0.3514\)
    3. \(1.0000\)
    4. \(10.0000\)
    5. \(0.8765\)

    Solution

    O evento \(|X|\geq c\) equivale a \(X^2\geq c^2\). Como \(X^2\geq0\), \[P(|X|\geq c)\leq\frac{E(X^2)}{c^2} =0.1235.\]


  4. Question

    Uma variável tem média 0 e desvio-padrão 1.5. Pelo limite de Chebyshev, qual é o limite superior para \(P(|X-\mu|\geq1.5\sigma)\)?


    1. \(2.2500\)
    2. \(0.4444\)
    3. \(0.5556\)
    4. \(0.6667\)
    5. \(1.0000\)

    Solution

    Chebyshev afirma \[P(|X-\mu|\geq k\sigma)\leq\frac1{k^2} =0.4444.\] O limite independe dos valores específicos de \(\mu\) e \(\sigma\).


  5. Question

    Sem supor uma família de distribuições, qual limite inferior Chebyshev garante para \(P(|X-\mu|<2.5\sigma)\)?


    1. \(0.6000\)
    2. \(0.8400\)
    3. \(0.7143\)
    4. \(0.1600\)
    5. \(0.4000\)

    Solution

    Pelo complemento da forma usual de Chebyshev, \[P(|X-\mu|<k\sigma)\geq1-\frac1{k^2} =0.8400.\]


  6. Question

    Se \(X\sim U(-\sqrt3,\sqrt3)\), então \(E(X)=0\) e \(\operatorname{Var}(X)=1\). Para \(k=1.2\), qual é o valor exato de \(P(|X|\geq k)\)?


    1. \(0.6928\)
    2. \(0.3072\)
    3. \(0.8333\)
    4. \(0.3056\)
    5. \(0.6944\)

    Solution

    O comprimento total do suporte é \(2\sqrt3\), e o intervalo central \((-k,k)\) tem comprimento \(2k\). Logo, \[P(|X|\geq k)=1-\frac{k}{\sqrt3} =0.3072.\] Chebyshev daria apenas o limite superior \(1/k^2=0.6944\).


  7. Question

    Se \(\phi\) é convexa, como a função -log(x) em seu domínio, e as esperanças existem, qual é a desigualdade de Jensen?


    1. \(\phi(E[X])\geq E[\phi(X)]\).
    2. \(\phi(E[X])=E(X)\phi(1)\).
    3. \(E[\phi(X)]\leq E(X)\).
    4. \(E[\phi(X)]=\phi(X)\) quase certamente.
    5. \(\phi(E[X])\leq E[\phi(X)]\).

    Solution

    Para funções convexas, \[\phi(E[X])\leq E[\phi(X)].\] O sentido se inverte para funções côncavas.


  8. Question

    Se \(E(X)=3.5\), qual limite inferior Jensen fornece para \(E(X^2)\)? Considere \(\phi(x)=x^2\).


    1. \(4.444\)
    2. \(7.500\)
    3. \(3.500\)
    4. \(19.750\)
    5. \(12.250\)

    Solution

    Como \(x^2\) é convexa, \[[E(X)]^2\leq E(X^2).\] Assim, \(E(X^2)\geq(3.5)^2=12.250\). O valor 19.75 foi usado apenas para gerar distratores plausíveis.


  9. Question

    Para os números positivos \((7, 1, 1, 1)\), qual é a média harmônica?


    1. \(1.2727\)
    2. \(0.7857\)
    3. \(2.5000\)
    4. \(1.6266\)
    5. \(10.0000\)

    Solution

    A média harmônica é \[HM=\frac{n}{\sum_i1/x_i}=1.2727.\] Além disso, Jensen permite estabelecer \(HM\leq GM\leq AM\); aqui \(GM=1.6266\) e \(AM=2.5000\).


  10. Question

    Se \(\phi=exp(x)\) é estritamente convexa e \(E[\phi(X)]<\infty\), quando ocorre igualdade em Jensen, \(\phi(E[X])=E[\phi(X)]\)?


    1. Sempre que \(E(X)=0\).
    2. Somente quando \(X\) é constante quase certamente.
    3. Somente quando \(\operatorname{Var}(X)=1\).
    4. Sempre que a distribuição de \(X\) é simétrica.
    5. Para toda variável contínua.

    Solution

    Para uma função estritamente convexa, Jensen é estrita quando \(X\) é não degenerada. A igualdade ocorre precisamente quando toda a massa de \(X\) está concentrada em um único valor, isto é, quando \(X\) é constante quase certamente.
