Exam 4

  1. Question

    Sabe-se que \(E(|X|^{5})<\infty\). Qual conclusão é garantida?


    1. A fgm de \(X\) existe para todo \(t\).
    2. \(E(|X|^{6})<\infty\).
    3. \(X\) tem distribuição simétrica.
    4. \(E(|X|^{1})<\infty\).
    5. \(X\) possui suporte limitado.

    Solution

    A existência de um momento absoluto de ordem \(m\) implica a existência de todos os momentos absolutos de ordens menores. Isso decorre de \(|x|^k\leq1+|x|^m\) para \(k\leq m\).


  2. Question

    Uma variável não negativa tem \(E(X)=7\). Pelo limite de Markov, qual é o maior valor garantido para \(P(X\geq15)\)?


    1. \(0.5333\)
    2. \(2.1429\)
    3. \(7.0000\)
    4. \(0.4667\)
    5. \(0.0311\)

    Solution

    A desigualdade de Markov dá \[P(X\geq c)\leq\frac{E(X)}c =\frac{7}{15}=0.4667.\]


  3. Question

    Sabe-se que \(E(X^2)=4\). Aplicando Markov a \(X^2\), qual limite superior se obtém para \(P(|X|\geq15)\)?


    1. \(4.0000\)
    2. \(0.2667\)
    3. \(0.1333\)
    4. \(0.9822\)
    5. \(0.0178\)

    Solution

    O evento \(|X|\geq c\) equivale a \(X^2\geq c^2\). Como \(X^2\geq0\), \[P(|X|\geq c)\leq\frac{E(X^2)}{c^2} =0.0178.\]


  4. Question

    Uma variável tem média 0 e desvio-padrão 4. Pelo limite de Chebyshev, qual é o limite superior para \(P(|X-\mu|\geq3\sigma)\)?


    1. \(9.0000\)
    2. \(0.1111\)
    3. \(0.3333\)
    4. \(0.8889\)
    5. \(1.7778\)

    Solution

    Chebyshev afirma \[P(|X-\mu|\geq k\sigma)\leq\frac1{k^2} =0.1111.\] O limite independe dos valores específicos de \(\mu\) e \(\sigma\).


  5. Question

    Sem supor uma família de distribuições, qual limite inferior Chebyshev garante para \(P(|X-\mu|<1.5\sigma)\)?


    1. \(0.4444\)
    2. \(0.5556\)
    3. \(0.6000\)
    4. \(0.3333\)
    5. \(0.6667\)

    Solution

    Pelo complemento da forma usual de Chebyshev, \[P(|X-\mu|<k\sigma)\geq1-\frac1{k^2} =0.5556.\]


  6. Question

    Se \(X\sim U(-\sqrt3,\sqrt3)\), então \(E(X)=0\) e \(\operatorname{Var}(X)=1\). Para \(k=1.1\), qual é o valor exato de \(P(|X|\geq k)\)?


    1. \(0.9091\)
    2. \(0.3649\)
    3. \(0.1736\)
    4. \(0.8264\)
    5. \(0.6351\)

    Solution

    O comprimento total do suporte é \(2\sqrt3\), e o intervalo central \((-k,k)\) tem comprimento \(2k\). Logo, \[P(|X|\geq k)=1-\frac{k}{\sqrt3} =0.3649.\] Chebyshev daria apenas o limite superior \(1/k^2=0.8264\).


  7. Question

    Se \(\phi\) é convexa, como a função x^2 em seu domínio, e as esperanças existem, qual é a desigualdade de Jensen?


    1. \(E[\phi(X)]=\phi(X)\) quase certamente.
    2. \(\phi(E[X])\leq E[\phi(X)]\).
    3. \(\phi(E[X])\geq E[\phi(X)]\).
    4. \(E[\phi(X)]\leq E(X)\).
    5. \(\phi(E[X])=E(X)\phi(1)\).

    Solution

    Para funções convexas, \[\phi(E[X])\leq E[\phi(X)].\] O sentido se inverte para funções côncavas.


  8. Question

    Se \(E(X)=2.5\), qual limite inferior Jensen fornece para \(E(X^2)\)? Considere \(\phi(x)=x^2\).


    1. \(2.000\)
    2. \(6.250\)
    3. \(2.872\)
    4. \(8.250\)
    5. \(2.500\)

    Solution

    Como \(x^2\) é convexa, \[[E(X)]^2\leq E(X^2).\] Assim, \(E(X^2)\geq(2.5)^2=6.250\). O valor 8.25 foi usado apenas para gerar distratores plausíveis.


  9. Question

    Para os números positivos \((4, 7, 1, 8)\), qual é a média harmônica?


    1. \(2.6353\)
    2. \(20.0000\)
    3. \(3.8687\)
    4. \(5.0000\)
    5. \(0.3795\)

    Solution

    A média harmônica é \[HM=\frac{n}{\sum_i1/x_i}=2.6353.\] Além disso, Jensen permite estabelecer \(HM\leq GM\leq AM\); aqui \(GM=3.8687\) e \(AM=5.0000\).


  10. Question

    Se \(\phi=-log(x)\) é estritamente convexa e \(E[\phi(X)]<\infty\), quando ocorre igualdade em Jensen, \(\phi(E[X])=E[\phi(X)]\)?


    1. Sempre que a distribuição de \(X\) é simétrica.
    2. Somente quando \(\operatorname{Var}(X)=1\).
    3. Somente quando \(X\) é constante quase certamente.
    4. Sempre que \(E(X)=0\).
    5. Para toda variável contínua.

    Solution

    Para uma função estritamente convexa, Jensen é estrita quando \(X\) é não degenerada. A igualdade ocorre precisamente quando toda a massa de \(X\) está concentrada em um único valor, isto é, quando \(X\) é constante quase certamente.
