Exam 4

  1. Question

    Uma variável discreta assume valores \((-2, 3, 4, 6)\) com probabilidades \((0.235, 0.471, 0.176, 0.118)\). Qual é \(E(X)\)?


    1. \(3.2941\)
    2. \(2.7500\)
    3. \(1.0000\)
    4. \(2.3529\)
    5. \(11.0000\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=2.3529.\]


  2. Question

    Se $f(x)=(2)x^{1}/49 para \(0<x<7\), qual é \(E(X)\)?


    1. \(3.5000\)
    2. \(4.6667\)
    3. \(0.6667\)
    4. \(10.5000\)
    5. \(2.3333\)

    Solution

    \[E(X)=\int_0^{7}x\frac{(2)x^{1}}{49}dx =7\frac{2}{3} =4.6667.\]


  3. Question

    Para valores \((-1, 1, 2, 3)\) com probabilidades \((0.286, 0.333, 0.238, 0.143)\), determine \(E(X^2)\) sem obter a distribuição de \(Y=X^2\).


    1. \(1.5238\)
    2. \(2.9167\)
    3. \(0.9070\)
    4. \(0.9524\)
    5. \(2.8571\)

    Solution

    Pela regra da esperança de uma função, \[E(X^2)=\sum_xx^2p_X(x)=2.8571.\]


  4. Question

    Se \(E(X)=-2.5\), qual é \(E(-3X+-5)\)?


    1. \(2.500\)
    2. \(-27.500\)
    3. \(-7.500\)
    4. \(7.500\)
    5. \(22.500\)

    Solution

    A linearidade fornece \[E(aX+b)=aE(X)+b=(-3)(-2.5)+-5 =2.500.\]


  5. Question

    Um jogo paga \((10, 0, 20)\) unidades monetárias com probabilidades \((0.368, 0.421, 0.211)\). Qual preço torna o ganho líquido esperado do jogador igual a zero?


    1. \(20.00\)
    2. \(7.89\)
    3. \(3.95\)
    4. \(10.00\)
    5. \(30.00\)

    Solution

    Se \(R\) é o prêmio e \(c\) o preço, o ganho é \(G=R-c\). Um jogo justo satisfaz \(E(G)=E(R)-c=0\), logo \(c=E(R)=7.89\).


  6. Question

    Se \(I_A=\mathbf{1}_A\) e \(P(A)=0.05\), qual é \(E(I_A)\)?


    1. \(0.048\)
    2. \(0.003\)
    3. \(0.050\)
    4. \(1.000\)
    5. \(0.950\)

    Solution

    A indicadora vale 1 em \(A\) e 0 fora de \(A\): \[E(I_A)=1P(A)+0P(A^c)=P(A)=0.05.\]


  7. Question

    Se \(X\) tem densidade exponencial \(f(x)=\lambda e^{-\lambda x}\), \(x>0\), com \(\lambda=3\), qual é \(E(e^{-0.5X})\)?


    1. \(0.1429\)
    2. \(0.2857\)
    3. \(6.0000\)
    4. \(0.8571\)
    5. \(0.2231\)

    Solution

    Pela regra de transformação dentro da esperança, \[E(e^{-tX})=\int_0^\infty e^{-tx}\lambda e^{-\lambda x}dx =\frac{\lambda}{\lambda+t} =0.8571.\]


  8. Question

    Considere \(f(x)=1/x^2\) para \(x>1\) e zero caso contrário. Qual afirmação sobre \(E(X)\) é correta?


    1. A esperança existe somente após restringir o cálculo a \(X>2\).
    2. A esperança é zero, pois a cauda tende a zero.
    3. A esperança vale 1, pois a densidade integra 1.
    4. A esperança não existe como número finito, pois \(\int_1^\infty x/x^2\,dx=\infty\).
    5. A esperança vale 2, pela simetria da densidade.

    Solution

    Embora \(f\) seja uma densidade, o primeiro momento absoluto diverge: \[E|X|=\int_1^\infty\frac1x\,dx=\infty.\] Normalização da densidade não garante existência da média.


  9. Question

    Uma variável geométrica \(X\) conta tentativas até o primeiro sucesso, com probabilidade \(p=0.4\). Usando \(E(X)=\sum_{k\geq0}P(X>k)\), qual é \(E(X)\)?


    1. \(0.6667\)
    2. \(0.4000\)
    3. \(1.5000\)
    4. \(0.6000\)
    5. \(2.5000\)

    Solution

    Como \(P(X>k)=(1-p)^k\), \[E(X)=\sum_{k=0}^{\infty}(1-p)^k=\frac1p =2.5000.\]


  10. Question

    Uma mistura escolhe a população 1 com probabilidade 0.8 e a população 2 com probabilidade 0.2. As médias são \(\mu_1=1\) e \(\mu_2=9.5\). Qual é a média da mistura?


    1. \(0.800\)
    2. \(-1.700\)
    3. \(5.250\)
    4. \(2.700\)
    5. \(10.500\)

    Solution

    Pela esperança total, \[E(X)=0.8\mu_1+0.2\mu_2 =2.700.\]
