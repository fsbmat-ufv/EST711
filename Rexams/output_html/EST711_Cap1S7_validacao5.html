Exam 5

  1. Question

    Uma função é dada por \(f(x)=c x^{4}\) para \(0<x<2\) e zero fora desse intervalo. Qual valor de \(c\) torna \(f\) uma densidade?


    1. \(0.31250\)
    2. \(0.50000\)
    3. \(0.03125\)
    4. \(0.15625\)
    5. \(0.40000\)

    Solution

    A normalização exige \[1=c\int_0^{2}x^{4}dx =c\frac{2^{5}}{5}.\] Logo, \(c=(5)/2^{5}=0.15625\).


  2. Question

    Se $f(x)=2x/16 para \(0<x<4\), qual é \(P(1.08<X<2.89)\)?


    1. \(0.2242\)
    2. \(0.4484\)
    3. \(0.1129\)
    4. \(0.4516\)
    5. \(0.5516\)

    Solution

    \[P(u<X<v)=\int_u^v\frac{2x}{a^2}dx =\frac{v^2-u^2}{a^2} =0.4484.\]


  3. Question

    Uma variável tem densidade \(f(x)=(4)x^{3}/625\) em \((0,5)\). Qual é \(F_X(2.66)\)?


    1. \(0.1205\)
    2. \(0.9199\)
    3. \(0.1506\)
    4. \(0.0801\)
    5. \(0.5321\)

    Solution

    Para \(0<x<a\), \[F_X(x)=\int_0^x\frac{(k+1)t^k}{a^{k+1}}dt =\left(\frac{x}{a}\right)^{k+1}.\] No ponto informado, isso vale 0.0801.


  4. Question

    Se \(F_X(x)=(x/7)^{4}\) para \(0<x<7\), qual é o quantil de ordem \(p=0.45\)?


    1. \(0.2870\)
    2. \(6.0282\)
    3. \(5.7333\)
    4. \(3.1500\)
    5. \(0.0643\)

    Solution

    Resolve-se \(F_X(q_p)=p\): \[\left(\frac{q_p}{7}\right)^{4}=0.45, \qquad q_p=7\,p^{1/4} =5.7333.\]


  5. Question

    Se \(X\) tem densidade \(f_X\) e \(Y=3X+2\), qual é a densidade de \(Y\)?


    1. \(f_Y(y)=f_X(y-2)\).
    2. \(f_Y(y)=f_X((y+2)/3)\).
    3. \(f_Y(y)=\dfrac1{9}f_X(y/3-2)\).
    4. \(f_Y(y)=3f_X(3y+2)\).
    5. \(f_Y(y)=\dfrac1{3}f_X\!\left(\dfrac{y-2}{3}\right)\).

    Solution

    A inversa é \(x=(y-2)/3\) e seu jacobiano absoluto é \(1/3\). Portanto, \[f_Y(y)=\frac1{3}f_X\left(\frac{y-2}{3}\right).\]


  6. Question

    Se \(X\sim U(0,4)\) e \(Y=X^2\), qual é \(f_Y(12.624)\)?


    1. \(1.77651\)
    2. \(0.88826\)
    3. \(0.00990\)
    4. \(0.03518\)
    5. \(0.07036\)

    Solution

    A inversa é \(x=\sqrt y\) e \(|dx/dy|=1/(2\sqrt y)\). Assim, \[f_Y(y)=\frac1{4}\frac1{2\sqrt y}, \quad 0<y<16,\] que no ponto dado vale 0.03518.


  7. Question

    Se \(X\sim U(-4,4)\) e \(Y=X^2\), qual é \(F_Y(2.156)\)?


    1. \(0.0851\)
    2. \(0.3670\)
    3. \(0.7341\)
    4. \(0.1347\)
    5. \(0.6330\)

    Solution

    Para \(0<y<a^2\), \[F_Y(y)=P(-\sqrt y\leq X\leq\sqrt y) =\frac{2\sqrt y}{2a}=\frac{\sqrt y}{a}.\] No ponto indicado, o valor é 0.3670.


  8. Question

    Uma distribuição é mistura de \(f_1\) e \(f_2\) com pesos 0.35 e 0.65. Para certo conjunto \(A\), \(P_1(A)=0.45\) e \(P_2(A)=0.7\). Qual é \(P(A)\) na mistura?


    1. \(0.3875\)
    2. \(0.1575\)
    3. \(0.5750\)
    4. \(0.6125\)
    5. \(0.3150\)

    Solution

    Probabilidades também se combinam pelos pesos da mistura: \[P(A)=0.35(0.45)+0.65(0.7) =0.6125.\]


  9. Question

    Se \(U\sim U(0,1)\) e \(Y=-\log(1-U)/1.5\), qual é a distribuição de \(Y\)?


    1. Normal com média \(1/1.5\).
    2. Gama com forma 1.5 e taxa 1.
    3. Uniforme em \((0,1/1.5)\).
    4. Exponencial com taxa \(1/1.5\), pois \(F_Y(y)=e^{-y/1.5}\).
    5. Exponencial com taxa 1.5, pois \(F_Y(y)=1-e^{-1.5y}\) para \(y>0\).

    Solution

    Para \(y>0\), \[P(Y\leq y)=P\{U\leq1-e^{-1.5y}\} =1-e^{-1.5y},\] que é a FDA exponencial de taxa 1.5.


  10. Question

    Uma variável \(X\) é absolutamente contínua e sua densidade em \(x_0=-2\) vale \(f_X(x_0)=0.4\). Qual é \(P(X=x_0)\)?


    1. \(F_X(x_0)\).
    2. 0.4.
    3. Não pode ser determinada sem integrar sobre toda a reta.
    4. \(0\).
    5. \(1-0.4\).

    Solution

    Em distribuições absolutamente contínuas, conjuntos com um único ponto têm probabilidade zero. O valor da densidade não é uma probabilidade pontual: \[P(X=x_0)=\int_{x_0}^{x_0}f_X(x)\,dx=0.\]
