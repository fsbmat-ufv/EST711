Exam 1

  1. Question

    Em 130 registros, 64 apresentam \(B\) e 43 apresentam simultaneamente \(A\) e \(B\). Qual é \(P(A\mid B)\)?


    1. \(0.3308\)
    2. \(0.6719\)
    3. \(0.4923\)
    4. \(0.3281\)
    5. \(0.5077\)

    Solution

    \[P(A\mid B)=\frac{P(A\cap B)}{P(B)} =\frac{43}{64}=0.6719.\]


  2. Question

    Se \(P(A)=0.75\) e \(P(B\mid A)=0.8\), qual é \(P(A\cap B)\)?


    1. \(1.0667\)
    2. \(0.6000\)
    3. \(0.1500\)
    4. \(1.5500\)
    5. \(0.8000\)

    Solution

    Pela regra da multiplicação, \[P(A\cap B)=P(A)P(B\mid A) =(0.75)(0.8)=0.6000.\]


  3. Question

    Três máquinas produzem proporções 0.35, 0.45, 0.20 dos itens, com probabilidades de defeito 0.07, 0.11, 0.09. Qual é a probabilidade de um item escolhido ao acaso ser defeituoso?


    1. \(0.0900\)
    2. \(0.2700\)
    3. \(0.0920\)
    4. \(0.9080\)
    5. \(0.0344\)

    Solution

    Pela probabilidade total, \[P(D)=\sum_{j=1}^3P(M_j)P(D\mid M_j) =0.0920.\]


  4. Question

    Uma condição tem prevalência 0.02. Um teste possui sensibilidade 0.86 e especificidade 0.85. Qual é \(P(\text{condição}\mid\text{positivo})\)?


    1. \(0.8600\)
    2. \(0.8500\)
    3. \(0.1048\)
    4. \(0.8952\)
    5. \(0.0200\)

    Solution

    Pelo teorema de Bayes, \[P(C\mid+)=\frac{P(+\mid C)P(C)} {P(+\mid C)P(C)+P(+\mid C^c)P(C^c)} =0.1048.\]


  5. Question

    Eventos \(A\) e \(B\) são independentes, com \(P(A)=0.8\) e \(P(B)=0.45\). Quanto vale \(P(A\cap B)\)?


    1. \(0.6400\)
    2. \(1.2500\)
    3. \(1.7778\)
    4. \(0.3600\)
    5. \(0.4500\)

    Solution

    A independência implica \[P(A\cap B)=P(A)P(B)=(0.8)(0.45) =0.3600.\]


  6. Question

    Se \(A\) e \(B\) são eventos independentes em um estudo de falha de componentes, qual afirmação é necessariamente verdadeira?


    1. \(A\) e \(B\) são necessariamente disjuntos.
    2. Exatamente um dos eventos deve ocorrer.
    3. \(P(A\cup B)=P(A)+P(B)\).
    4. \(A^c\) e \(B^c\) também são independentes.
    5. \(P(A\mid B)=P(B\mid A)\) para quaisquer marginais.

    Solution

    Temos \[P(A^c\cap B^c)=1-P(A)-P(B)+P(A)P(B) =P(A^c)P(B^c),\] logo os complementos são independentes. Independência não significa disjunção.


  7. Question

    Em uma moeda honesta lançada duas vezes, seja \(A=\) “cara no primeiro resultado”, \(B=\) “cara no segundo” e \(C=\) “os resultados são iguais”. Como se classificam esses eventos?


    1. São independentes dois a dois, mas não mutuamente independentes.
    2. Apenas \(A\) e \(B\) são independentes.
    3. São mutuamente independentes.
    4. Nenhum par é independente.
    5. São mutuamente exclusivos.

    Solution

    Cada evento tem probabilidade \(1/2\) e cada interseção de dois tem probabilidade \(1/4\). Porém, \[P(A\cap B\cap C)=\frac14\neq\frac18=P(A)P(B)P(C).\] Logo, independência aos pares não implica independência mútua.


  8. Question

    Uma urna contém 4 bolas vermelhas e 8 azuis. Duas são retiradas sem reposição. Dado que a primeira foi vermelha, qual é a probabilidade de a segunda também ser vermelha?


    1. \(0.2500\)
    2. \(0.2727\)
    3. \(0.7273\)
    4. \(0.3636\)
    5. \(0.3333\)

    Solution

    Restam 3 vermelhas entre 11 bolas. Assim, \[P(V_2\mid V_1)=\frac{3}{11} =0.2727.\]


  9. Question

    Dois componentes independentes têm confiabilidades 0.8 e 0.82. O sistema é do tipo série. Qual é sua confiabilidade?


    1. \(0.8200\)
    2. \(0.8000\)
    3. \(0.0360\)
    4. \(0.6560\)
    5. \(0.3440\)

    Solution

    Em série, ambos precisam funcionar e as probabilidades são multiplicadas. A confiabilidade é 0.6560.


  10. Question

    Ensaios de Bernoulli independentes têm probabilidade de sucesso \(p=0.6\). Qual é a probabilidade de o primeiro sucesso ocorrer exatamente no ensaio 4?


    1. \(0.0384\)
    2. \(0.0256\)
    3. \(2.4000\)
    4. \(0.1296\)
    5. \(0.9744\)

    Solution

    São necessários 3 fracassos seguidos de um sucesso: \[P(X=4)=(1-p)^{3}p =0.0384.\]
