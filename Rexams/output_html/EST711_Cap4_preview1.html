Exam 1

  1. Question

    Seja \(X_1, X_2, \ldots, X_n\) uma amostra aleatória de uma distribuição com média \(\mu\) e \(n = 25\). Considere o estimador \(T_n = 1.5 \cdot \bar{X}_n\) para estimar \(\mu\), onde \(\bar{X}_n\) é a média amostral.

    O vício (viés) de \(T_n\) é definido como \(B(T_n) = E(T_n) - \mu\). Como \(E(\bar{X}_n) = \mu\), temos \(E(T_n) = 1.5 \mu\), e portanto:

    \[B(T_n) = 1.5\,\mu - \mu = (1.5 - 1)\mu.\]

    Para \(\mu = 2\), o vício de \(T_n\) é:


    1. -1
    2. 1
    3. 3.03
    4. 0.251
    5. 3

    Solution

    O vício de \(T_n = 1.5\bar{X}_n\) como estimador de \(\mu\) é: \[B(T_n) = E(T_n) - \mu = 1.5\mu - \mu = (1.5 - 1)\mu.\] Para \(\mu = 2\): \[B(T_n) = (1.5 - 1) \times 2 = 1.\]


  2. Question

    Seja \(X \sim \text{Bernoulli}(p)\), com funcao de probabilidade: \[P(X = x; p) = p^x(1-p)^{1-x}, \quad x \in \{0, 1\}.\]

    A informacao de Fisher sobre o parametro \(p\) e definida como: \[I(p) = E\!\left[\left(\frac{\partial}{\partial p}\log f(X; p)\right)^2\right] = \frac{1}{p(1-p)}.\]

    Para \(p = 0.8\), qual e o valor de \(I(p)\)?


    1. 39.453125
    2. 39.0625
    3. 26.5625
    4. 6.25
    5. 0.1601

    Solution

    Para \(X \sim \text{Bernoulli}(p)\): \[\log f(x; p) = x\log p + (1-x)\log(1-p).\] \[\frac{\partial}{\partial p}\log f = \frac{x}{p} - \frac{1-x}{1-p}.\] \[I(p) = -E\!\left[\frac{\partial^2}{\partial p^2}\log f\right] = \frac{1}{p} + \frac{1}{1-p} = \frac{1}{p(1-p)}.\]

    Para \(p = 0.8\): \[I(0.8) = \frac{1}{0.8 \times 0.2} = \frac{1}{0.16} = 6.25.\]


  3. Question

    A familia exponencial de distribuicoes uniparametrica tem a forma: \[f(x; \theta) = \exp\{c(\theta)T(x) - d(\theta)\}\,h(x),\] onde o suporte de \(X\) deve ser independente de \(\theta\).

    Qual das seguintes distribuicoes nao pertence a familia exponencial de 1 parametro?


    1. \(\text{Exponencial}(\theta)\) — pertence a familia exponencial com parametro natural \(\eta = -\theta\).
    2. \(U(0,\theta)\) — pois o suporte depende de \(\theta\) e a funcao nao tem a forma fatoravel \(\exp\{c(\theta)T(x) - d(\theta)\}h(x)\) com suporte fixo.
    3. \(\text{Bernoulli}(p)\) — pertence a familia exponencial com parametro natural \(\eta = \log(p/(1-p))\) (logito de \(p\)).
    4. \(N(\mu, \sigma^2)\) com \(\sigma^2\) fixo — pertence a familia exponencial com parametro natural \(\eta = \mu/\sigma^2\).
    5. \(\text{Poisson}(\lambda)\) — pertence a familia exponencial com parametro natural \(\eta = \log\lambda\).

    Solution

    A distribuicao \(U(0, \theta)\) (uniforme no intervalo \((0, \theta)\)) nao pertence a familia exponencial porque seu suporte \((0, \theta)\) depende do parametro \(\theta\). A representacao na forma exponencial exige que o suporte seja fixo (independente do parametro).

    As demais distribuicoes pertencem a familia exponencial de 1 parametro:

    • \(\text{Bernoulli}(p)\): \(P(x;p) = \exp\{x\log(p/(1-p)) + \log(1-p)\}\), com \(\eta = \log(p/(1-p))\).
    • \(\text{Poisson}(\lambda)\): \(f(x;\lambda) = \exp\{x\log\lambda - \lambda - \log(x!)\}\), com \(\eta = \log\lambda\).
    • \(\text{Exp}(\theta)\): \(f(x;\theta) = \exp\{-\theta x + \log\theta\}\), com \(\eta = -\theta\).
    • \(N(\mu,\sigma^2)\) (\(\sigma^2\) fixo): forma exponencial com \(\eta = \mu/\sigma^2\).

  4. Question

    O conceito de estatistica suficiente e central na teoria da estimacao pontual. Seja \(X_1, \ldots, X_n\) uma amostra aleatoria com distribuicao parametrizada por \(\theta\) e seja \(T = T(X_1, \ldots, X_n)\) uma estatistica.

    Qual das seguintes opcoes define corretamente o que significa \(T\) ser suficiente para \(\theta\)?

    \(T\) e suficiente para \(\theta\) se:


    1. A distribuicao de \(T\) e da forma \(N(\theta, \sigma^2/n)\) assintoticamente, pelo Teorema Central do Limite.
    2. A variancia de \(T\) e minima entre todos os estimadores de \(\theta\).
    3. \(T\) e suficiente se e somente se \(T = (X_{(1)}, X_{(n)})\) (estatisticas de ordem extremas).
    4. \(T\) e nao-viesada para \(\theta\), isto e, \(E_\theta(T) = \theta\) para todo \(\theta\).
    5. A distribuicao condicional \((X_1,\ldots,X_n) \mid T = t\) nao depende de \(\theta\).

    Solution

    A definicao correta: \(T\) e suficiente para \(\theta\) se a distribuicao condicional \((X_1, \ldots, X_n) \mid T = t\) nao depende de \(\theta\).

    Interpretacao: dado o valor de \(T\), a amostra nao contem informacao adicional sobre \(\theta\). Em outras palavras, \(T\) captura toda a informacao relevante sobre \(\theta\) contida na amostra.

    As demais opcoes descrevem outras propriedades (nao-viesamento, variancia minima, distribuicao assintótica, forma especifica) que sao independentes da suficiencia.
