Exam 3

  1. Question

    Em 140 registros, 53 apresentam \(B\) e 29 apresentam simultaneamente \(A\) e \(B\). Qual é \(P(A\mid B)\)?


    1. \(0.5472\)
    2. \(0.4528\)
    3. \(0.2071\)
    4. \(0.6214\)
    5. \(0.3786\)

    Solution

    \[P(A\mid B)=\frac{P(A\cap B)}{P(B)} =\frac{29}{53}=0.5472.\]


  2. Question

    Se \(P(A)=0.7\) e \(P(B\mid A)=0.3\), qual é \(P(A\cap B)\)?


    1. \(1.0000\)
    2. \(0.2100\)
    3. \(0.4900\)
    4. \(0.3000\)
    5. \(0.4286\)

    Solution

    Pela regra da multiplicação, \[P(A\cap B)=P(A)P(B\mid A) =(0.7)(0.3)=0.2100.\]


  3. Question

    Três máquinas produzem proporções 0.30, 0.25, 0.45 dos itens, com probabilidades de defeito 0.12, 0.07, 0.04. Qual é a probabilidade de um item escolhido ao acaso ser defeituoso?


    1. \(0.0715\)
    2. \(0.0233\)
    3. \(0.0767\)
    4. \(0.2300\)
    5. \(0.9285\)

    Solution

    Pela probabilidade total, \[P(D)=\sum_{j=1}^3P(M_j)P(D\mid M_j) =0.0715.\]


  4. Question

    Uma condição tem prevalência 0.05. Um teste possui sensibilidade 0.98 e especificidade 0.91. Qual é \(P(\text{condição}\mid\text{positivo})\)?


    1. \(0.3643\)
    2. \(0.9800\)
    3. \(0.6357\)
    4. \(0.0500\)
    5. \(0.9100\)

    Solution

    Pelo teorema de Bayes, \[P(C\mid+)=\frac{P(+\mid C)P(C)} {P(+\mid C)P(C)+P(+\mid C^c)P(C^c)} =0.3643.\]


  5. Question

    Eventos \(A\) e \(B\) são independentes, com \(P(A)=0.45\) e \(P(B)=0.25\). Quanto vale \(P(A\cap B)\)?


    1. \(0.7000\)
    2. \(0.8875\)
    3. \(0.1125\)
    4. \(0.2500\)
    5. \(1.8000\)

    Solution

    A independência implica \[P(A\cap B)=P(A)P(B)=(0.45)(0.25) =0.1125.\]


  6. Question

    Se \(A\) e \(B\) são eventos independentes em um estudo de resultados de ensaios, qual afirmação é necessariamente verdadeira?


    1. \(A\) e \(B\) são necessariamente disjuntos.
    2. Exatamente um dos eventos deve ocorrer.
    3. \(A^c\) e \(B^c\) também são independentes.
    4. \(P(A\cup B)=P(A)+P(B)\).
    5. \(P(A\mid B)=P(B\mid A)\) para quaisquer marginais.

    Solution

    Temos \[P(A^c\cap B^c)=1-P(A)-P(B)+P(A)P(B) =P(A^c)P(B^c),\] logo os complementos são independentes. Independência não significa disjunção.


  7. Question

    Em uma moeda honesta lançada duas vezes, seja \(A=\) “cara no primeiro resultado”, \(B=\) “cara no segundo” e \(C=\) “os resultados são iguais”. Como se classificam esses eventos?


    1. São independentes dois a dois, mas não mutuamente independentes.
    2. Apenas \(A\) e \(B\) são independentes.
    3. Nenhum par é independente.
    4. São mutuamente exclusivos.
    5. São mutuamente independentes.

    Solution

    Cada evento tem probabilidade \(1/2\) e cada interseção de dois tem probabilidade \(1/4\). Porém, \[P(A\cap B\cap C)=\frac14\neq\frac18=P(A)P(B)P(C).\] Logo, independência aos pares não implica independência mútua.


  8. Question

    Uma urna contém 7 bolas vermelhas e 4 azuis. Duas são retiradas sem reposição. Dado que a primeira foi vermelha, qual é a probabilidade de a segunda também ser vermelha?


    1. \(0.6364\)
    2. \(0.4000\)
    3. \(0.5455\)
    4. \(0.7000\)
    5. \(0.6000\)

    Solution

    Restam 6 vermelhas entre 10 bolas. Assim, \[P(V_2\mid V_1)=\frac{6}{10} =0.6000.\]


  9. Question

    Dois componentes independentes têm confiabilidades 0.94 e 0.82. O sistema é do tipo paralelo. Qual é sua confiabilidade?


    1. \(0.2292\)
    2. \(0.9892\)
    3. \(0.8200\)
    4. \(0.0108\)
    5. \(0.9400\)

    Solution

    Em paralelo, o sistema falha apenas quando ambos os componentes falham. A confiabilidade é 0.9892.


  10. Question

    Ensaios de Bernoulli independentes têm probabilidade de sucesso \(p=0.2\). Qual é a probabilidade de o primeiro sucesso ocorrer exatamente no ensaio 3?


    1. \(0.1280\)
    2. \(0.5120\)
    3. \(0.6000\)
    4. \(0.4880\)
    5. \(0.0080\)

    Solution

    São necessários 2 fracassos seguidos de um sucesso: \[P(X=3)=(1-p)^{2}p =0.1280.\]
