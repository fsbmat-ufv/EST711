Exam 1

  1. Question

    Uma variável discreta assume valores \((-1, 0, 2, 4)\) com probabilidades \((0.412, 0.059, 0.235, 0.294)\). Qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(5.0000\)
    3. \(2.0588\)
    4. \(1.2500\)
    5. \(1.2353\)

    Solution

    A esperança é a média ponderada: \[E(X)=\sum_xxp_X(x)=1.2353.\]


  2. Question

    Se $f(x)=(3)x^{2}/64 para \(0<x<4\), qual é \(E(X)\)?


    1. \(1.0000\)
    2. \(3.0000\)
    3. \(2.0000\)
    4. \(0.7500\)
    5. \(5.3333\)

    Solution

    \[E(X)=\int_0^{4}x\frac{(3)x^{2}}{64}dx =4\frac{3}{4} =3.0000.\]


  3. Question

    Para valores \((-1, 3, 4, 5)\) com probabilidades \((0.071, 0.143, 0.286, 0.500)\), determine \(E(X^2)\) sem obter a distribuição de \(Y=X^2\).


    1. \(4.0000\)
    2. \(6.9167\)
    3. \(16.0000\)
    4. \(4.1429\)
    5. \(18.4286\)

    Solution

    Pela regra da esperança de uma função, \[E(X^2)=\sum_xx^2p_X(x)=18.4286.\]


  4. Question

    Se \(E(X)=-2.5\), qual é \(E(4X+-4)\)?


    1. \(-26.000\)
    2. \(-14.000\)
    3. \(-10.000\)
    4. \(-44.000\)
    5. \(-6.500\)

    Solution

    A linearidade fornece \[E(aX+b)=aE(X)+b=(4)(-2.5)+-4 =-14.000.\]


  5. Question

    Um jogo paga \((20, 10, 5)\) unidades monetárias com probabilidades \((0.312, 0.250, 0.438)\). Qual preço torna o ganho líquido esperado do jogador igual a zero?


    1. \(20.00\)
    2. \(11.67\)
    3. \(5.47\)
    4. \(10.94\)
    5. \(35.00\)

    Solution

    Se \(R\) é o prêmio e \(c\) o preço, o ganho é \(G=R-c\). Um jogo justo satisfaz \(E(G)=E(R)-c=0\), logo \(c=E(R)=10.94\).


  6. Question

    Se \(I_A=\mathbf{1}_A\) e \(P(A)=0.2\), qual é \(E(I_A)\)?


    1. \(0.800\)
    2. \(1.000\)
    3. \(0.160\)
    4. \(0.040\)
    5. \(0.200\)

    Solution

    A indicadora vale 1 em \(A\) e 0 fora de \(A\): \[E(I_A)=1P(A)+0P(A^c)=P(A)=0.2.\]


  7. Question

    Se \(X\) tem densidade exponencial \(f(x)=\lambda e^{-\lambda x}\), \(x>0\), com \(\lambda=3\), qual é \(E(e^{-2X})\)?


    1. \(1.5000\)
    2. \(0.0025\)
    3. \(0.4000\)
    4. \(0.2000\)
    5. \(0.6000\)

    Solution

    Pela regra de transformação dentro da esperança, \[E(e^{-tX})=\int_0^\infty e^{-tx}\lambda e^{-\lambda x}dx =\frac{\lambda}{\lambda+t} =0.6000.\]


  8. Question

    Considere \(f(x)=1/x^2\) para \(x>1\) e zero caso contrário. Qual afirmação sobre \(E(X)\) é correta?


    1. A esperança vale 1, pois a densidade integra 1.
    2. A esperança é zero, pois a cauda tende a zero.
    3. A esperança não existe como número finito, pois \(\int_1^\infty x/x^2\,dx=\infty\).
    4. A esperança vale 2, pela simetria da densidade.
    5. A esperança existe somente após restringir o cálculo a \(X>1\).

    Solution

    Embora \(f\) seja uma densidade, o primeiro momento absoluto diverge: \[E|X|=\int_1^\infty\frac1x\,dx=\infty.\] Normalização da densidade não garante existência da média.


  9. Question

    Uma variável geométrica \(X\) conta tentativas até o primeiro sucesso, com probabilidade \(p=0.45\). Usando \(E(X)=\sum_{k\geq0}P(X>k)\), qual é \(E(X)\)?


    1. \(1.2222\)
    2. \(0.5500\)
    3. \(2.2222\)
    4. \(0.8182\)
    5. \(0.4500\)

    Solution

    Como \(P(X>k)=(1-p)^k\), \[E(X)=\sum_{k=0}^{\infty}(1-p)^k=\frac1p =2.2222.\]


  10. Question

    Uma mistura escolhe a população 1 com probabilidade 0.45 e a população 2 com probabilidade 0.55. As médias são \(\mu_1=-0.5\) e \(\mu_2=9\). Qual é a média da mistura?


    1. \(-3.725\)
    2. \(-0.225\)
    3. \(4.250\)
    4. \(8.500\)
    5. \(4.725\)

    Solution

    Pela esperança total, \[E(X)=0.45\mu_1+0.55\mu_2 =4.725.\]
