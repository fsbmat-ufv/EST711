Exam 3

  1. Question

    Sabe-se que \(E(|X|^{4})<\infty\). Qual conclusão é garantida?


    1. A fgm de \(X\) existe para todo \(t\).
    2. \(X\) tem distribuição simétrica.
    3. \(E(|X|^{3})<\infty\).
    4. \(X\) possui suporte limitado.
    5. \(E(|X|^{5})<\infty\).

    Solution

    A existência de um momento absoluto de ordem \(m\) implica a existência de todos os momentos absolutos de ordens menores. Isso decorre de \(|x|^k\leq1+|x|^m\) para \(k\leq m\).


  2. Question

    Uma variável não negativa tem \(E(X)=20\). Pelo limite de Markov, qual é o maior valor garantido para \(P(X\geq30)\)?


    1. \(0.3333\)
    2. \(0.0222\)
    3. \(0.6667\)
    4. \(1.5000\)
    5. \(20.0000\)

    Solution

    A desigualdade de Markov dá \[P(X\geq c)\leq\frac{E(X)}c =\frac{20}{30}=0.6667.\]


  3. Question

    Sabe-se que \(E(X^2)=22\). Aplicando Markov a \(X^2\), qual limite superior se obtém para \(P(|X|\geq10)\)?


    1. \(22.0000\)
    2. \(0.7800\)
    3. \(1.0000\)
    4. \(0.4690\)
    5. \(0.2200\)

    Solution

    O evento \(|X|\geq c\) equivale a \(X^2\geq c^2\). Como \(X^2\geq0\), \[P(|X|\geq c)\leq\frac{E(X^2)}{c^2} =0.2200.\]


  4. Question

    Uma variável tem média 3.5 e desvio-padrão 2. Pelo limite de Chebyshev, qual é o limite superior para \(P(|X-\mu|\geq1.5\sigma)\)?


    1. \(1.7778\)
    2. \(0.4444\)
    3. \(0.6667\)
    4. \(0.5556\)
    5. \(2.2500\)

    Solution

    Chebyshev afirma \[P(|X-\mu|\geq k\sigma)\leq\frac1{k^2} =0.4444.\] O limite independe dos valores específicos de \(\mu\) e \(\sigma\).


  5. Question

    Sem supor uma família de distribuições, qual limite inferior Chebyshev garante para \(P(|X-\mu|<4\sigma)\)?


    1. \(0.9375\)
    2. \(0.0625\)
    3. \(0.8000\)
    4. \(0.2500\)
    5. \(0.7500\)

    Solution

    Pelo complemento da forma usual de Chebyshev, \[P(|X-\mu|<k\sigma)\geq1-\frac1{k^2} =0.9375.\]


  6. Question

    Se \(X\sim U(-\sqrt3,\sqrt3)\), então \(E(X)=0\) e \(\operatorname{Var}(X)=1\). Para \(k=1.4\), qual é o valor exato de \(P(|X|\geq k)\)?


    1. \(0.5102\)
    2. \(0.8083\)
    3. \(0.1917\)
    4. \(0.4898\)
    5. \(0.7143\)

    Solution

    O comprimento total do suporte é \(2\sqrt3\), e o intervalo central \((-k,k)\) tem comprimento \(2k\). Logo, \[P(|X|\geq k)=1-\frac{k}{\sqrt3} =0.1917.\] Chebyshev daria apenas o limite superior \(1/k^2=0.5102\).


  7. Question

    Se \(\phi\) é convexa, como a função x^2 em seu domínio, e as esperanças existem, qual é a desigualdade de Jensen?


    1. \(\phi(E[X])\geq E[\phi(X)]\).
    2. \(E[\phi(X)]\leq E(X)\).
    3. \(\phi(E[X])=E(X)\phi(1)\).
    4. \(E[\phi(X)]=\phi(X)\) quase certamente.
    5. \(\phi(E[X])\leq E[\phi(X)]\).

    Solution

    Para funções convexas, \[\phi(E[X])\leq E[\phi(X)].\] O sentido se inverte para funções côncavas.


  8. Question

    Se \(E(X)=2.5\), qual limite inferior Jensen fornece para \(E(X^2)\)? Considere \(\phi(x)=x^2\).


    1. \(6.250\)
    2. \(2.500\)
    3. \(2.872\)
    4. \(8.250\)
    5. \(2.000\)

    Solution

    Como \(x^2\) é convexa, \[[E(X)]^2\leq E(X^2).\] Assim, \(E(X^2)\geq(2.5)^2=6.250\). O valor 8.25 foi usado apenas para gerar distratores plausíveis.


  9. Question

    Para os números positivos \((11, 1, 2, 11)\), qual é a média harmônica?


    1. \(25.0000\)
    2. \(6.2500\)
    3. \(0.4205\)
    4. \(2.3784\)
    5. \(3.9442\)

    Solution

    A média harmônica é \[HM=\frac{n}{\sum_i1/x_i}=2.3784.\] Além disso, Jensen permite estabelecer \(HM\leq GM\leq AM\); aqui \(GM=3.9442\) e \(AM=6.2500\).


  10. Question

    Se \(\phi=-log(x)\) é estritamente convexa e \(E[\phi(X)]<\infty\), quando ocorre igualdade em Jensen, \(\phi(E[X])=E[\phi(X)]\)?


    1. Para toda variável contínua.
    2. Somente quando \(\operatorname{Var}(X)=1\).
    3. Sempre que \(E(X)=0\).
    4. Somente quando \(X\) é constante quase certamente.
    5. Sempre que a distribuição de \(X\) é simétrica.

    Solution

    Para uma função estritamente convexa, Jensen é estrita quando \(X\) é não degenerada. A igualdade ocorre precisamente quando toda a massa de \(X\) está concentrada em um único valor, isto é, quando \(X\) é constante quase certamente.
